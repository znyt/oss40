　　在李景禹这边在网上搜索着叶帝他们的这场训练赛的时候，网上很多关注着叶帝的粉丝，这个时候也都是打开了刚刚叶帝发布的这个视频。
<br/>　　“雪人加蛇女，这不是北美赛区前一阵拿出来的套路么，没想到被你们给用上了，你们是想用这个阵容来偷龙吧。”看着叶帝他们这边的选人，李景禹一针见血的说道。
<br/>　　“这个套路不错啊，算的上是一个小大招了吧，在赛场上如果玩的好的话，弄不好还真的能够让你们在20分钟左右拿下大龙。”秦宵跟着点头说道，眼中光辉闪闪。
<br/>　　不光是他，就是连带着李景禹几人也都是如此，看到蓝队几人的这个表情，叶帝他们哪里不知道这个时候对方在想着什么。
<br/>　　“就算被你们知道了也没什么，这一局比赛下来我们发现了，这个战术有着很大的弊端，雪人这个英雄真的是太费了。”叶帝混不在意的说道。
<br/>　　之前的那场训练赛，可以说完全是他们凭借着个人的实力取胜的，雪人这个英雄根本没有发挥出应有的作用。
<br/>　　在面对着类似皇族这样的强队，在他们看到自己等人选择的这个阵容的时候，他们就已经猜到了自己这边的想法了。
<br/>　　所以在20分钟那个时间段偷龙根本就是一个不现实的想法，虽然他们的打龙速度很快，但是雪人这个英雄作用还是太低了，真打起团来，能够发挥的作用并不大。
<br/>　　如果这个套路是第一次出现在赛场上的话，那可能还能大对手一个措手不及，但是这个战术已经在北美那边使用过了，大家看过比赛的应该都知道了，所以并不可行。
<br/>　　“确实，在看到这个阵容的时候对方就知道你们对龙有想法了，所以无论是大龙和小龙他们肯定会有所防备的，真要是团起来，应该是很难打赢。”听着叶帝的话，秦宵点头应着。
<br/>　　“这一场你们应对这个狮子狗做的很出色啊，飞刀这个你应该像夜雪学习学习，看到他布置的那几个视野的位置没有，那是防备狮子狗的最佳位置。”看着叶帝几人的应对，李景禹开口对着飞刀说道。
<br/>　　终于当时间来到最后最后的大龙团的时候，所有人的目光都专注了起来。
<br/>　　因为看着进度条大家都知道，此刻距离比赛结束已经没有多少时间了，前面并没有香蕉封神的表现，那么也就是说接下来就是封神之战的关键时刻了。
<br/>　　只不过这个时候大家倒是有些好奇，香蕉现在明明自己一个人在下路了，而且有着机械先驱的牵制，他们实在是想不到香蕉该如何发挥。
<br/>　　这一场训练赛的视频是以香蕉的视角发出去的，所以此刻镜头还在下路的武器的身上。
<br/>　　当大龙的血量还剩下不到一半的时候，只见场上的武器果断的在对方的眼皮子底下选择了传送。
<br/>　　这个时候楚白衣哪里会让他过去，这一局楚白衣带的可是净化技能，他是没办法跟着香蕉一起传送的。
<br/>　　如果这一波真的让香蕉自己过去的话，那么对面可就是5打4了，虽然雪人很废，但是皇族这边却是绝对没有赢下来的道理的。
<br/>　　正在观战中的秦宵等人自然也知道这个道理，其实在楚南几人的眼里，香蕉的这一波操作是有瑕疵的。
<br/>　　在他们眼里，其实在叶帝几人开大龙的时候，他就应该向后撤退，找好位置传送的。
<br/>　　反正机械先驱已经被他拖到下路了，这个时候肯定是没有他赶的快的，当然了就算是没有提前传送，但是这一波也不能就在对方的眼皮子底下传送的，要知道机械先去可是有技能能够打断他的传送效果的。
<br/>　　下一刻场上的机械先驱在看到武器的动作果然是向前走来，一个w技能重力场放在了武器的脚下，企图用这个技能来打断武器的传送。
<br/>　　重力场的眩晕时间是要小于传送的时间的，所以这个时候楚白衣倒是很轻松。
<br/>　　然而几乎就在机械先驱过来的一刹那，原本还在传送的武器居然立刻中断了传送，直接开启e技能反击风暴，随后一个q技能直接朝着机械先驱这边跳了过来。
<br/>　　原来就在刚刚那个时候，楚白衣为了释放w技能，他已经是走到了武器的q技能的跳斩范围之内。
<br/>　　在跳到对方身上的时候，香蕉这边连忙再次按着自己的e技能，直接将机械先驱眩晕在了那里。
<br/>　　看到这一幕其实蒋念他们都懂了，原来刚刚香蕉在对方眼皮子底下传送，就是为了吸引对方，而这个吸引，此刻看起来却是一个阳谋，就算是当时楚白衣知道，恐怕也别无选择。
<br/>　　因为如果他不打断香蕉的这个传送的话，让香蕉进入到主战场后，他们损失的将会变的更多。
<br/>　　此刻他这边虽然被眩晕了，但是并不代表着就没有逃跑的力气，别忘了他的手中可是还带着净化呢。
<br/>　　几乎在被香蕉眩晕的同一刻，楚白衣这边便是已经极限的利用净化解开了这个眩晕效果，直接一个闪现逃跑。
<br/>　　这个时候楚白衣很稳，甚至都没有选择利用自己的q技能加速逃跑，而是果断的选择了闪现。
<br/>　　只不过他这边很稳，香蕉那边却是已经杀意已决，在他这边闪现的时候，香蕉这边也是同步的跟上了一个闪现，破败直接一口吸在了对方的身上，刹那间机械先驱的移动速度便是锐减。
<br/>　　接下来的事情就很简单了，一个跑一个在后面追，虽然机械先驱还有q技能的加速效果，但是香蕉这边同样也有着q技能的跳斩，并且他的q技能的冷却时间要更加的断。
<br/>　　就这样在追到对方的高地防御塔后，在香蕉被防御塔击打了几下之后，他终于是成功的将机械先驱给击杀了。
<br/>　　接下来大龙buff加身，香蕉一人孤军深入，一人单枪匹马直接将对方的基地给推了下来。
<br/>　　看到最后香蕉一个人拯救赛场，所有观看着这一个视频的观众都跟着激动了起来。
