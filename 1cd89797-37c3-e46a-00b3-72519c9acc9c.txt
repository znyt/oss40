　　第336章
<br/>　　樊梨花面色稍霁，抵着程大雷那柄用杀气凝成的剑仿佛消失了，如此程大雷才松口气。
<br/>　　倒是面前这些女人露出迷茫模样。她们也是附近城镇的人，有些是被人送给莫鸣米，有些是很小年纪就被莫鸣米掳来。在琴川关倒也称得上养尊处优，只不过以声色哄莫鸣米开心罢了。现在，你告诉她们可以自由了，她们倒是要问一句：什么是自由？
<br/>　　暂时，她们还是留在将军府内，真要放在外面，还要小心程大雷的手下色迷心窍，糟蹋了她们。就连程大雷也差点把持不住，发生这种事也是稀松平常。
<br/>　　府里的丫鬟仆役也得想办法安抚，万一他们其中有一两个对莫鸣米忠心耿耿者，抽冷子暗算程大雷怎么办。
<br/>　　还有城内莫鸣米的亲信，比如表哥表弟，叔叔大爷之类的，这些若直接杀了，有亏公道，如果留着，却又难免养虎为患。
<br/>　　程大雷到此时自然也明白人命贱如狗，可惜他终究不属于这个世界，某些行为准则他虽然正在学，但暂时还学不来。
<br/>　　好在莫鸣米没什么亲信，他的夫人已经悬梁自尽，琴川关内没有他的子嗣。据说莫鸣米有个儿子，在其他人地方谋生，具体去向程大雷也不了解。
<br/>　　这给程大雷省下很多麻烦，万一他有个三岁的儿子，杀不杀都是个问题。
<br/>　　刚刚占领琴川关，百废待兴，城内的俘虏倒是好处理，把他们的军官关起来，好好安抚，或赏或罚，再通过这些军官控制普通士兵。
<br/>　　程大雷忙得焦头烂额。
<br/>　　“程大雷，是不是应该给附近的其他势力去个信儿，告诉他们我们来了。”
<br/>　　在整个山寨，对程大雷直呼其名者，要么是李行哉，要么是李婉儿。
<br/>　　说这话的是李婉儿，如果不是她提醒，程大雷还真想不起来。
<br/>　　是呐，初来乍到，生活要抓，军事要抓，外交工作也得搞起来。
<br/>　　“好好好，就由你来给他们写信，告诉他们我们来了，以后咱们你好我好大家好。”琴川关附近有几座城池，几道关卡，以前都是莫鸣米的关系，现在程大雷自然要接过来。
<br/>　　这些事还真得李婉儿来做，蛤蟆寨文笔不错的人，也就是她和李行哉了。
<br/>　　“要不要请他们过来做客，我们初来乍到，做做东道主。”李婉儿。
<br/>　　“哪有时间啊，不过你也可以写上，但我估计他们不会来。”
<br/>　　程大雷还有更重要的事情要做，那就是修建四级山寨。琴川关有高墙，有铁矿……这些恰是修建四级山寨必须的。如果在蛤蟆岛基础上开四级本，还不知要花多长时间。而眼下只要将被毁坏的建筑修复，再做一些改造就可以。
<br/>　　四级本比三级本多了很多建筑，例如市场，兵营，还能开新的系统商店，这正是程大雷目前需要的。
<br/>　　“大当家，发现了一些东西，你过来看。”徐神机来到程大雷面前。
<br/>　　“什么东西呐？”程大雷道。
<br/>　　“你看见就明白了。”徐神机道。
<br/>　　程大雷随着徐神机来到琴川关的矿场，刚进去他就吓了一条，只见一群群人聚在一起。他们身上几乎无任何血色，瘦骨嶙峋，像极了程大雷曾在电视上见过的难民。
<br/>　　“这些人是干什么的？”程大雷问。
<br/>　　“已经问清楚了，都是莫鸣米抓来挖矿的，有些被抓来三个月，有些被抓来一年，时间最长的没有超过三年。”
<br/>　　“三年以上的呢？”程大雷问。
<br/>　　“死光了。”
<br/>　　“呃……”
<br/>　　“大当家，这些人该怎么处理？”
<br/>　　“先找地方看起来，有愿意回家的就送他们回家。”
<br/>　　“大当家，还有一些东西。”
<br/>　　“还有什么？”程大雷吓了一跳：“能不能一次把话说完，还有更恐怖的东西？”
<br/>　　“不，不，这次一点也不恐怖。”徐神机领着程大雷来到矿场的仓库。
<br/>　　程大雷进入后就愣住了，里面密密麻麻摆满盔甲、兵器、箭头，装备打造得都很精良，完全是统一制式。
<br/>　　“点过数目了么？”程大雷问。
<br/>　　“还没有，但看上去至少有三五万的装备吧，这都够装备一支大军了。”
<br/>　　“是呐，莫鸣米准备这么多装备做什么，也没见他拿出来用啊。”程大雷暗暗琢磨着，把严笛杀死后，也就没有知道内情的人。
<br/>　　“回头找人问问，看俘虏中有知道的没有。”
<br/>　　……
<br/>　　李婉儿所写的信递出去后，果然如程大雷所料，并没有人来找程大雷麻烦，大家都默认了程大雷取代莫鸣米。但同时，也没有人来程大雷这里赴宴，回信大概用身体不便，要事缠身之类的借口推脱了。
<br/>　　但于此同时，程大雷占领琴川关的事情已经传开，甚至在很短时间内便传到长安，或者说，从一开始就有眼睛盯着这里。
<br/>　　“婉儿还在他哪里？”明帝合上手中的书卷。
<br/>　　“不仅公主殿下，我们还看到了六王子。”一人隐在黑暗中，看不清模样。
<br/>　　“老六……”明帝顿了顿：“还活着。”
<br/>　　“他们警戒得很严，我们没办法靠近，只是远远看了一眼，是王子殿下。”
<br/>　　“天下还有鱼龙卫进不去的地方。”明帝摇摇头：“他们还好么？”
<br/>　　“穷山僻壤，自然不能和长安相比，二位殿下都受了委屈，但看上去精神还好。”
<br/>　　“老六就是喜欢胡搞，婉儿也越来越没规矩。我这两个孩子，一个也不让我省心呐。”明帝叹口气，道：“那山贼占了琴川关？”
<br/>　　“莫鸣米已死，琴川关现在已经落入他的手中。”
<br/>　　“用了半年时间，才做到这一步，动作也真够慢的。”明帝忽然笑笑：“那边应该很不开心吧？”
<br/>　　相府。
<br/>　　一个留着几缕山羊胡的老者面沉似水，大厅内弥漫着一股令人压抑的气氛。
<br/>　　“大人，派谁去解决这件事？”
<br/>　　“让伯康去吧，这人能用就用，不能用就杀，绝不可让琴川落入外人之手。”
<br/>　　说罢，老者叹了口气，拱手向天。
<br/>　　“明帝陛下，高呐。”
<br/>　　
<br/>(本章完)
