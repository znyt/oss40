        落以柔抹去眼角的泪水，低着头，走了出去。
<br/>
<br/>        刘叔进来后，一边给傅夜沉换床单被套，一边忍不住唠叨了几句：“大少爷，落小姐对您一片真心，您这是何苦呢？”
<br/>
<br/>        傅夜沉不吭声，自顾自地坐在躺椅上继续抽烟，待刘叔把床单被套换好后，他突然接到了一个电话。
<br/>
<br/>        “一起准备就绪。”对方只说了这一句话。
<br/>
<br/>        傅夜沉将指间的雪茄，按灭在一旁的烟灰缸中，冷冷地说道：“好！我今晚会准时到，按计划行动。”
<br/>
<br/>        刘叔见傅夜沉起身换衣服，早已见怪不怪，但仍旧会很深沉地叹口气，叮嘱道：“大少爷，一定要注意安全。”
<br/>
<br/>        “嗯。”傅夜沉应了声。
<br/>
<br/>        这半年来，他可以说是一直在玩命。
<br/>
<br/>        就连傅老太爷也管不了他。
<br/>
<br/>        刘叔心疼老太爷，也更心疼这大少爷：“大少爷，傅氏集团不能没有继承人。如果您不愿意再娶别的女人，其实可以做试管婴儿，找个女人代孕就是。傅老太爷也是这个意思。您看……”
<br/>
<br/>        “傅家又不只是我一个儿子，不是还有我弟弟阿景吗？你跟老爷子说。我要是真没了，傅氏集团便交给阿景继承。”傅夜沉一边扣着衣襟上的扣子，一边云淡风轻地回答。
<br/>
<br/>        刘叔反对道：“傅氏集团能有今天，是大少爷您的功劳。是您扩大了傅氏集团的版图，到头来交给二少爷，那岂不是……”太便宜二少爷了。
<br/>
<br/>        后面的那几个字，刘叔说不出来。
<br/>
<br/>        为什么所有的好事都归了二少爷？
<br/>
<br/>        傅夜沉倒是一脸无所谓，但等到静下心来的时候，他的声音有些暗哑：“或许，阿景的命比我好。”
<br/>
<br/>        一个月后……
<br/>
<br/>        一辆白色的厢式运输车在安防局的大门前停了下来，遇上这种来历不明的车辆，安防局大门四周立即冒出两支武装部队，数十杆枪口对准了这辆车。
<br/>
<br/>        司机举起手从驾驶座上跳下来，支支吾吾地说道：“我、我是受了‘五帝门’的帝少所托，来给你们局长送大礼的！”
<br/>
<br/>        一听说是“大礼”立即全局警备。
<br/>
<br/>        这辆运输车的后车厢，被各种仪器扫描检测，确定是安全的后，才被打开了车厢的大门。
<br/>
<br/>        当夏阳枭接到下属的汇报时，顿时一脸诧异。
<br/>
<br/>        车厢一共四个人，是冒险者集团里的高管八爷，以及其他三个杀手：三爷、五爷和六爷。
<br/>
<br/>        他们的手脚都被手铐束缚着，因为被注射了安眠药，所以，当手下将他们押入暗牢的时候，他们都没有醒过来。
<br/>
<br/>        与他们被一起送过来的，还有一台平板电脑。
<br/>
<br/>        平板电脑里像是被人植入了病毒程序，只要有人拿起平板电脑，就会自动播放着，他们这几个人所干的一些非法事情。
<br/>
<br/>        视频是偷拍的，但很明显，他们是被人下了套。
<br/>
<br/>        夏阳枭看完平板电脑里的内容后，立即下令让人彻查傅氏集团的资金去向。
<br/>
<br/>        结果，傅氏集团的财务被查了一通后，全都显示正常。
<br/>
<br/>        夏阳枭现在设想的是，傅夜沉一定是花了巨额在用“黑控黑”的方法，在对冒险者集团的人“赶尽杀绝”。
<br/>
<br/>        他给他们下套，抓住他们又不杀了他们，而是直接往他安防局里送，很显然是在逼他们安防局。
<br/>
<br/>        那平板电脑里的视频一看就不是原件，很显然，他留了底。
<br/>
<br/>        所以，安防局的人若不顺着他傅夜沉的意思，依法办了那几个人，事情闹大后，他夏阳枭也会被安上一个包庇罪。
<br/>
<br/>        傅夜沉在逼他！
<br/>
<br/>        他查冒险者集团查了近四年的时间，才查出个矛头，而他傅夜沉却只用了半年的时间，就折断了这个集团的左膀右臂。
<br/>
<br/>        越是这样的结果，越令夏阳枭感到不安，因为傅夜沉已经脱离了他的控制，完全不再听他的指令。
<br/>
<br/>        更何况，傅夜沉养了一批忠心耿耿的精英，在他卖命，而且还把所有的事情都做得天衣无缝。
<br/>
<br/>        而他现在，完全找不到傅夜沉的弱点。
<br/>
<br/>        夏阳枭原本以为，童筱颖会是傅夜沉的弱点，但是他没料到，童筱颖出事的那天，傅夜沉连面都未露过。
<br/>
<br/>        这个没有弱点的干儿子，迟早会成为他最大的威胁。
<br/>
<br/>        才安静了半年，连华生又被夏阳枭叫到局里问话。
<br/>
<br/>        问话的重点，还是有关于傅夜沉。
<br/>
<br/>        “老大、夏局、夏叔叔！”连华生把能用上的称呼，都用上了，“您就放过我行吗？您怀疑我跟阿沉有联系，都怀疑了大半年了。对我各方位的监听监视，也没见您得出个什么结果来啊！”
