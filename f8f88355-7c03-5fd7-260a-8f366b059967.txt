　　影天使的语气非常肯定，因为他跟子午交手过不止一次，他败给过子午很多次，每一次失败，子午的“绝对防御”起到了很大的作用，所以影天使一直以来都在研究“绝对防御”并且试图破解“绝对防御”。可以说，子午的“绝对防御”已经成了影天使的心病，让影天使无时不刻都想要战胜它。
<br/>　　在他销声匿迹的这么长时间里，他一直在研究如何攻破“绝对防御”，而他也确实做到了，所以他摆下了这次对决，他自认为自己自己能赢，而且会赢得非常漂亮。
<br/>　　可现在事实却并不是如此，他居然无法攻破子午的防御。
<br/>　　“既然研究了‘绝对防御’这么长时间，你难道还认不出来吗？”子午双手还在键盘上敲打着，说道：“过去了这么长时间，你一直在进步，我没有道理原地踏步。这还是‘绝对防御’，只是这已经不再是以前你所熟悉的‘绝对防御’。这个‘绝对防御’是经过我改进的，而这个改进后的‘绝对防御’除了我的学生外，从没有在世界上出现过，你觉得能在这不到十分钟的时间内攻破吗？”
<br/>　　此时此刻，影天使已经明白子午想要干什么，这家伙从走进这栋大楼，不，是在他接下自己挑战书的那一刻起，他就做好了跟自己同归于尽的打算！这个混蛋！
<br/>　　“老师！”
<br/>　　子午塞在耳朵中的耳麦，传来云熙的声音。子午身上的战场视频系统将发生在房间里的情况完整的反馈回指挥中心，所以指挥中心中所有人都知道了子午的打算，其中最紧张的就是云熙。
<br/>　　云熙从小失去父母，哥哥也因为要寻找父亲的下落，很早就出国，所幸云家人一直把她当成云家的公主，从小没有吃苦，把所有最好的都给她，但是这并不能弥补云熙从小缺少的父母关爱，直到遇见子午。
<br/>　　子午没有结婚，也没有孩子，在把云熙作为自己接班人来培养后，子午不仅把她当做自己的学生，也把她当成自己的女儿；云熙跟子午之间是亦师亦父的关系。
<br/>　　“傻丫头，人总是要死的，就要看自己死的有没有价值。”子午双手还在键盘上敲打着，他显得非常平静，说道：“能拉着影天使一起去死也不算亏啊，我们做了这么多年对手，死了能继续当对手也不错啊，‘十二主天使’之一的影天使，不算亏啊。”
<br/>　　“师傅啊……”
<br/>　　云熙急的直跺脚，眼眶发红。
<br/>　　“丫头，你做得很好，你很出色配得上‘GHOST’这个名字！”子午说道：“以后我不在了，你就是新的‘GHOST’。但是记住，不要用你掌握的东西去做违法的事情，不然我做鬼也不会放过你！其实我也不需要担心这点，因为你是云家的孩子。”
<br/>　　“我不要啊。”云熙流着泪，说道：“我不要做‘GHOST’啊，你才是‘GHOST’，只有你才能是‘GHOST’！”
<br/>　　“‘GHOST’不是唯一的！”子午说道：“你的水平已经不比我差了，以后你一定会比我更厉害！今天以后，这个世界上SHADOW不会再存在，但是GHOST还会存在。对了丫头，我死后给我上坟的时候，记得给我买好一点的烟，你这次给我买的烟实在太难抽了，呛死人了。不要买外国的烟，抽不习惯啊。”
<br/>　　“老师啊……”
<br/>　　“承影。”子午说道。
<br/>　　“我在。”赵天宇看着指挥中心正前方的大屏幕，回答道。
<br/>　　“谢谢你。”子午对着赵天宇感谢道：“如果当时不是你救我，我现在应该被关在美国佬的监狱中；如果不是你救了我，我还不知道，原来作为一个黑客也是可以做到很多以前我不敢想的事情，原来黑客也是可以爱国的。”
<br/>　　子午看了眼塑胶炸药计时器上的时间，说道：“时间差不多了，我跟傻丫头再说几句。”
<br/>　　“老师……”
<br/>　　“丫头。”子午说道：“老师不可能永远是你的老师，学生总归是要超过老师的。记住我叮嘱你的，以后买烟记得买好一点的，全都是承影那家伙，原来我有多大的烟瘾，全都是他让我戒烟了。”
<br/>　　“如果不是我，你早就抽烟抽死了。”赵天宇说道：“如果不是我，你哪来这么好的学生。”
<br/>　　赵天宇知道再难改变现状，倒是跟子午开启了玩笑。
<br/>　　“老师，还有时间，你赶紧想办法从里面出来啊！”云熙着急道：“你还有时间的，你是‘GHOST’，你一定可以跑出来的！”
<br/>　　“没用的丫头。”子午轻轻摇头，说道：“SHADOW毕竟是SHADAOW，他是为了杀我才设下的这个局，他又怎么能这么容易就让我跑出去？”
<br/>　　“SHADOW！！”
<br/>　　子午抬起头，看着墙壁上液晶屏上的摄像头，大声说道：“还有两分钟，你觉得你能从房间里逃出来吗！？就算我现在什么都不做，你也不可能从这个房间里逃出来。我告诉你，这个‘绝对防御’是我创造的，也只有我才能破解它！要想破解这个‘绝对防御’只有用我独创的‘圣剑’，除了我之外，这个世界上还有两个人会！也就是说，就算我今天死了，以后还会有两个GHOST出现，但是SHADOW不会再有！”
<br/>　　“疯子，你这个疯子！”影天使此时已经无法再保持之前的镇定，他的双手在键盘上不断的敲打着，可是究竟敲打出来的是什么东西，恐怕连他自己也不知道。
<br/>　　影天使站起身，踢开椅子，用来拍打着这扇把房间锁死的门，大声喊道：“智天使，救我出去！！智天使，快点把我救出去！！”
<br/>　　而此时的智天使早就已经离开了这层楼，电梯中，智天使拿出手机，拨打了一个号码，对吴云说道：“准备撤退，影天使败了，外面的人很快就会攻进来！”
<br/>　　“明白！”
<br/>　　吴云关上目镜，从玻璃墙背后撤离。
<br/>　　“狂龙，吴云那兔崽子撤了！”
<br/>　　始终盯着吴云情况的慕容飞，立刻就发现了吴云撤离的动作。
