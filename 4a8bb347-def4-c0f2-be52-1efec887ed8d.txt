　　以叶少阳的修为，顶天了也最多能同时激活二十张暗金神符，对他来说不算什么，但这房间四壁全贴满了暗金神符，不用想也知道，在暗金神符之间还贴了许多“引流符”。
<br/>　　引流符不是用来引产和人流的，而是一种本身没属性的功能性灵符，必须与一张攻击性灵符结合，当这张引流符接触到任何法术的灵力时，会将灵力引到攻击性灵符上，激活这张灵符。
<br/>　　一张引流符，可以搭配三张攻击性灵符使用，也就是说，法师可以不用针对每一张灵符逐个作法，通过引流符，可以用法术的灵力直接激活灵符。
<br/>　　不过引流符最大的弊端，是有几秒钟的延迟，在实战中，每一秒都相当珍贵，对手不可能给你足够的时间作法，因此在实战中引流符并不实用，只能靠提前布置，关键时候激活，才能瞬间发挥威力，但这样的话，一样还是有延迟，对手看到苗头，只要不是智障，当然拔腿就跑，只要躲开灵符的范围，也就没用了。
<br/>　　所以这引流符，除了极少数特殊情况，基本就是鸡肋。
<br/>　　但是，引流符加上暗金神符，再加上玻璃镜的地面，那就好玩了：灵符打在玻璃镜上，反射出去，落在引流符上，再点燃更多地暗金神符，这才造成了眼前这可怕的局面。
<br/>　　面对枪林弹雨一般的星光，影魅自身难保，只能松开了叶少阳，收了无间地狱的幻象，但这时候，叶少阳却反过来将勾魂索捆在了他的身上——勾魂索是阴间法器，能索拿一切形态的邪物。叶少阳扣紧了勾魂索，死死拉住，冲着影魅眨巴着眼睛。
<br/>　　“你跟着我冲进来，光想着杀我，没想到这地面是镜子铺的吧？让我来告诉你这是怎么回事……”叶少阳主动开口讲述起来，倒不是炫耀，而是故意说这个来分他的心，同时也算得上是一次泄愤。
<br/>　　“你千变万化，如影随形，什么法器都奈何不了你，自从第一次跟你照面之后，我就一直在琢磨啊，怎么才能对付你，上一次，就在这里，又让你坑了一次，后来我就想到了这一招，镜子，没错，任凭你手段通天，无形无相，你唯一不能穿过的，就是镜子！
<br/>　　所以在下墓之前，我布置了这里！当时为了提防你在暗处发现，我在外头又着急大伙一起布置了阵法，确保你不在附近偷窥我……我画了几百张暗金神符，金粉都用了特么的好几两，多大的成本的，就是为了有这么一天，把你困死在这里！”
<br/>　　说到这里，叶少阳也感觉很爽，有一种扬眉吐气的感觉，最近一直憋在虚幻空间里，积累了一肚子怨气，本来是发泄在星月奴身上的，结果那一架也没打上，更加憋闷了，正好影魅就找上门来了。
<br/>　　“五百张九星神符的威力，影魅，够你特么喝一壶了吧！”
<br/>　　影魅听叶少阳说出真相，心惊胆战，多少年来，不管面对什么样的对手，他从来会没害怕过——就算对手强过自己，凭他这身无形无相的特殊手段，逃走总是没问题的，但是这一刻，他害怕了。
<br/>　　但是眼前的局面，让他连害怕都没时间了，在抵挡了一轮星光的攻击之后，他周身的结界破碎，按照他的习惯，是该遁地逃走的，但玻璃镜没法穿透，影魅心中发狠，将一身修为逼出体表，一瞬间震开了叶少阳的束缚，拖着勾魂索，朝最近的墙壁飞去。
<br/>　　这一点，倒是有些出乎叶少阳的意外，虽然这房间被他布置得像是铁桶阵一样，但其中最大的隐患，就是影魅强行逃走。
<br/>　　因此，他之前才故意一步步让影魅把自己逼到绝境，开启无间地狱，以为能搞定自己，在他用尽全力的一瞬间，激活灵符，然后趁着影魅最惊慌失措的一瞬间，用勾魂索困住他。
<br/>　　但这种环境之下，他居然能强行挣脱逃走，的确让人吃惊。
<br/>　　影魅在空中疾飞，也不躲避那雨点般的星光（太多也躲不了，而且浪费时间），拼着一身修为，任凭那些星光打在自己身上，就像子弹一样，每有一道星光穿过，就是一个窟窿，然后一股黑烟冒出来。
<br/>　　就算冲出去，估计没有一年半载，也修复不过来了。
<br/>　　想到这影魅觉得自己有点狼狈，但转念一想，毕竟保存了性命，在即将接触到墙壁的一刻，他得意地说道：“叶少阳，你是个厉害的，但千算万算，你少算了一样东西——我的实力，比你想得要强太多，待我养息一番，再来杀你！下一次，我倒要看看你还能有什么手段。”
<br/>　　“你没机会了。”叶少阳望着他，轻轻一笑，露出两个虎牙，人畜无害的感觉。
<br/>　　话音未落，一道人影，手握长剑，急速射了出去。
<br/>　　吴嘉伟，一直没直接动手的吴嘉伟！
<br/>　　被影魅认为只是一个普通角色，丝毫没放在心上的他，此刻展示出了极致的速度，手中的藏锋剑如一道光射过来。影魅刚要躲避，剑锋从他眉心中间穿过去，人影没动。
<br/>　　吴嘉伟停步，头也不回。
<br/>　　影魅捂着脑袋，停顿了几秒钟，身体突然剧烈颤抖起来，双手抱着脑袋，在地上翻滚着——方才那一剑，已经将剑气留在了他体内，此刻发作了。
<br/>　　无数的星光，不断打在他身上，从他身体洞穿过去，留下一道道的黑烟，他的身体迅速膨胀起来，黑影在地上展开，飞速蔓延……
<br/>　　叶少阳不知道怎么回事，招呼吴嘉伟一起躲到角落里，在身前摆下雕母大钱镇守，抵挡黑影的靠近。两人一脸懵逼地看着这一幕，难道他这都不死？
<br/>　　黑影蔓延到了整个房间，然后……维持了不到十秒钟，突然听见噗嗤一声，好像什么地方漏气了，然后见看到一股黑色浓烟，在房间里升腾起来，带着一种炙热的威势。
<br/>　　叶少阳过去开门，跟吴嘉伟走到外面站着。
<br/>　　“少阳哥！”周静茹一干人在远处看到他们出来，立刻大声叫起来。
