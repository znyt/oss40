　　听到慕清霄自称九品炼丹师，清微不动声色，但心中却再一次掀起滔天巨浪，九品炼丹师，确定没有逗老夫？
<br/>　　当初，蜀山派祖师爷太清真人在开山立派前，在丹道上便拥有非同寻常的成就。
<br/>　　直到后来，蜀山派在没有出现一位杰出的炼丹师，以至于炼丹一脉淡出视线，炼丹师绝对是比修士更尊贵的存在。
<br/>　　可面前这位年轻的有些不像话的大乘期修士，居然还是一名九品炼丹师！
<br/>　　九品，九品炼丹师，更上一层可就是十品，十品过后就能炼制仙丹，也称为仙品炼丹师。
<br/>　　后者现在不过大乘期修士，等渡劫成仙，成为仙品炼丹师已经是板上钉钉的事情，若是拉入蜀山派，绝对是蜀山的福气。
<br/>　　蜀山派祖师爷太清真人在上，恐怕也会欣慰。
<br/>　　见清微陷入沉默，慕清霄脸上浮现一抹不悦：“看来，清微掌门是看不上在下的身份，也罢，在下还是另寻他处吧。”
<br/>　　说罢，慕清霄转身就要离开。
<br/>　　“小道友，请留步。”
<br/>　　下一秒，身后就响起清微挽留的话语。
<br/>　　转过身来，见清微一副笑眯眯的模样，慕清霄暗骂后者臭不要脸的，还地仙呢，真是变脸比翻书还快。
<br/>　　“清微掌门还有何事？如果无事，在下就不打扰了。”
<br/>　　看出了慕清霄有些不悦，但清微也不怕丢脸，依旧是一副笑眯眯的模样，和煦无比。
<br/>　　“正如小友所说，蜀山乃是天下妙法齐聚之所，而且小友与蜀山有些因果，正巧老夫也有重启妙丹堂的想法，如有小友帮助，想必妙丹堂定能长久下去。”
<br/>　　慕清霄有些无语，当真佩服清微脸皮子的功底。
<br/>　　先前对自己爱理不理，现在摆出九品炼丹师的身份，立刻就有了重启妙丹堂的想法，您老真是思维逆转啊。
<br/>　　见慕清霄陷入思考，清微顿时觉得有戏，立刻开出条件。
<br/>　　“妙丹堂重启，小友身为九品炼丹师，必然是妙丹堂长老，本门任何能够学习的妙法都对长老开放……”
<br/>　　“在下就算去别的仙门，地位恐怕也不比在蜀山低，同样也能获得不少妙法吧？”
<br/>　　慕清霄实话实说，他九品炼丹师的身份，无论摆在何处，地位都非常的高，哪怕在神界，估计天帝也会将他供着。
<br/>　　以他的年纪与修为，突破地仙只是时间与资源问题。
<br/>　　突破地仙后，那可就是仙品炼丹师啊，两者概念完全不一样，哪怕六界中，恐怕都找不出一位仙品炼丹师。
<br/>　　只要清微不蠢，绝对会开口拉拢他，慕清霄坚信。
<br/>　　“小友说笑了，不是老夫吹，其他仙门哪能与蜀山相比，蜀山传承千年，底蕴万万不是其他门派能比的，况且一旦妙丹堂重启，后山的药园都有小友执掌，你看如何？”
<br/>　　为了拉拢慕清霄，清微也不惜下了血本，只要慕清霄进入蜀山，地位除了蜀山五老，绝对不再其他人之下。
<br/>　　甚至连药园的掌控权都让了出来，那里可是蜀山重地，其中灵药绝对整个仙界最多的地方。
<br/>　　慕清霄眼神一亮，若说他来蜀山只是为了掠夺积分，现在看来，收获远比预料中要多出太多，整个药园啊！
<br/>　　“清微掌门说话可算数？”
<br/>　　清微眼中闪过一抹笑意，道：“自然，只要小友证明九品炼丹师的身份，老夫即可宣布重启妙丹堂，给予你实权。”
<br/>　　“嗤嗤……”
<br/>　　话语刚落，慕清霄伸出右手，一抹黑色火焰自掌心升腾而起，微微摇曳间，空间都肉眼可见的扭曲起来。
<br/>　　“好诡异的丹火！”
<br/>　　见慕清霄释放出丹火，清微眼中一阵骇然，对他九品炼丹师的身份已经信了九分。
<br/>　　甚至，他从慕清霄掌心中那没有一丝温度的丹火上，感到一阵令人余悸的气息，仅仅释放出一缕就能扭曲空间，这是何等的恐怖！
<br/>　　丹火？
<br/>　　慕清霄嗤之以鼻，神火也是区区丹火能够比拟的，但脸上却挂着沐浴春风一般的笑容。
<br/>　　“不错，真是想不到，老夫竟能遇到小友这等天人，待老夫与各位长老商议过后，便准备重启妙丹堂一事。”
<br/>　　“也好。”
<br/>　　顺利的将慕清霄拉进蜀山，清微心情美滋滋的，命人给慕清霄准备住处后，便前往正殿召集其余四位长老商议妙丹堂的事情。
<br/>　　蜀山五长老，又称蜀山五老，是蜀山仙剑派第二十二代掌门时期修为和地位最高的五位长老。
<br/>　　分别是掌门清微，与苍古、净明、幽玄、和阳四位长老。
<br/>　　要说到这五位长老，那与剧情开端就脱不了干系，而且邪剑仙的诞生，完全就是因为蜀山这五位长老。
<br/>　　当年，蜀山五长老因为提升修为，将邪念排出体外，并封入锁妖塔中，造成了邪剑仙的诞生。
<br/>　　此后，邪剑仙逃出锁妖塔，五长老欲将其收服，清微在与邪剑仙交手的时候被其用镇妖剑刺死，其余四位拼劲余力维持锁妖塔结界，后被邪剑仙控制，最终全部殒命。
<br/>　　从一定程度上讲，因私心而终致大害，正表明世事因果循环之理。
<br/>　　“慕长老，弟子常胤，受掌门之命给长老准备住处，请随弟子来。”
<br/>　　面前粉面书生一样的青年，正是蜀山派前山大殿的守门弟子，擅于八卦各路消息的小道士。
<br/>　　“带路吧。”
<br/>　　“慕长老，蜀山共有七座供长老居住的宫殿，分别是天枢宫，天璇宫，天玑宫，天权宫，玉衡宫，开阳宫，摇光宫，各有不同……”
<br/>　　根据常胤的叙述，天枢宫是蜀山掌门的住处，其余四名长老，分别住在天玑，天权，玉衡，开阳宫中。
<br/>　　唯独剩下天璇与摇光两座宫殿，慕清霄身为妙丹堂长老，自然能在两座宫殿中选择其一。
<br/>　　“天璇与摇光，那座宫殿距离药园比较接近？”
<br/>　　常胤似乎听出了言下之意，微笑道：“天璇宫距离药园更近，而且旁边正是妙丹堂，似乎那里一直不曾有长老住过。”
<br/>　　如果慕清霄没记错，蜀山派祖师以前似乎就是住在天璇宫，因为太清真人擅长炼丹，也就没有舍近求远。
