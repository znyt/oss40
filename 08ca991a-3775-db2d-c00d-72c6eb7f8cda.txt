　　“握草！不是吧！不过就是突破金丹境而已，居然引来了雷劫？”
<br/>　　叶尘忍不住爆了一声粗口。
<br/>　　一般来说，修真之人，逆天而行，每当修为有了重大突破，都会引来天地之力排斥，从而天降神雷劈之，修真之人谓之雷劫！
<br/>　　不过，按照一般情况，化神之下的修士，实力太弱，是不足以引来雷劫的。
<br/>　　而如今，叶尘才仅仅只是踏入金丹境，竟然就引来了雷劫，不说绝无仅有，就算是在修真界，也是极为的罕见。
<br/>　　至少叶尘前世八百年，还从来没有听说过这种情况。
<br/>　　可此时此刻，哪里容叶尘多想？
<br/>　　咔嚓！
<br/>　　一道紫色神雷突然劈落下来，瞬间便狠狠击中了叶尘的头顶。
<br/>　　饶是叶尘如今，身体是何等的强悍，便是中小型核武都可以硬抗，居然险些被这道神雷劈懵！
<br/>　　而且没等叶尘缓过劲来，
<br/>　　咔嚓！
<br/>　　咔嚓！
<br/>　　紧跟着，又是一连两道神雷！
<br/>　　“琉璃金身！”
<br/>　　“法相天地！”
<br/>　　危机关头，叶尘顿时一个激灵，当即将两大神通施展了出来。
<br/>　　即便是如此，
<br/>　　一连两道神雷劈落，叶尘整个人也是七荤八素，险些晕厥了过去，而他的肉体，更是被活生生劈成了一具焦炭！
<br/>　　好在三道神雷过后，天空上的黑云便开始缓缓消散，否则若是再来几下，叶尘都没有信心可以撑下去！
<br/>　　好在，正式跨入金丹境之后，叶尘的恢复能力，也飞速提升，达到了一个极为恐怖的程度。
<br/>　　“滋滋滋~”
<br/>　　才不一会的功夫，原本被劈焦的肉体，便开始迅速的褪去，然后生出新的血肉。
<br/>　　片刻之后，
<br/>　　“彭！”
<br/>　　叶尘身体表面的那层焦炭，直接爆裂开来，而下方已经生出新的血肉肌肤。
<br/>　　此时的叶尘，浑身有玄光流转，皮肤白皙如玉，熠熠生辉，犹如谪仙降世。
<br/>　　叶尘脚下一顿，直接腾空而起，感觉不但自己体内的伤势已经好了大半，更有一种难以形容的力量感，充斥着身体的每一个细胞。
<br/>　　“哈哈哈！”
<br/>　　叶尘不由得哈哈大笑了一阵，继而一掌朝下方拍落！
<br/>　　轰隆！
<br/>　　这一掌落下，下方海面之上，登时便出现一道长达数百米的凹型掌印，足足数十米深！
<br/>　　彭！
<br/>　　海面上犹如被一枚核弹轰炸，瞬间炸裂开来，激起数百米高的海浪，朝远处滚滚汹涌而去！
<br/>　　“好强！虽然修为并没有提升太多，可是这金丹之力，比之前可要纯粹太多了！”
<br/>　　“我如今的实力，比之前至少强大了五倍都不止！倘若加上逆生百转的神通，便是遇到化神天君，也有一战之力！”
<br/>　　一时间，叶尘自信满满，眼眸中闪过一道炙热之色。
<br/>　　早知道凝聚出金丹之后，实力居然提升如此之多，他倒也不必摧毁传送阵了，像那段九德之流，就算不施展逆生百转，他也可以轻松灭之！
<br/>　　“罢了！事已至此，我还是回家要紧，等到蓬莱四大宗门将传送阵修复，必定不会善罢甘休……”
<br/>　　一念及此，叶尘缓缓闭上双眼，开始将神魂之力完全释放了出来。
<br/>　　“眼下要先找到人，问一下这里的具体方位！”
<br/>　　嗡~~
<br/>　　随着修为突破到了金丹境，叶尘的神魂力量也跟着水涨船高，足足可以覆盖方圆八百里的范围。
<br/>　　“十点钟方向，六百里左右，似乎有一艘游轮，去看看！”
<br/>　　叶尘有所发现之后，当即身体一晃，直接朝目标赶了过去。
<br/>　　唰！
<br/>　　天空一道残影划过，叶尘瞬间便已经跨越了数十里的距离。
<br/>　　他如今的速度也有所提升，全力催动之下，已经几乎可以达到二十倍音速！
<br/>　　不一会的功夫，叶尘便已经追上那艘游轮，
<br/>　　这艘游轮很破，船顶挂着一张骷髅图案的旗帜，船身上涂写着一些叶尘也不认识的文字。
<br/>　　“看这样子，倒有点像海盗船……”
<br/>　　叶尘心里暗自嘀咕了一句，也没有多想，他才懒得理会这艘船的来历，只要能问清楚这里的具体方位即可。
<br/>　　直接纵身一跃，便落在了游轮的甲板之上。
<br/>　　轰隆！
<br/>　　叶尘并没有刻意隐藏自己的气息，身体落在甲板之上，顿时发出一声巨响，连同整嗖游轮，都跟着一阵剧烈晃动。
<br/>　　很快，
<br/>　　由于叶尘动静太大，顿时便吸引了船上人员的注意。
<br/>　　蹬蹬蹬！
<br/>　　一群手持枪械、穿着古怪的人，从四面八方冲了过来，而且嘴里大喊大叫，不知道说的是什么语言，叽里咕噜的，叶尘一个字也没有听懂。
<br/>　　哗啦啦！
<br/>　　这些人冲出来之后，当即将叶尘包围在中央。
<br/>　　为首的是一名，满脸络腮胡的中年黑人男子，冲着叶尘又是叽里咕噜说了一通。
<br/>　　叶尘依旧一个字也没有听懂，索性直接用英语道：
<br/>　　“谁能告诉我，这里是什么地方？”
<br/>　　那中年黑人男子又是叽里咕噜一通。
<br/>　　叶尘不由得摇了摇头，也懒得再和他们浪费时间，索性直接朝驾驶舱走去。
<br/>　　游轮上肯定有地图和GPS定位系统，叶尘只要找到，自己就能确定自己的位置。
<br/>　　那中年黑人男子见叶尘如此不把他们放在眼里，顿时两眼一瞪，声音陡然提高了一个八度，又叽里咕噜说了一句什么话，而且满脸的杀意！
<br/>　　叶尘理都不理，继续缓步朝驾驶舱走去。
<br/>　　那中年黑人男子彻底怒了，直接冲着旁边大吼了一声，当即便有三名黑人青年站了出来，拿起手中的步枪，对着叶尘开火了！
<br/>　　砰砰砰！
<br/>　　一连三发子弹射出，全都射向叶尘的要害位置。
<br/>　　可惜，连核武都伤不了叶尘，更何况区区子弹？
<br/>　　那三颗子弹连叶尘身前三米都无法进入，就被叶尘周身的护体真元，自动弹飞了出去。
<br/>　　噗噗噗！
<br/>　　开枪的那三名黑人青年，瞬间便被自己打出的子弹击毙。

<br/>　　以他们普通人的眼力，自然看不出其中端倪，还以为对方会妖术，全都吓得不轻。
<br/>　　那中年黑人男子愣了片刻，当即又大吼了一声。
<br/>　　众人纷纷抽出腰间的匕首，便又朝叶尘扑了过来。
<br/>　　叶尘不由得摇了摇头，脚下步子不停，随手弹出几道剑气，
<br/>　　以叶尘如今的实力，便是天仙一级的武者，都可以轻易斩杀，更遑论这些普通人？
