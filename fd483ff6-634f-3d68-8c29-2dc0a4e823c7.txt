        “住手！”

<br/>        谁知另一只强劲有力的大手，像是不费吹灰之力般擒住了季运恒的手腕，将他的这一巴掌，拦在了半空中。
<br/>
<br/>        傅夜沉黑眸深邃，薄唇微启，淡漠地开口：“放开她。”
<br/>
<br/>        季运恒身心一怔，双眸微抬，惊愕地对上傅夜沉的目光时，莫名其妙地感到不寒而栗。
<br/>
<br/>        “你、你是谁？敢管本少的事！”
<br/>
<br/>        季运恒咽了咽口水，在傅夜沉面前嚣张的气势却变得格外畏缩。
<br/>
<br/>        傅夜沉黑眸微眯，目光狭长，淡漠的薄唇，勾唇冷笑：“呵？”
<br/>
<br/>        一刹那，季运恒就缩成了耗子，做贼心虚地甩开了童筱颖的手。
<br/>
<br/>        傅夜沉见季运恒放开了童筱颖，同时也松手放开了他。
<br/>
<br/>        童筱颖趁机挪步，远离季运恒。
<br/>
<br/>        季运恒心里忽然有点儿不甘心，抬腿就往傅夜沉身上踢去。
<br/>
<br/>        而傅夜沉是练过身家底子的，季运恒这三脚猫的功夫，在他眼里压根就不算什么。
<br/>
<br/>        童筱颖惊呼得叫了声：“小心！”
<br/>
<br/>        傅夜沉一个眼疾手快，完全让季运恒打个措手不及，防不胜防地就抓住季运恒的腿，来了个过肩摔。
<br/>
<br/>        “啊——”季运恒被重重地摔在地上后，痛得打起了滚来。
<br/>
<br/>        他下意识地滚到了童筱颖的脚边，童筱颖几乎是条件反射般地连忙躲在了傅夜沉的身后。
<br/>
<br/>        “你、你……”季运恒在地上痛得欲哭无泪。
<br/>
<br/>        “滚！”傅夜沉凝眉呵斥。
<br/>
<br/>        季运恒不得不从地上爬起来，一边揉着自己的屁股，一边指着傅夜沉的鼻子破口大骂：“小子，你给本少等着！”
<br/>
<br/>        “还不滚？”傅夜沉眸光微沉。
<br/>
<br/>        季运恒见这男人一脸阴晴不定像是要拆了他骨头的样子，吓得连滚带爬地往那边的电梯口逃去。
<br/>
<br/>        走廊上瞬间安静了下来。
<br/>
<br/>        这里是“夜惑”的安全通道里，外边的聒噪，与这里的安静，恍若是两个不同的世界。
<br/>
<br/>        傅夜沉微微侧身，微微眯起狭长的黑眸，低头淡漠地注视着身边这个戴着白羽假面眼镜，看起来有些狼狈的女人。
<br/>
<br/>        只见她长发凌乱地搭落在肩窝上，白皙的脖颈上有几处暗紫绯红的吻痕。
<br/>
<br/>        他的视线一路而下，看到她大腿上还有几道淤青，顿时一脸不悦地皱起了眉头。
<br/>
<br/>        童筱颖见这男人正在打量着自己，下意识地双手护胸，往后退了一步，并拢双腿，垂下眼帘，弱弱地说了声：“谢谢。”
<br/>
<br/>        忽然，一抹灰影掠过余光。
<br/>
<br/>        童筱颖下意识地抬眸，只见面前这男人将一件银灰色的西装外套，披在了她的肩膀上。
<br/>
<br/>        傅夜沉凝视着面前这个楚楚可怜的女人，收回双手时，指腹在她的脸庞边停顿了一下，才放下双手，淡漠地转身离去。
<br/>
<br/>        刚刚……
<br/>
<br/>        他是怎么呢？
<br/>
<br/>        为什么会有一种想捧着她的脸吻她的冲动？
<br/>
<br/>        为什么会有一种想把她搂入怀中好好疼爱的冲动？
<br/>
<br/>        明明那女人不是“漫雪”……
<br/>
<br/>        傅夜沉心思深沉地离开了安全通道，留下童筱颖愣在原地对他匪夷所思。
<br/>
<br/>        他……
<br/>
<br/>        应该是没有认出她来。
<br/>
<br/>        她脸上还戴着白羽假面眼镜。
<br/>
<br/>        童筱颖暗自松了口气。
<br/>
<br/>        回到自己的休息室里后，童筱颖脱了身上的衣服，进了卫生间，冲了个澡。
<br/>
<br/>        她要洗去身上那些别的男人的味道……
<br/>
<br/>        季运恒碰过的地方，她一一搓了七八回。
<br/>
<br/>        洗完澡后，童筱颖换回了自己的衣服，并坐在梳妆镜前，拿起粉底，来遮掩脖子上的吻痕以及大腿上的淤青。

<br/>        回家前，她将那件外套递给了唐倩兮，让唐倩兮帮忙还给那个好心的男人。

<br/>
<br/>        “回头陪我去健身房练练！”傅夜沉看了一眼连华生那憋屈的脸，忍俊不禁道。
