　　楚风眠凝聚远古战龙之力，一爪猛然轰击而下。
<br/>　　这阿修罗的恐怖魔意，在这战龙之爪的面前，被毫无悬念的被碾压。
<br/>　　如今楚风眠身上的力量，虽然是神力境极限的地步，但是单纯以力量上来将，楚风眠的力量甚至都是要超越了生死境高阶的强者。
<br/>　　一般的生死境高阶的强者，甚至都不是楚风眠的一招之敌，在这战龙之爪的面前，都会被瞬间镇压。
<br/>　　他的实力，已经接近了生死境巅峰的地步，只要是楚风眠一旦是度过生死大劫。
<br/>　　楚风眠的实力都有可能会超越生死境巅峰，踏入到传说中生死台阶九重的地步。
<br/>　　“给我镇！”
<br/>　　楚风眠大手一挥，又是一道战龙之爪直接镇压下去。
<br/>　　刚刚那一爪，已经是将这阿修罗身上的魔意，全部打碎。
<br/>　　而这一爪的目标，则正是这阿修罗本身。
<br/>　　“可恶，这到底是什么力量！我感觉到，此人还没有踏入生死境，神力境就有如此实力，怎么可能！”
<br/>　　阿修罗连连后退，脸色也尽是不可置信。
<br/>　　他的魔意，居然被全部打碎。
<br/>　　“可恶！我可是阿修罗，天魔一族中的王族，怎么会败在一个神力境小子的手中！魔意附体！远古魔身！”
<br/>　　阿修罗狰狞的大吼一声，他的身躯，身上的鳞片，头上的尖角，都在进行这蜕化。
<br/>　　这一步步蜕化，令阿修罗身上的气息都是变的强大许多，身上的魔意也变得更加古老。
<br/>　　这阿修罗的血统，居然是极为纯粹，如今凝聚魔意，逐渐的化身成为了远古阿修罗的样子。
<br/>　　现在的阿修罗一族，只是远古阿修罗的血脉遗留，力量是不足一。
<br/>　　曾经远古时代，诸天天魔，席卷而下，单单是阿修罗一族，都足以摧毁一方庞大势力。
<br/>　　“小子，给我死吧！”
<br/>　　阿修罗猛然长啸一声，手中骨剑，再次挥动，无数魔意，席卷这他的骨剑，一剑挥出。
<br/>　　这骨剑挥出的，乃是玄妙无比的剑术。
<br/>　　阿修罗一族，向来都是剑道高手，传承剑道，这种传承比起现在北域中的武道世家还要厉害。
<br/>　　骤然之间，这一剑已经是向着楚风眠斩杀过来，万千剑影，令人捉摸不清，无法抵挡。
<br/>　　这一剑中蕴含的力量，显然是必杀之剑，要将楚风眠彻底斩杀的一剑，是这一尊阿修罗的剑道，发挥到了淋漓尽致。
<br/>　　这一剑，让一尊生死境高阶的强者来，也要被斩杀。
<br/>　　“小心！”
<br/>　　白萱急忙出声提醒道，这一道剑术，简直玄妙无比，还要超越了白萱的剑术。
<br/>　　起码白萱在这剑术面前，只怕是不堪一击，极有可能就会被直接斩杀。
<br/>　　只可惜，这阿修罗如今遇到的，却是碰到了剑道的祖宗。
<br/>　　“这阿修罗一族的剑术，倒是有些门道。”
<br/>　　楚风眠看到这阿修罗的剑术，心中也是不由的感叹一声。
<br/>　　剑术，并非是人类一族的特权，像是阿修罗一族，都是用剑的高手。
<br/>　　而在龙族中，也出现过颜古龙帝这样的人物。
<br/>　　这阿修罗一族的剑术，对楚风眠来说，也有着不小的参考，真正至强的剑术，必须要以取各家所长才行。
<br/>　　“破！”
<br/>　　楚风眠口中只是淡然吐出一个字，在他的手中，祖龙至尊剑猛然出窍。
<br/>　　剑锋一点，化为了一道剑气，猛然斩了过去。
<br/>　　这一股剑气上，蕴含有着楚风眠恐怖剑意，席卷而下，顿时阿修罗的剑锋，被瞬间剿灭。
<br/>　　“阿修罗，死吧！我将你的身躯跟骨剑融为一体，锻造出一把新的灵器！”
<br/>　　楚风眠大手一挥，一股灵力抓了下去，将这阿修罗跟骨剑，同时抓了起来。
<br/>　　“碰！”
<br/>　　随着楚风眠手一捏，这阿修罗的身躯便是开始扭曲起来。
<br/>　　“不，你休想炼化我！我是尊贵的天魔王族！阿修罗！渺小的人类怎么可能炼化我！”
<br/>　　从阿修罗的身上，无数魔意肆意爆发出来，种种武技，连连打出，在疯狂的挣扎着。
<br/>　　妄想是从楚风眠的手中逃出。
<br/>　　这是真正的在拼命了，阿修罗，本就是天魔一族中完美的战士。
<br/>　　在这拼命的时刻，爆发出全力，也是恐怖到了极致。
<br/>　　这无数轰击轰击出来，居然是隐隐约约，要从楚风眠的手中挣脱出来一样，想要逃出楚风眠的手掌心。
<br/>　　“无用功，乖乖被我炼化吧！”
<br/>　　楚风眠冷漠的声音响起。
<br/>　　这阿修罗，无论的如今如何挣扎，想要从楚风眠的手中逃出去，也绝无可能。
<br/>　　从楚风眠的身躯中，战龙血脉的力量再次被催动，一下子落下，这阿修罗便是被直接炼化，镇压。
<br/>　　他的任何反抗，都被死死的镇压。
<br/>　　“轰！”
<br/>　　这阿修罗的身躯，直接在这力量面前，被碾碎。
<br/>　　其中的无数魔血，顿时是跟他手中的骨剑，融为一体，开始疯狂燃烧起来。
<br/>　　楚风眠要以这阿修罗的血来祭炼这骨剑。
<br/>　　这骨剑，本就是阿修罗以本命精血祭炼出来的，如今楚风眠将这阿修罗的所有精血都给燃烧融入其中，将这骨剑升华。
<br/>　　最终这骨剑的样子，变化的更加狰狞，其中上面一股股恐怖至极的魔意，令人感觉到，都不由的汗毛林立起来。
<br/>　　这骨剑中，融入了这阿修罗终生的武技，他的所有魔意，都被楚风眠融入到了这其中。
<br/>　　简直便是一把有灵之剑。
<br/>　　“这把骨剑，就送给你了，你还差一件趁手的兵器，这骨剑中也蕴含有着阿修罗独特的剑术，你也可以仔细参悟一番。”
<br/>　　楚风眠看着白萱，大手一挥，这骨剑直接扔向了白萱。
<br/>　　白萱接过，脸色大喜。
<br/>　　阿修罗一族的骨剑，本就是极为强大的灵器，足以比起地级灵器还要强大。
