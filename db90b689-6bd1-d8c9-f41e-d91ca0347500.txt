　　第197章
<br/>　　大战过后的幽州，百业凋零，一切等待新生。
<br/>　　从官道之上，行来一队人，一支百人伍的骑兵队伍，中间是一辆四匹马拉着的大撵。
<br/>　　兵丁人强马壮，身上是整齐的黑色皮甲，座下马一根杂毛也无，保持统一的黑色。
<br/>　　要说江南并不缺少豪富，有些富豪出门的派头，甚至比这更大。然而，这支队伍高高在上的贵气，绝不是江南水乡那种柔软之地可以产生的。于是这些人的来历也就可以轻易猜出来，他们必定来自京城。
<br/>　　大撵内坐着两个人，一黑袍长须之人道：“倒是麻烦刘公公走这一遭，一路颠簸，刘公公受苦了。”
<br/>　　“都是给陛下办差事，哪有什么苦不苦，是咱家的幸事才对。”杂色袍子的人拉开轿帘向外看了一眼：“戎族作乱，幽州的百姓受苦了。”
<br/>　　“哎……”黑袍男人轻叹一声，语气也十分无奈。
<br/>　　“崔家选得好女婿啊。”杂色袍子的人忽然笑了笑，笑得颇有深意。
<br/>　　“哈，哈，为国为民，本该如此。”黑袍人。
<br/>　　还未入城，前方道路上便有迎接之人，一乘小轿停在路边。杨龙停向轿外张望：“人来了没有。”
<br/>　　“主公稍安勿躁，探子回来的消息，距咱们还有十里地。”
<br/>　　“十里地也快了，我该下轿迎着。”
<br/>　　“主公，不必如此吧。”
<br/>　　“你有所不知，这次传旨的太监是明帝的贴身之人，在明帝耳边一句话，就能定人祸福。”
<br/>　　“是那位内廷刘阿吉？”
<br/>　　“正是他。”
<br/>　　“他那样的身份，这次肯跑这么远过来，受这舟车劳顿。”李善言一惊。
<br/>　　车队过来，远远就能看到旗帜飘摇。杨龙停快步过去，冲着车队深施一礼。
<br/>　　“幽州王杨，恭迎刘公公，崔大人。”
<br/>　　若对一个太监，幽州王是不用行礼的，对方再是明帝的红人，身份上也大不过他这一方诸侯。主要是对方如今代表的是明帝。
<br/>　　“不必如此，不必如此。”刘阿吉快步下来，弯着腰扶起杨龙停：“王爷是帝国功臣，这是折咱家的寿呐。”
<br/>　　孙德长身边的男人姓崔，名叫崔云间，他冲杨龙停点点头，却没有说什么话。
<br/>　　李善言打量着刘阿吉，老实说，他对太监还是有些好奇的。只见刘阿吉并没有传说中太监那种猥琐的模样，反而相貌堂堂，言谈举止令人如沐清风。
<br/>　　“果然，能站在帝国顶端的男人都不是简单人物呐。”李善言心里嘀咕一句：“虽然对方不是男人。”
<br/>　　“刘大人，城里都已准备好，咱们到城里说话。”
<br/>　　“王爷请……”
<br/>　　“刘大人请……”
<br/>　　“王爷请……”
<br/>　　客套是很麻烦的事情，单单在谁先行的事情上，二人就耽误了很长时间。进得城来，事情更是麻烦，酒宴招待一概不能少，而吃饭时又有颇多规矩。
<br/>　　过程中，杨龙停把李善言叫到一旁：“我让你准备两个十四岁的秀丽女子你准备好了么，今夜送到刘公公房里伺候。”
<br/>　　“已经准备好了。”李善言撇撇嘴：“只是一个太监，也喜欢女人？”
<br/>　　“太监为什么不可以喜欢女人，谁告诉你太监不喜欢女人，你当过太监嘛，你有什么发言权。”
<br/>　　杨龙停打仗时下体受了一些伤，不是很喜欢和李善言聊这个。
<br/>　　一番忙碌，过程将其复杂，直到第二天中午，刘阿吉才算是喜净风尘，与杨龙停聊起了正事。
<br/>　　大厅内，除了杨龙停，还有落叶城城主方伯山，赤沙城城主董千玉，北镇城城主石幕西。
<br/>　　刘阿吉抿了一口茶，缓缓道：“诸位，也不需这么严肃，咱家这次来，只是把圣上的意思传达给大家，诸位都是对帝国有功之臣，其他一应礼数免了就是。”
<br/>　　“这次戎族乱我帝国，虽说最后被赶跑了，但百姓却是民不聊生。尤其幽州一地，更是直面戎族最强大的北蛮部，最后能守住这里，在座诸位居功至伟。”
<br/>　　“为国为民，份内之事而已。”杨龙停。
<br/>　　“好，好，好！”刘阿吉道：“这三个好，可不是我说的，而是陛下亲口说的。王爷，你知道陛下如何夸赞你么？”
<br/>　　刘阿吉笑了笑，道：“这边的战况传过去后，陛下击节赞叹，说若帝国都是王爷、方城主这样的虎将，那戎族又有何惧之有。”
<br/>　　方伯山一楞：“陛下也知道我？”
<br/>　　“经常提起呢。”
<br/>　　今天的事情，杨龙停是知道点消息，而方伯山是刚从青州回来，此刻完全是一头雾水。
<br/>　　帝国与戎族这一战，可以说一败涂地。而这个时候，偏偏有个幽州，偏偏有座青牛山，成了戎族的绞肉机，抗击戎族三月之久不说，还全歼北蛮部十万大军。如此，在整条战线都被人扒掉裤衩的情况下，青牛山这场胜利有多重要也就可想而知了。
<br/>　　方伯山不明白的是，这件事和自己有什么关系。
<br/>　　其实这件事也怪程大雷，他成天一个暴发户的样子，腰上挂三枚帅印，见人就显摆：我是幽州王亲封的游骑将军，落叶城城主……和我一起打仗去吧。
<br/>　　其实很多人都不明白怎么会事，便以为是杨龙停和方伯山派遣得他，等这件事传到京城，也就成了杨龙停和方伯山死守青牛山，歼敌十万。
<br/>　　“陛下说了，该罚的要罚，该赏的要赏，让天下百姓看看，帝国也有英雄，对于英雄怎么赏赐都不为过。这次陛下定了六大功臣，王爷居六大功臣之首。朝廷正在定日子，到时候王爷还要去一趟京城，也在天下人面前讲讲，您是如何抗击戎族的，教教其他人该怎么打仗。说真的，咱家离京时，京城的史官正为您著书立传呢，哈哈。王爷你可是一战扬名，天下尽知呐。”
<br/>　　“啊，这个，哈哈……本王拙口笨舌，只知埋头为陛下分忧，为百姓解难，至于这说话……呵呵，实在的不擅长呐。”
<br/>　　“王爷过谦了。”刘阿吉抿口茶：“还有一件小事，程大雷是那个，咱家想见见他。”
<br/>　　
<br/>(本章完)
