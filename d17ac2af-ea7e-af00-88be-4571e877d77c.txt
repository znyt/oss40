　　四宝嚷道：“蠃鱼？山海经里那个？”
<br/>　　徐文长点头，道：“不错，山海经有云：蠃鱼，鱼身而鸟翼，音如鸳鸯，见则其邑大水。便是它了。”
<br/>　　叶少阳顿时两眼发直，四宝笑着在他肩膀上拍了一巴掌，“中彩了啊。”
<br/>　　上古异兽，光这四个字，就足以说明实力了。
<br/>　　“我怎么没听说过什么四大教主？”吴嘉伟问。
<br/>　　“有的，只是太阴山一向神秘，他们的层级建制，我们并不了解，关于四大教主，我们了解也不多。”徐文长意味深长地说道，“阴司最大的弱点，其实就是这个。”
<br/>　　“什么意思？”叶少阳问。
<br/>　　“无极鬼王的身份，你们也都知道了，他曾经是大帝门下弟子，在阴司呆过百年，对阴司的一切都了如指掌，可是我们对太阴山却是一无所知，不知道太阴山在哪，不知道他们有多少人，又有多少强者，否则，怎么会让他们一步步发展壮大到能与阴司抗衡的地步。”
<br/>　　林三生点头道：“这是没错，俗话说知己知彼，现在人家对我们是自己知己知彼，这就比较危险了。”
<br/>　　叶少阳沉吟半晌，问徐文长：“我一直有个疑问啊，无极鬼王曾是大帝门人，大帝看人是很准的吧，为什么没有看出他有问题，完全没提防他？还有，在鬼王出去自立门户之后，大帝难道没有手段来遏制他吗，为什么任凭人家发展起来？”
<br/>　　徐文长叹了口气，苦笑道：“他在大帝门下时，是最好的弟子，资质极高，悟性也极强，正是他悟性强，所以想的多，他喜欢跟大帝论道，虽然往往有不同意见，但大帝只当是辩论，并不打压，他与大帝的理念不同，却越走越远，最终离开阴司，在鬼域流浪。
<br/>　　到这个时候，大帝对他还是有包容之心的，哪里知道，鬼王得了奇遇，开创了太阴山……之后的事，你们也知道了，太阴山离阴司太远了，中间隔着很多割据势力，阴司当时不可能也没打算去征讨他，等成了气候之后，一切都晚了。”
<br/>　　大伙听了这番陈年往事的讲述，也是感慨不已。
<br/>　　叶少阳问道：“还有个问题。鬼王……到底有多强？”
<br/>　　徐文长目光深邃地望着远处，道：“道门七绝之中，他对道的理解最深，天赋之强，亘古未有。”
<br/>　　“比道风还强？”
<br/>　　“这个……道风的天赋，在于个人修行，在这方面的天赋，无人能跟他相比，但鬼王不同，鬼王追求的，是洪荒宇宙的终极奥义，大概就是你么人间的哲学吧，可他偏偏在修行上又有极高的天赋，因此他不光是哲学家，还是个行动派，他这种人，要么是帝王，要么就是天下最大的反贼……”
<br/>　　林三生道：“我觉得，我能理解他，成王败寇，假如真让他统一了鬼域，三界也一定还有秩序，只不过是另一套秩序了。”
<br/>　　徐文长道：“这话你可以说，我不能说。阴司千百年来不断调整，才有今天的制度，他不一定是最好的，但一定是最适合三界生灵的！别的不说，如果让鬼王推翻阴司，再立规则，三界生灵要付出多么惨重的代价？”
<br/>　　叶少阳摊了摊手道：“这话你应该跟鬼王去说。”
<br/>　　徐文长冷哼一声：“鬼王若是听劝，哪里会有今天！”
<br/>　　大伙也就不说什么了。
<br/>　　“你这包里装的什么？”叶少阳看他今天背了一个很大的褡裢，问道。
<br/>　　“待会就派上用场了。”
<br/>　　正聊着，徐文长袖子里突然着起火来，他用力一抖，一张燃烧的灵符飞出来，立刻正色说道：“岳家军过桥了，我们也出发吧！”
<br/>　　当下打开虚空裂缝，钻了进去。大伙紧随其后。
<br/>　　跟昨天一样，还是从涅槃禅杖中飞出去，落在城南枉死城中。
<br/>　　城中的守卫，比昨天要多，牢牢守在烽火台四周。
<br/>　　大伙先到烽火台上，朝外面望去，围城的鬼兵比昨天只多不少，在城外一圈圈地转着圈。
<br/>　　“他们也不进攻，光这么围着是什么意思？”叶少阳不解问道。
<br/>　　“这叫围点打援，他们知道强攻不下阴司，只是不断施压，让那些驻外军队回兵来救，如果来救，就被内外夹击，如果不来，那兵力越来越多，早晚能强攻下酆都城。”徐文长解释。
<br/>　　林三生评价道：“从鬼王这一手安排来看，他是有谋略的，在下一盘大棋。”
<br/>　　时间紧要，大伙不敢耽误，在徐文长的带领下飞上南城楼，然后按照之前设计的战术，小九现出真身，背着叶少阳，率先冲了出去，大伙紧随其后。
<br/>　　从结界出去，立刻有黑压压一片的鬼兵奔袭过来，黑压压的一群，叶少阳还是第一次见到这么多敌人，心中暗骂了一声卧槽，摸了一把准备好的灵符，凌空打出去。
<br/>　　小九口中喷出妖风，首当其冲的鬼兵被妖风吹化了不少，后面的也被吹飞，小九落地之后，再次跳起，这时候鬼兵又蜂拥而来，说是鬼兵，其实不光是鬼，也有不少妖和邪灵，如蝗虫一般铺天盖地，没有一处死角，只能硬闯。
<br/>　　以小九的实力，对付这些普通邪物没什么问题，但中间也有一些实力不错的，就要费点事，好在叶少阳他们这边人也多，又提前分配了位置，有前锋有防御有侧翼，大家只要一往无前地冲，不用去管什么，大概用了一刻钟时间，总算冲出了包围圈，回头望去，精魄横飞。
<br/>　　“咱杀了有多少个？”叶少阳问。
<br/>　　“不知道，得有……几百个？”四宝接了一句。
<br/>　　突围之后，大伙一路向南，顺着荫水河走，再没遇到什么抵抗。
<br/>　　这一走就走了有一两个小时，直到山也看不见了，放眼四周是一块巨大的平原，河流也越来越窄，然后分叉，就像血管一样，遍布整个平原。
<br/>　　大伙不得不从这些溪流上跨过去，有时候会惊起一些水中的邪物，纷纷被他们击毙。
<br/>　　叶少阳几次询问还有多远，徐文长总回答快了快了，又往前走了不知道多远，前方彻底没路了，只有一片汪洋大海，但是没有风浪，水面平静地像一面镜子。
