　　第192章
<br/>　　程大雷立在山头，众家首领一一上山。山梯修得十分狭窄，只能由一个人先爬上去，及到黄昏，才有十几人爬上山头。
<br/>　　“高飞虎，高飞豹。”程大雷开始发令。
<br/>　　“大当家。”
<br/>　　“命你们二人带五百兄弟，护送山寨得老弱妇孺撤离。”
<br/>　　“我们要撤？”二人是第一次知道程大雷得心意。
<br/>　　“守不住，只能让老弱先走。”程大雷道：“你们从这里下山，到港口找船下海，去南方，那里安稳些。”
<br/>　　“山外都是戎族，想走怕是不太容易。”刘悲叹口气道。
<br/>　　“所以要分批走，夜里走，慢慢走。”程大雷。
<br/>　　“慢慢走？他们还会给我们时间么？”刘悲又叹口气。
<br/>　　“会有时间得。”程大雷忽然转过身，冲关鱼道：“二爷，我还需要你带五百兄弟去做一件事……”
<br/>　　“若我们一走，山寨只剩下不到一千弟兄，戎族若是攻来，咱们还守得住么？”刘悲一直叹气。
<br/>　　“我会想办法拖住他们，为大家争取时间。”程大雷说罢，看着刘悲道：“你再叹气我揍死你信不信。”
<br/>　　刘悲张张嘴，没有说话，心里想的却是：你一个人怎么守？
<br/>　　“报……”一声长呼，一个喽啰跑到山下，大声道：“大当家，戎族大兵集结，看情形随时准备打过来。”
<br/>　　“大当家……”
<br/>　　刘悲和徐神机几人齐呼：
<br/>　　“大势不可逆，大家一起走吧，能走出几人是几人。”
<br/>　　“莫怕。”程大雷立在最高处：“今日，且看本当家为你们呼风唤雨。”
<br/>　　……
<br/>　　戎族的九家大军重新聚在一起，兵如狼，将若虎，兵雄将猛，十万虎狼之师集结，旌旗蔽空，戈甲连野。
<br/>　　大帐内，充斥着愉悦欢快的空气，在友好活泼的气氛中大家交流着最近几场大战的心得。
<br/>　　“要说打得漂亮，还是铁石一战，日落之前便攻破了城门，城门一破，南人就没了抵抗力，只知道像绵羊一样逃命。”
<br/>　　“铁石城一共才杀多少人，还是和青州兵打得过瘾，他们还想和我们骑兵对冲，就该教教他们做人。”
<br/>　　“没人能与我们的战士在马上抗衡。这次血洗落叶城，可是抢到不少东西。”
<br/>　　众人一边大口灌酒，一边欣赏大帐内女姬的歌舞，这些女人都是从幽州抢来的。会议，就在如此热闹的气氛下进行。
<br/>　　“那么大家想想，明天该怎么打蛤蟆寨？”大王子发言。
<br/>　　热闹的气氛冷了冷，但随之便喧哗起来。
<br/>　　“十万人一人一口吐沫，也把他淹死了，还用想什么法子。”
<br/>　　“咱们大军扑过去，估计对方就吓尿了裤子。”
<br/>　　金问道看着大帐内的热闹，一直没有说话，也一直没有喝酒。
<br/>　　所谓越炫耀什么，越代表缺少什么。众人不断炫耀战绩，其实是想给自己建立自信心。如此也变相证明，他们没有多少自信。
<br/>　　的确，大家在幽州攻城略地，帝国军见到戎族不是望风而逃，便是瑟瑟颤抖。那种感觉告诉大家，我们攻无不破，战无不胜，我们的骑兵天下无双。
<br/>　　我们，是天生的战士。
<br/>　　可是，一旦进入青牛山范围，大家好像坠入一种奇怪气场，身上都带了负面Buff，智商被削弱，武力被削弱，运气值减半。
<br/>　　蛤蟆寨并不好打。
<br/>　　凡是和程大雷交过手的人都明白这点，而在所有人中，金问道体会最深。初次交手一战，程大雷以城门为铒，诱敌深入，将瓮城化作一口大锅，一锅煮了金问道五千大军。
<br/>　　以城为锅，以杀戮为火，以尸体和鲜血熬成的一锅汤，世间几人有勇气喝下这口汤？
<br/>　　那尸山血海的一幕，常常在金问道梦中出现，他还算强大的，意志稍弱者便精神崩溃，之后便再也握不住刀。
<br/>　　“明日攻城，此战必是血战，还请诸位……”
<br/>　　“报……”探子闯入大帐，打断大王子的话：“禀，蛤蟆寨城头点着篝火，不知道他们在做什么？”
<br/>　　“咦！”
<br/>　　所有人都皱起眉头，这是做什么，因为守不住了，便要玩火自焚。
<br/>　　几家王子离开大帐，来到蛤蟆寨城门前。
<br/>　　隔着老远看到程大雷立在城头，披头散发，身穿大袍，手里握着一柄剑，围着篝火便唱便跳，嘴里嘀嘀咕咕不知在做什么。
<br/>　　“他在搞什么？”
<br/>　　所有人都是一头雾水，大王子说这话时看着金问道，他和程大雷接触稍微多些。
<br/>　　“我也搞不懂呐。”金问道呐呐一句，忽然一惊：“他是不是在求雨？”
<br/>　　“求雨，他还有这本事！”大王子一惊。
<br/>　　“确实是见他这么做过，但究竟他会不会法术我也不太清楚。”
<br/>　　“去一个人看看，听听他在念什么咒语。”
<br/>　　“小心暗箭。”
<br/>　　有一探子冒着生命危险潜到城下，侧耳倾听，其实离得太远也听不太清楚，模模糊糊大概是：
<br/>　　“苍茫的天涯是我的爱
<br/>　　绵绵的青山脚下花正开”
<br/>　　等探子打探回来，把消息禀报给几人，几人都皱起了眉头。
<br/>　　“弯弯的河水从天上来……”金问道一惊：“他是在祈雨！”
<br/>　　众人心里都有些发毛，法术这种事还是有些瘆人的。大王子摆摆手，压住众人的议论。
<br/>　　“哼，装神弄鬼，还真能被他求下雨来。大家莫被他哄了，明日攻城不变，一定要打破蛤蟆寨。”
<br/>　　留下几个人盯着程大雷，其他人散去，准备明日的攻城。但心里都留下种子，隐隐约约，感觉好像又落入那种奇怪的氛围中。
<br/>　　次日天亮，大军整备待发，投石机，云梯等攻城器械在前，盾兵在后，骑兵整装待发，大军乌压压一片，一眼望不到头。
<br/>　　城头，程大雷还在转圈，口中嘀嘀咕咕。
<br/>　　“哼，死到临头，任你装神弄鬼也改变不了什么。”大王子骑在马上，大手一挥：“进……”
<br/>　　咔嚓！
<br/>　　忽然一声雷鸣，乌云汇聚，伴随着电闪雷鸣，滂沱大雨倾盆而下。
<br/>　　大雨哗哗打在身上，烈马被惊雷吓得长鸣，兵慌马乱，大军难发。
<br/>　　大王子一句话憋在胸口，手尴尬的杵在半空中，愣了半晌：“收兵。”
<br/>　　终于，大家再次溺入那种奇怪的氛围中。
<br/>　　青牛山的雨季到了。
<br/>
