<br/>    不过一会儿，苏青阳和刘芒和是赶到了此处，然后他们就看到了在周天星辰大阵之中，正蹲坐在一缕星光之前的陈明。“宗师这是在干嘛？”

<br/>        雪自悟道：“原来有丹药真的可以为所欲为！天劫头上过，我自磕丹药！”

<br/>        苏青阳道：“只是这么渡天劫，也未免太奢侈了，这一场天劫渡下来，怕不是要数千万灵石的丹药吧？”

<br/>        紫霞仙子道：“就是我等，也很难弄到这么多的灵石，更何况，还要全部买到丹药吗，这根本不可能。也唯有他们这种炼丹师，才能做到的。”
<br/>    
<br/>        刘芒道：“我曾经听说药王谷的药王，渡劫之时就是吃了价值九千万灵石的丹药，强行扛过了整场天劫！”
<br/>    
<br/>        四人对视了一眼，炼丹师，都是狠人啊，吃灵丹都不带眨眼的！
<br/>    
<br/>        人王们对于陈明这种诡异的渡劫方法，也是差点惊掉了一地的眼珠子，“原来天劫，还能这么渡的？”
<br/>    
<br/>        “舒服倒是挺舒服，老道诗兴大发，决定吟诗一首，袖中丹药千千万，云淡风轻渡天劫。天劫之威千夫惧，俯首甘为丹药奴！”

<br/>        “这位道友，当真是吟的一首好诗啊！”
<br/>    
<br/>        “只是这个方式，实在是学不来啊，莫说这么多的丹药，就是数千万灵石，也得攒几辈子。”

<br/>        魔宗七劫这个时候是彻底傻眼了，“渡天劫还带你这么玩的啊！”
<br/>    
<br/>        “他这分明就是耍赖！”
<br/>    
<br/>        “渡天劫乃是与天争命，求取一线生机，他倒好，直接变成了丹药展示环节了！”
<br/>    
<br/>        “怪不得炼丹师，阵法师，卦师，炼器师这么少，都这么奇葩的吗？”
<br/>    
<br/>        陈明此刻却是懒得管其他人的议论，全身心的投入了观察周天星辰大阵之中，反正丹药还多，不急，我们慢慢来，甚至于，陈明还会因为这星光变化太慢，担心天劫过去，时不时挑衅一下劫云。
<br/>    
<br/>        魔宗七劫已经感受到了情况不对了，“按照他这个渡劫法，只要丹药不枯竭，绝对能够撑过去！”
<br/>    
<br/>        “他要是撑过去，那就是一位新的大君，嗯，我怎么感觉到周围的这些正道的眼光不太对！”
<br/>    
<br/>        “师兄师弟们，快走，此地不宜久留！”
<br/>    
<br/>        魔宗七劫迅速撤走，等到陈明渡完了天劫，那可就有机会收拾他们了！
<br/>    
<br/>        陈明继续灌下一瓶丹药，云淡风轻的走向下一缕星光，周围的人王也是眼中精彩连连，陈明渡劫，似乎并没有任何危险啊！

<br/>        等到一天之后，陈明已经是观察了四十二处星光，只剩下四十六处星光还未观察完，但是这些观看陈明渡劫的人王们，却是看得有些腻了，紫霞仙子道：“早就知道宗师修仙路彻底跑偏了，你说我们为什么要来看他渡劫呢？”

<br/>        苏青阳道：“好像是没有任何用处啊，他这个渡劫方式，我们完全学不到任何东西啊！”

<br/>        刘芒道：“我们也是快要突破大君的人，本来以为宗师渡劫，我们能够从中学到什么东西，没有想到，居然是这般结果！”
<br/>    
<br/>        雪自悟伸手捂额道：“算了算了，懒得看了，这看起来没意思啊，本来是好好的渡天劫，与天争命，热血沸腾的事情，现在看起来一点热血的感觉都没有啊！青阳道友，不如我们来对弈一局，等总是渡劫完毕，如何？”
<br/>    
<br/>        苏青阳道：“不错，对于安排众多人王反攻的事宜，就交给紫霞仙子了，我们俩就来对弈一局吧！”

<br/>        可能是因为的缘故，这些人更倾向于将这一切都交给紫霞仙子。
<br/>    
<br/>        而且凭借着这个光环，紫霞仙子号令众人丝毫不是问题。

<br/>        紫霞仙子虽然实力可能和他们不相上下，但是号召力可是甩了他们几条街，毕竟是悟道榜前十之中唯一的一位女子。

<br/>        紫霞仙子毕竟是有着lv3的女王，安排起来井井有条，规规矩矩，不过一会儿，便已经将众多人王分配了下去，一时之间，众多人王都有了自己的事做，有的前去布防，有的前去接应其他的人王。

<br/>        苏青阳笑道：“道友这一手，可要思虑再三啊！”

<br/>        雪自悟扔动手中的骰子，一抛，骰子落地，最后停留在六点上，雪自悟将自己的白子向前挪了六步，已经是在苏青阳的前方，“哈哈哈哈，青阳道友，我这一手，如何？”

<br/>        于是陈明挥手，直接朝着劫云就是一指点出，一指通灵！
