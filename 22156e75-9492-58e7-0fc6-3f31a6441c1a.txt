　　听他们口无遮拦，欧阳少恭微微一笑，道：“小兄弟，据我所知，天墉城有一种法术叫顺风耳，能听见很远地方的声音。”
<br/>　　“啊啊——”
<br/>　　几名新进弟子面面相觑，心头狂跳，若真是如此，刚才的对话，不会已经传入那名师兄耳中了吧？
<br/>　　到时候，若是给他们穿小鞋，岂不是惨了？
<br/>　　见他们一个个吓得脸色惨白，欧阳少恭道：“不过已经失传了，你们不用害怕，随便说，他肯定听不见的。”
<br/>　　回过神来，几名新进弟子尴尬不已，岂能不知道，欧阳少恭是在耍他们。
<br/>　　“我们不过就是随便说说而已，不像有些人，一上来就攀关系拍马屁。”
<br/>　　听到他们的话，风晴雪想到欧阳少恭的能力，好心劝阻道：“你们最好不要随便说欧阳大哥，不然......”
<br/>　　“不然怎么样，他还敢出手教训我不成？”风晴雪话语还未落下，便被那名颇为嚣张的新进弟子给打断。
<br/>　　“哎，算了算了，以后都是同门，现在专心对付妖怪最重要，我听说，这天墉城是天下的清气聚合之地，有许多妖物环伺的，”
<br/>　　嚣张弟子不屑道：“怕什么，现在都丑时了，哪来什么妖怪啊，要有妖怪早就出来了。”
<br/>　　就在这时，黑雾遮挡住皎洁的月光，树林中一阵狂风吹起，树枝摇曳，树林四周顿时响起一阵沙沙声响。
<br/>　　“唰唰——”
<br/>　　陡然，只见虚空中无数道绿色残影袭来，顿时吓得周围新进弟子面色惨白，双腿打颤犹如筛糠。
<br/>　　甚至，有胆魄不足的，直接吓得裤裆都湿成一片，惊恐的尖叫声响起，其余新进弟子则脸色煞白，拔腿就跑。
<br/>　　“妖怪，真的有妖怪，妖怪杀人啦！”
<br/>　　“快跑，快跑啊！”
<br/>　　“师兄，师兄快救救我们！”
<br/>　　惊恐的尖叫声与求救声，新进弟子犹如惊弓之鸟，吓的四处逃窜，百里屠苏则坐在树杆旁，望着虚空中的小精灵。
<br/>　　看了眼天上的小精灵，风晴雪拉了拉欧阳少恭的长袍，道：“欧阳大哥不用怕，这里没有妖气，只有一些小精灵而已。”
<br/>　　欧阳少恭微微一笑，看了眼若无其事的百里屠苏，道：“我们去师兄那里，这样就不会被小精灵捉弄了。”
<br/>　　“好啊。”
<br/>　　旋即，欧阳少恭与风晴雪，便移步向百里屠苏走去。
<br/>　　与此同时，翡翠谷外两名身影，正鬼鬼祟祟的靠近这里，来者不是别人，正是阴险的陵端，与他的小弟肇临。
<br/>　　“去。”
<br/>　　来到翡翠谷外，只见陵端手捏法术，紫芒乍现，一颗金色的葫芦就出现在空中。
<br/>　　陵端阴险一笑，葫芦盖立刻开启，伴随着一阵尖锐的声响，两只狰狞的大鸟，便从其中奔腾而出，以极快的速度向翡翠谷中掠去。
<br/>　　待狰狞大鸟冲进翡翠谷，陵端将金色葫芦收回，满脸期待之色。
<br/>　　肇临望着大鸟掠去的方向，担忧道：“二师兄，不会被别人发现吧？”
<br/>　　闻言，肇临冷笑一声，道：“要是被发现了，那也是大师兄的责任，谁让他下山，没有关好妖怪。”
<br/>　　“二师兄，好歹是同门师兄弟，这样对付百里屠苏真的好吗？”
<br/>　　“那个百里屠苏，天天跟芙蕖师妹待在一起，芙蕖师妹肯定是被他给骗了。”
<br/>　　说罢，凌端眼中满是阴险，道：“而且，他仗着大师兄处处跟我做对，如今大师兄下山，没人护着他，正是教训他的好机会。”
<br/>　　“可是......”
<br/>　　“有什么好可是的，姑获鸟已经放出来了，现在想收手也迟了，我们走吧。”
<br/>　　收回担忧的目光，肇临一脸无奈，他心中总觉得这样是错的，但他也阻止不了，摇了摇头跟了上去。
<br/>　　...
<br/>　　...
<br/>　　此时，翡翠谷中，百里屠苏眉头深深的皱在一起，隐约察觉到了有东西在靠近。
<br/>　　“不好。”
<br/>　　就在这时，两只姑获鸟忽然自高空俯冲而下，利爪挥舞间，血液飞溅，惨叫声绕耳不绝。
<br/>　　新进弟子们更是吓得肝胆欲裂，他们都只是普通人，何时见过这种场面。
<br/>　　“咻咻——”
<br/>　　百里屠苏刚要拔剑，银光乍现，数十道剑影自天际纵贯而下，剑气四射，姑获鸟尸首分离，最后无力的坠落在地面。
<br/>　　“空明剑！”
<br/>　　两只姑获鸟瞬息间就被斩杀，百里屠苏震惊不已，同时也看出了刚才的招式，纵贯而下的剑影，可不就是天墉城最强的剑诀之一‘空明剑’吗？
<br/>　　拥有如此实力的，绝对是长老层次的修士，于是搭在剑柄上的手也松了开来，同时也暗自送了口气。
<br/>　　他自然看出了两只妖兽是姑获鸟，而且实力都不弱，至少是金丹期，还是高级金丹，若是他出手，可能真应付不过来。
<br/>　　两只姑获鸟瞬间被斩杀，一众弟子都处于懵逼状态，百里屠苏则抬起头来，不知何时，树尖上已经站着一名白色身影。
<br/>　　来者一袭白衣，立于树尖，黑发飘扬，面如冠玉，仙风道骨。
<br/>　　其实，慕清霄来这里已经很长时间，没想到陵端真敢放出姑获鸟来陷害同门。
<br/>　　慕清霄虽对天墉城没多少感情，但他心里却极其不喜欢这种宗门败类，平时一副狂妄自大的模样，暗中却是卑鄙小人。
<br/>　　两只姑获鸟虽被他瞬间斩杀，但实力当真不弱，虽被封印数十年，修为早已从化神掉落至金丹。
<br/>　　但稍有不慎，这些新进弟子都有殒命的危险，而且只有百里屠苏一人，对付两只姑获鸟，根本应付不过来。
<br/>　　轻轻一跃，慕清霄就落在地面上，瞥了眼地上的姑获鸟尸体，眼中闪过一抹凌厉的光芒。
<br/>　　“是......是天墉城的真人，多谢真人出手相救！”
<br/>　　“我的妈呀，刚才的剑术真是碉堡了！”
<br/>　　“这不就是，今日报名处的真人吗？”
<br/>　　百里屠苏回过神来，连忙迎了上来，微微行礼，道：“多谢凝丹长老相助。”
<br/>　　“无妨，你便是陵越的师弟，百里屠苏吧，你小的时候，我还见过你一面呢。”
<br/>　　望着眼前，容貌清秀的百里屠苏，或者说韩云溪，慕清霄有些感慨，光阴当真是弹指一瞬。
