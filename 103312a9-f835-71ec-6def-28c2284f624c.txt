　　佟千雪的微信还关注了一些军事公众号，更多的去了解这支时时刻刻保护着这个国家安定，可是却很少被人所关注的军队。
<br/>　　每次在网络上看到那些战乱国家的惨状，佟千雪就会想到网络上那句话，“我们没有生活在一个和平的年代，我们只是生活在一个和平的国家。”
<br/>　　佟千雪是张国荣的歌迷，曾经也为张国荣的自杀而感到痛心；而现在每年的四月一日，佟千雪除了知道是“愚人节”和张国荣的忌日外，还知道了一个叫“王伟”的飞行员以及他那架编号“81192”的歼-8Ⅱ战斗机。
<br/>　　平时佟千雪空闲的时候会去看一些综艺类节目或者动漫，现在佟千雪喜欢上了一只以兔子为主角，名叫《那年那兔那些事》的国产动画，这部动画虽然不是什么大投资制作，但是却能带给佟千雪特别的感动，尤其当两只兔子趴在雪地中，说出那句“不能放弃，祖国就在身后；不能落泪，眼泪也会冻住”，等到冲锋开始，一只兔子最终被冻死在雪地中的时候，一度让佟千雪落泪。
<br/>　　不仅如此，佟千雪还买了许多跟军事有关的书籍。这些书并非是军队内部的高度保密资料，可依旧让佟千雪成了一个“军事专家”，在看一些以特种部队为题材的影视剧时，会让她忍不住吐槽影视剧中“特种部队”的不专业。
<br/>　　就连宋菀都大呼，佟千雪不是自己以前所熟悉的那个学姐了，充分融入到“军嫂”这个新的人设中去了。除了不能像普通夫妻那样，心情好的时候可以撒个娇，心情不好的时候可以发个小脾气吵个嘴，其实做军嫂也没什么不好的。
<br/>　　慕容飞从K国离开，回到国内转机时，把自己的东西交给其他人后，自己一个人折身回到了杭山市。
<br/>　　有了老婆，慕容飞一年回杭山市的次数比以前几年加起来都多，如果被黄虹知道，又该气的从椅子上跳起来，大骂他是个“有了老婆忘了娘”的“不孝子”了。
<br/>　　出了杭山市国际机场候机楼，慕容飞坐上出租车以后，才发现自己没有带钱包！
<br/>　　机票是赵天宇帮忙准备的，而平时在龙隐部队吃住都在部队里，就跟深山野人一样很少有外出的机会，所以慕容飞根本没有随时携带钱包的习惯，尤其这次是直接从K国转机回的杭山市。
<br/>　　“师傅，麻烦您在这里等一下，我没带钱包，我回去拿了钱再来付车钱。”慕容飞向出租车司机打招呼道。
<br/>　　出租车司机看了眼小区大门，佟千雪这个小区在杭山市很出名，能住的起这样的小区也不像是会来车费的人，就勉为其难的答应了。
<br/>　　慕容飞刚走到小区门口，就被保安给拦了下来，“你是谁啊？来这里找谁？看你好像不是我们这里的业主吧。”
<br/>　　“我找佟千雪，我是她丈夫，她住在……”
<br/>　　“行了，你就别吹牛了。”慕容飞话还没说完，就被保安给打断了，说道：“佟医生一直都是一个人住，根本就没看到她有跟男人进出过这里，还说是佟医生的老公，吹牛也不知道编个靠谱点的理由。”
<br/>　　佟千雪住的楼层高，如果站在阳台上，能正好看到小区大门那边。佟千雪依靠在窗户前，看到了还门卫处跟保安正在交涉的慕容飞，佟千雪现在这个角度看不到慕容飞的侧脸，可仅仅只是看到一个背影，还是令佟千雪心头一颤，赶紧走到客厅，拉开阳台落地窗走到阳台上，当看到还在跟保安进行交涉的慕容飞的侧脸时，佟千雪眼眶不禁有些发红。
<br/>　　今天佟千莹正好休息，儿子祁飞送到他外公外婆那里去了，佟千莹就来照顾自己的妹妹。毕竟很多不方便跟长辈们交流的悄悄话，亲姐妹之间交流起来更方便。
<br/>　　看到急匆匆往外走的佟千雪，佟千莹开口问道：“你上哪儿去，走的这么急？”
<br/>　　“他回来了！”
<br/>　　佟千雪打开门，连鞋子都没有换，踢踏着拖鞋就跑了出去。
<br/>　　“他回来了？”
<br/>　　佟千莹一头雾水，一时间还没从自己妹妹的话里反应过来；片刻之后，佟千莹明白了自己妹妹在跑出去的时候，嘴里所说的那个“他”是谁了，也只有慕容飞回来了，才能让自己妹妹如此失态。
<br/>　　慕容飞真的回来了？！
<br/>　　佟千莹虽然不如佟千雪那么清楚的了解慕容飞在什么样的部队服役，但是从杭山市特警队队长、市公安局的人对慕容飞那么客气的态度就能知道，慕容飞所服役的部队绝对不是什么简单的常规部队。佟千莹不了解军队，可是也知道越是非常规部队，想要探亲休假就越困难。
<br/>　　佟千莹赶紧走到阳台上果然是看到了门卫处的慕容飞。慕容飞，佟家人对他还是充满感激的，因为慕容飞的关系，以前由于祁伟坚而卡着没有解决，比如祁飞户口迁移，祁伟坚找人上门闹事的事情全都很轻松的得以解决。
<br/>　　杭山市公安局局长在部队服役的时候，是赵国邦手下的兵，赵天宇上次来杭山市的时候，去拜访过他，第二天局长就亲自上门表示有任何困难都可以跟他联系，让那个祁伟坚副局长亲戚连个屁都不敢放。
<br/>　　佟千雪走得急，脚趾都被脚上的塑料拖鞋给磨掉了一块皮，可佟千雪根本没有感觉到，此时她只想快点跟慕容飞见面。
<br/>　　门卫处，慕容飞不断的解释，可是那个保安说什么都不相信慕容飞的话，等在大门口的出租车司机也开始怀疑慕容飞是不是坐霸王车的，犹豫着是不是要报警。
<br/>　　慕容飞拿出自己的军官证，说道：“这是我的证件，我把它放在这里总可以了吧，或者你打个电话给佟千雪确认我的身份。”
<br/>　　“现在社会上假军官多的是，专门骗小姑娘，别以为你拿着一个小本子就能糊弄过去。”保安还是拒绝相信慕容飞的身份，说道：“你别想骗我去打电话，然后偷偷溜进去，这招没用！”
