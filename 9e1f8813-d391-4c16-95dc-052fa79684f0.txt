　　两年后……
<br/>　　华夏整个华南区域的城市，丧尸已经全部清扫干净，这一举动震惊了所有国外的基地，他们开始大量收购狼牙专利的丧尸引诱剂，开始推行同样的政策。
<br/>　　狼牙的发展也越发壮观，以两座城市为中心，开始不断往外扩建。
<br/>　　城市内也全部通电，生活已经追上了文明时代的步伐，只是还有部分科技没有恢复，但比起其他地方，狼牙地界已经发达的令全世界人神往！
<br/>　　通往宝岛的船只也在这两年内往返多次，与宝岛上的各个基地接上轨，往来过程中虽有摩擦，但在大势所趋之下，双方都达成了共识。
<br/>　　一切都在楚涵的计划中发展，没出差错。
<br/>　　只是这一天，刘玉定怒发冲冠的冲到楚涵面前，一沓文件‘啪’的用力拍在桌上：“你不要以为你是我长官，又帅又高还战力无边，我就怕你了！”
<br/>　　楚涵头也不抬，吃着尚九娣送来的爱心早餐，摸着在旁边乖乖坐着的上官雨馨柔夷，一脸幸福的样子。
<br/>　　上官雨馨羞红了脸，连忙起身告辞：“我先走了。”
<br/>　　“啊？啊！别啊嫂子，我等会儿再来好了。”刘玉定连忙退后几步。
<br/>　　“不不，你们谈正事吧，我也该走了，南都基地明早还有个会议。”说着，上官雨馨便起身离开，走到门边又回头冲着楚涵说了一句无声的口语：
<br/>　　我明天晚上来！
<br/>　　上官荣已经辞退职务整日钓鱼，动不动钓上来比人大的凶兽，总之挺闹腾的。
<br/>　　与尚九娣和步纱常年住在狼牙不一样，上官雨馨接下了父亲的职务，现在是南都基地的首领，功夫繁忙的很。
<br/>　　但楚涵坑了高少辉的天照3000，直接作为订婚礼送给了上官雨馨，南都到狼牙的距离虽远，但驾驶着天照3000却只是分分钟的事。
<br/>　　于是在没人知道的情况下，上官雨馨便经常跑过来与楚涵私会，根本不耽误工作。
<br/>　　可以说这两年楚涵的小日子，过的相当舒坦！
<br/>　　等到上官雨馨走后，刘玉定又摆起臭脸，指着楚涵的鼻子大吼：“长官！你今天必须要给我解释清楚，这一大沓计划方案，到底是什么个意思！”
<br/>　　“全体狼牙精兵！你知道这数量是多少吗？三十万啊！你让三十万狼牙精兵在一周内集合，还跑到这个什么地方，排一个爱心阵队？”
<br/>　　刘玉定越说越气，将那一沓纸在桌子上拍的‘啪啪’直响：“你给我解释一下！是不是现在敌人都被你干掉了，你开始闲着没事干了！”
<br/>　　楚涵吃完早餐，指着那一沓文件道：“我一听就知道这方案你没看完，没看完就有脸来找我吼呐？”
<br/>　　刘玉定额头青筋直跳：“老子看完了！你别跟我贫！我强忍着抽你的冲到看完的！后面还有一系列让我忍无可忍的方案，什么全体射击队用最先进枪支射·出烟花效果，什么神隐战队去野外拉怪来一首群狼之歌。”
<br/>　　“你特么……”刘玉定忍无可忍：“你不要逼我揍你！”
<br/>　　楚涵瞥了他一样：“你确定你要对我动手？”
<br/>　　刘玉定瞬间萎了：“对不起长官，是我飘了，我打不过你……”
<br/>　　楚涵笑道：“安排一下。”
<br/>　　刘玉定不信邪：“你真的不告诉我原因？”
<br/>　　楚涵摆摆手：“到时候你就知道了。”
<br/>　　……
<br/>　　一周的时间很快就过去，狼牙的三十万全体精兵忽然的调动，引起了所有人的注意，不少基地都派人跟着前往，甚至还有人专门拿着摄像机录下来。
<br/>　　楚涵也早早到达预定地点，与平时的不着调不一样，他今天穿的尤其正式。
<br/>　　这是一处山谷，或者应该说是阴阳谷外的一处山谷，这里是地球磁场被大面积屏蔽中裂开的那道裂缝！
<br/>　　若说宇宙时空中有探测仪能探测到这里，这是唯一的降落地。
<br/>　　此时此刻在整个山谷之中，狼牙战团的全体三十万成员都排列成一个巨大的爱心形状，他们整装待发，每一小队的人都面色庄重。
<br/>　　几个中将虽然一脸的哔了狗，但也依旧站得笔直。
<br/>　　神隐在山谷外拉怪，渐渐的已经有狼嚎声由远及近，很快这狼之歌就会彻响整个山谷。
<br/>　　射击队已经准备好，陈少爷一脸的视死如归，他已经彻底放弃楚涵了，他觉得这个老大可能精神不太好。
<br/>　　杀羽和黑芒两队则是一左一右的跟在楚涵身后，一个个气势如虹，那样子真是威风凛凛！
<br/>　　一切准备就绪，然后开始等。
<br/>　　各个基地派来的人不断录着，只是一个小时过去，两个小时过去，这帮人手都酸了，还是什么都没发生。
<br/>　　搞什么？
<br/>　　很多人傻眼。
<br/>　　唯独狼牙战团的魔鬼训练，让他们再如何内心疑惑，也没表现在脸上，一个个站如松的在原地一动不动。
<br/>　　包括楚涵！
<br/>　　楚涵可能是最严肃庄重的一个，慎重到让所有人都惊讶。
<br/>　　终于，他的眼神一动，抬头看向了天空！
<br/>　　这一举动引起了所有摄影师的注意，他们瞬间打起精神，将摄像机对准楚涵所望的天空方向。
<br/>　　但十分钟过去，依旧什么都没发生。
<br/>　　外来基地的人开始窃窃私语，但也不敢太过分，只能这么陪着楚涵等。
<br/>　　终于在楚涵抬头望天的二十分钟后！
<br/>　　一个巨大的黑影，出现在了天空之上的云层里！
<br/>　　晴空万里，瞬间变得一片阴影。
<br/>　　不少人都震惊起来，狼牙战团的三十万精兵也瞳孔剧烈收缩，这是什么？
<br/>　　黑影越来越大，也越来越清晰，渐渐的开始呈现出一艘船的样子！
<br/>　　那些摄影师一个个大喊大叫，惊恐无比。
<br/>　　狼牙战团的三十万精兵都震惊，有人开始躁动。
<br/>　　这时候的楚涵一喝：“安静！”
<br/>　　瞬间，全体死寂。
<br/>　　飞船开始越来越低，最终在距离地面百米的地方停下，一根锁链从其上落下来，垂在了楚涵面前。
<br/>　　楚涵也有些难以抑制内心的情绪，声音颤抖着出声道：“仪仗起！”
<br/>　　轰——
<br/>　　瞬间，无数发光弹射向天空，一颗颗绽放，连起来正是一个巨大的爱心形状。
<br/>　　射击队数着拍子，让这爱心连续不断，也持久不停！
<br/>　　周围的狼嚎声也彻响了起来，热闹非凡！
<br/>　　与此同时，一个白衣女子也从那锁链的顶端出现，与楚涵相隔着百米的高空，女孩的手有些颤抖，面容有些疲惫，但面上却是一个灿烂的笑容。
<br/>　　正是白允儿！
<br/>　　她眼角有些晶莹，也是第一次这么肆无忌惮的笑。
<br/>　　她还是带着那透明的银色手套，一拉锁链就顺着滑落而下，快速的才百米之外扑向楚涵。
<br/>　　楚涵张开手臂，一把接住她。
<br/>　　“我回来了。”
<br/>　　“欢迎回家。”
