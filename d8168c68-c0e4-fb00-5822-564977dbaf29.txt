        &lt;h3&gt;第三卷第二百六十三章  宁雅，  我是你坚强的后盾&lt;/h3&gt;
<br/>
<br/>        宁雅和宁家商谈合作的时候，凌辰一直在宁家公司大楼下的咖啡厅坐着，他要了一杯不加糖拿铁，轻轻的喝一口，香浓的苦苦的咖啡味立刻传遍四肢百骸。
<br/>
<br/>        他要坐在这里等宁雅，他要在这里默默的替宁雅加油，给宁雅支持。
<br/>
<br/>        董亚子出去调研市场了，原本他要和董亚子一起的，但是他很想陪着宁雅。
<br/>
<br/>        “亚子，我今天就不出去了，我去宁家楼下。”凌辰吃完早餐对董亚子说。
<br/>
<br/>        “可怕的男人，真的是爱美人不爱江山，我真是后悔了当初替你说话。”董亚子恶狠狠的看着凌辰。
<br/>
<br/>        “我就想陪着宁雅，我知道我没有权利进入宁家，我就在他们楼下等着她，我要让她知道她还有我，她不是孤军奋战。亚子，你也是一个有爱的人，你看，我好不容易在你的帮助下赢回宁雅的心，你也不希望你的努力付诸东流吧？”凌辰再一次对董亚子说。
<br/>
<br/>        “嘶，凌辰，你这厚脸皮真的是让我吃惊，快滚吧，我要去调研市场了。”董亚子笑着说。
<br/>
<br/>        她很担心刘诚之，刘诚之到甸城之后一直给她发信息报的都是平安，但是对于工作上的的事情只字不提，他们在这边也没有什么动静，山西是凌辰的大本营，有一点点小东西也只是小鱼小虾，激不起太大的浪。
<br/>
<br/>        她需要在别的地方寻找突破口，让刘诚之好开展工作。
<br/>
<br/>        “谢谢善解人意，美丽可人的亚子姑娘，凌某人感激不尽，等我和宁雅结婚的时候好好敬你几杯酒。”
<br/>
<br/>        “啧啧啧，八字还没有一撇，就想着结婚？”董亚子毫不犹豫的打击凌辰。
<br/>
<br/>        “人生需要有梦想，更需要有希望，亚子妹妹，拜拜了。”凌辰快乐的拿起自己的包转身离开董亚子，飞奔到宁家公司的楼下。
<br/>
<br/>        他看着宁雅和律师走进宁家公司的大楼，拿起手机给宁雅发了一个信息：“日月同在，天地同辉，凌辰永远和宁雅在一起，凌辰永远是宁雅坚强的后盾。”
<br/>
<br/>        坐在咖啡厅里，他拿出笔记本处理工作，虽然在等宁雅，工作却不耽误。
<br/>
<br/>        打开邮箱，发现刘诚之发的邮件：“董总，甸城山高水深，环境清幽，有兽有鱼，适合养生，我一时三刻不想回去，如果可以，请让人接替我的位置，问候大家！”
<br/>
<br/>        凌辰想了想，将邮件转发给何少华：“何总，你这是在挖我的墙角？”
<br/>
<br/>        凌辰知道等不到何少华的回复，因为在何少华的心目中，他凌辰就是一颗空心的大葱，但是他会看到这封邮件的。
<br/>
<br/>        刘诚之这是发现端倪了？刘诚之去甸城这么快就发现端倪，这很不正常，除非对方有意识让刘诚之知道。
<br/>
<br/>        凌辰不知道的是，原本他只是一个生意人，无形之中被扯进了一个巨大的漩涡之中，他只能跟着这个漩涡旋转，无法脱身也无法袖手旁观。
<br/>
<br/>        凌辰没有给刘诚之回邮件，他还没有想好怎么回，等下午和董亚子汇合之后再回复。
<br/>
<br/>        坐着等了几个小时，宁雅终于从大楼里走了出来，凌辰急速的跑出咖啡厅，站在宁雅的面前：“成功了？”
<br/>
<br/>        “成功了！”宁雅商谈结束后才发现凌辰的信息，她心里微暖，这么多年，凌辰其实是一直站在她身边的，“你怎么在这里？”
<br/>
<br/>        “我要在这里陪着你！”凌辰急切的看着宁雅，眼里盛满宁雅的自信美丽的脸，世间只有宁雅一人。
<br/>
<br/>        “我要先去酒店，下午可能得回一趟家里，晚上飞回北京。凌辰，谢谢你！”宁雅微笑着说。
<br/>
<br/>        凌辰的心里有些不舍，他很想和宁雅单独呆一段时间，他们总是空中恋爱，网络交集，在现实中都是匆匆聚匆匆散，这让他很没有安全感。
<br/>
<br/>        “不能明天走吗？我想晚上请你一起吃饭，就我们俩。”凌辰的语气带了点点恳求。
<br/>
<br/>        无论是男人还是女人，在恋爱的时候心境都会变，别说女人会患得患失，男人也会。
<br/>
<br/>        “行程都定好了，陈子昂将有些计划提前了，我已经约好了明天要见的人，不得不走。”宁雅解释着，其实这一次见凌辰，她的心境发生了很大的变化，她也想和凌辰多点时间接触。
<br/>
<br/>        “那好吧，我以后去北京找你。”凌辰有点委屈巴拉的。
<br/>
<br/>        “咦，你看看你这委屈的样子，好像我欺负你了一样。中午我们一起吃饭啊，傻瓜。”宁雅看了一下时间，还有半小时就到饭点了。
<br/>
<br/>        “你中午不和你爸爸一起吃饭？”凌辰以为宁雅中午会和宁家一起吃饭，合作都谈成功了，不是应该庆祝一下吗？
<br/>
<br/>        “我和他们一起吃饭，谁和你一起吃饭？董亚子吗？宁玥吗？”宁雅笑着问凌辰。
<br/>
<br/>        “啊呀，满大街的酸爽味道，又酸又甜，我好喜欢！我去将笔记本收拾了。”凌辰的笔记本还在咖啡厅放着。
<br/>
<br/>        “这是你男朋友？”王律师笑着问宁雅。
<br/>
<br/>        宁雅笑而不语。
<br/>
<br/>        “你这男朋友心里装的都是你啊，小雅，很幸福吧？”王律师接着问。
<br/>
<br/>        “我怎么不知道他的心里都是装着我呢。”宁雅心里甜甜的。
<br/>
<br/>        凌辰出来，看了一眼宁雅身边的王律师心里想：“这个人怎么还在？没有听到他们中午要一起吃饭吗？”
<br/>
<br/>        “小雅，你们去吃饭，我先回酒店，将电子版的合同发给陈总，让她审批一下，下午就可以签合同了。”王律师看着凌辰那嫌弃的眼神很知趣的说话了。
<br/>
<br/>        “中午一起吃饭吧，到了饭点了。”凌辰不得不开口挽留。
<br/>
<br/>        “我要是中午和你们一起吃饭，你这眼神都会将我杀死，我还是保命去吧。”王律师哈哈大笑。
<br/>
<br/>        “你真是个有趣的人呢，小雅你看？”凌辰被王律师戳穿了心思，有点不好意思。
<br/>
<br/>        “王律师还有事情，我们去吃饭吧。”宁雅笑着开口。
<br/>
<br/>        俩个人真准备走，后面传来了宁玥的声音：“凌辰哥哥，姐姐，你们等一下。”
<br/>
<br/>        “怎么了？”凌辰心里不爽，真是一点眼色也没有。
<br/>
<br/>        “爸爸说中午请你们一起吃饭，第一是庆祝陈宁两家成功合作。”宁玥浅浅而笑。
<br/>
<br/>        “第二呢？”宁雅问宁玥。
<br/>
<br/>        “第二，凌辰哥哥有亲人到来，为凌辰哥哥的亲人接风。”宁玥的脸上露出了羞涩。
<br/>
<br/>        “我的亲人？我怎么不知道？”凌辰很奇怪，他怎么不知道他家里来人了？
<br/>
<br/>        “你去看看不就知道了。”
