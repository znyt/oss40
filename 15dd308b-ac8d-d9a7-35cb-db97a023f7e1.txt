　　“什么！”
<br/>　　神王路西法简直无法相信，自己所看到的一切。
<br/>　　眨眼之间，叶尘的气息竟然一举超过了他，甚至到了连他都为之心悸的地步！
<br/>　　“不！我不相信！这世上，怎么会有如此逆天的秘术？”
<br/>　　路西法不甘的大吼了一声，再次使出了自己的最强一击。
<br/>　　然而这一次，叶尘连眼皮都没有抬一下，直接随手一剑斩落。
<br/>　　轰隆！
<br/>　　气势磅礴的神王路西法，竟然被叶尘一剑斩飞了出去！
<br/>　　并且将一座高达上千米的山峰，硬生生撞的粉碎！
<br/>　　“这……”
<br/>　　所有人看到这一幕，顿时全都目瞪口呆，那数十名神界高手面如土色，就连一直在观望状态的魔、仙、妖三界高手，同样也个个脸色剧变，
<br/>　　“这叶狂仙怎么会这么强大！”
<br/>　　“没想到此子的实力，比数月之前，强大了这么多！”
<br/>　　“看来连神王路西法也不是此人的对手，照这么下去，神界这些高手怕是要全军覆没了！”
<br/>　　……
<br/>　　而与此同时，
<br/>　　叶尘一剑将路西法击飞之后，已经再次抡起紫琼剑，追了上去。
<br/>　　“誓死保护神王大人！”
<br/>　　那些原本已经元气大伤的神界众强者，看到这一幕，一个个如同飞蛾扑火般，朝叶尘扑了过去。
<br/>　　“生命燃烧术！”
<br/>　　数十名神界强者，眼看连施展了秘术后的神王，都不是叶尘的对手，估计自知今日在劫难逃，一个个竟然以生命燃烧术，疯狂般朝叶尘发起最后一击！
<br/>　　数十名元婴境强者的拼死一击，连天地都为之变色，整个苍穹都被那耀眼的光芒所遮挡，即便是相隔十余里外的神龙卫众人，都能清晰的感受到那种灼热，以及那令人颤抖的气息。
<br/>　　毫不夸张的说，这些神界强者们，随便其中一人，都足以横扫一国，在不动用大型核武的情况下，便是百万军队都拿他们毫无办法。
<br/>　　而此时，这些让世俗仰望的强者们，却以一种同归于尽的方式，向叶尘发起自杀式的进攻！
<br/>　　然而，在叶尘眼中，这些人的行为，似乎并没有引起他任何情绪波动，
<br/>　　“一群蝼蚁！不自量力！”
<br/>　　轰轰轰！
<br/>　　砰砰砰！
<br/>　　叶尘不退反进，直接冲杀进入这些神界强者群中，
<br/>　　右手挥剑！
<br/>　　左手挥拳！
<br/>　　嘭嘭嘭！
<br/>　　叶尘简直犹如一架人形坦克，所过之处，那些神界强者，要么被叶尘一剑劈成两半，要么直接一拳打爆！
<br/>　　不一会的功夫，叶尘已经轻松斩杀了十余人，
<br/>　　“我来！”
<br/>　　剩下的神界强者中，忽然窜出一名须发尽白的白袍老者。
<br/>　　原来，此人便是神界之中，仅次于神王路西法的神界大祭司，不但实力不再路西法之下，甚至论资历，比路西法还要年长百岁。
<br/>　　“海耶斯！我最忠诚的朋友，出来吧！”
<br/>　　随着白袍老者一阵吟唱，他那宽大的袖袍之中，陡然发出一道妖异的光芒，继而形成了一个类似于黑洞的诡异漩涡，显然是一件空间类的法宝！
<br/>　　轰隆！
<br/>　　忽然一具庞然大物，自那空间法宝中窜出！
<br/>　　“快看！龙！竟然是龙！”
<br/>　　远处围观的华夏军队，顿时纷纷指着空中的景象，大呼大叫起来。
<br/>　　只见那庞然大物，一身黑色鳞甲，足足有数百米长，和西方神话传说中的巨龙极为相似。
<br/>　　“吼！！”
<br/>　　那黑色巨龙出现之后，顿时发出一道怒吼，朝叶尘扑了过去。
<br/>　　叶尘脸色依旧淡漠，
<br/>　　“一头化形境都没有到的孽畜，也敢在我面前猖狂？给我死来！”
<br/>　　随着声音落下，叶尘直接遥空一拳砸出！
<br/>　　轰！
<br/>　　一道硕大的赤色拳头，狠狠砸落在那黑色巨龙的脑袋之上。
<br/>　　彭！
<br/>　　那黑色巨龙的脑袋，瞬间便被叶尘一拳打爆！
<br/>　　“什么！”
<br/>　　白袍老者顿时两眼瞪得老大，险些喷出一口老血。
<br/>　　他原本以为，这头妖龙就算打不过叶尘，至少也能撑上一时半刻。
<br/>　　而他则可以利用这个世界，护送神王路西法逃离此地。
<br/>　　然而他怎么也没有想到，叶尘的实力居然强大到如此地步！
<br/>　　他精心培养了数百年的妖龙，竟然如此的不堪一击，被叶尘随手一拳打爆。
<br/>　　“这家伙到底是什么来头？难道他是那个世界……”
<br/>　　还没等他的话说话，叶尘一拳爆了妖龙之后，已经穿梭空间，来到了他的跟前，
<br/>　　“你也死吧！”
<br/>　　同样又是一拳！
<br/>　　“不！！！”
<br/>　　白袍老者第一时间迅速暴退，可是叶尘却如影随形一般，离他越来越近！
<br/>　　最终，那颗拳头终究还是落在了他的脑袋之上。
<br/>　　“彭！”
<br/>　　没有任何悬念，白袍老者的脑袋，应声而爆！
<br/>　　“嘶~~~”
<br/>　　不但剩下的神界强者们，纷纷倒吸了一口凉气，就连一直观望的魔祖和无涯老人，此时的心情也难以用语言来形容。

<br/>　　无涯老人忽然冒出这么一句来，显然已经有些动摇。
<br/>　　魔祖脸色一阵变幻，最终双眸一凝，
<br/>　　“此子的进步速度实在可怕，倘若今日不杀了他，我们四界早晚都要被他所灭！”

<br/>　　无涯老人说完这话，身体微微一晃，在原地消失不见。
