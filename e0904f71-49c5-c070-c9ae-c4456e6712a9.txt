　　苏莫见众人如此，不禁微微一怔。
<br/>　　自己在十八层怎么了？难道进入十八层很夸张吗？
<br/>　　“段惊天，你这个理由很强大，强大的让我惊讶！”
<br/>　　褚彦笑了起来，面带戏虐之色，道：“那今日，我就看看，你是否进入了十八层？”
<br/>　　褚彦感觉有些好笑，这个段惊天也太能吹了吧！
<br/>　　他打定主意，等会，段惊天不能证明自己到了十八层，他也极尽羞辱。
<br/>　　“段惊天，把你抓去的火元精给我吧，我给予你奖励！”
<br/>　　这时，中年执事开口了，他也不认为苏莫能进入十八层。
<br/>　　但是，不管进入了多少层，该给的奖励还是要给的。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即，他手掌一番，掌中银光闪动，出现了一团银色的火焰。
<br/>　　这银色的火焰，便是第十七层的火元精。
<br/>　　“十七层的火元精！”
<br/>　　中年执事见此，顿时眼眸一亮，毕竟，能进入第十七层，已经非常的惊人了。
<br/>　　平常时候，在人王宫的外宫弟子之中，十年都难有一人进入第十七层。
<br/>　　执事震惊了，虽然苏莫没有进入十八层，但是十七层，也足够厉害了。
<br/>　　褚彦和其他人，均是有些惊讶，这个段惊天居然进入了第十七层，果然厉害。
<br/>　　不过，很快他们便纷纷冷笑了起来，毕竟，第十七层虽然厉害，但是和第十八层相比，可不是一个概念。
<br/>　　“段惊天，你刚才不是说进入了第十八层吗？你这是在打自己的脸吗？”褚彦冷笑着说道。
<br/>　　“哈哈，牛皮吹破了吧？”
<br/>　　“谎言根本站不住脚！”
<br/>　　不少人大笑出声，这段惊天吹牛完全没有不考虑能不能圆谎。
<br/>　　简直就是个笑话啊！
<br/>　　苏莫面色平静，没有理会众人，又向执事抱拳，道：“执事大人，我在第十八层，得到了此物，不知是何物？”
<br/>　　言语之中，苏莫手掌中光芒一闪，一个金色的光团出现了，闪烁着夺目的精光。
<br/>　　噶~~~
<br/>　　霎时之间，所有人的脸庞都凝固了，包括中年执事，均是目不转睛的望着苏莫手中的金色光团。
<br/>　　众人的眸中，露出震惊之色，仿佛这金色的光团，是某种不可思议之物一般。
<br/>　　“这……这不可能？”
<br/>　　褚彦一脸的懵逼，口中低低的自喃，他简直不敢相信自己的眼睛，段惊天居然真的进入了十八层！
<br/>　　因为此物，只有第十八层才拥有，而且，只有真神境以下修为的弟子，才能收取。
<br/>　　这一刻，褚彦懵逼了，一众弟子懵逼了，连中年执事也懵逼了。
<br/>　　第十三位！
<br/>　　从熔岩窟建立以来，这数十万年以来，第十三位进入第十八层的外宫弟子，出现了！
<br/>　　“执事大人，此物不简单？”
<br/>　　苏莫见众人的模样，诧异的问道，同时他心中亦是非常好奇，此物到底是什么。
<br/>　　“段惊天，没想到你真的是绝世妖孽！”
<br/>　　中年执事回过神来，眼眸大亮，再次打量了苏莫一番。
<br/>　　随即，他又道：“这是我们人王宫的长老火陵神王的道统，当然，这只是一小部分火陵神王修炼火源之道的经验！”
<br/>　　“道统？”苏莫闻言讶然，原来是一位神王的道统。
<br/>　　他心中明了，这火陵神王，很可能就是创造出熔岩窟的人。
<br/>　　而且，此人，必定是一位修炼火属性大道的神王。
<br/>　　“段惊天，可惜你不是修炼火属性力量的武者，不然可拜火陵神王为师！”执事叹了口气说道。
<br/>　　他能看出来，苏莫体内充满了五行之力，并不是单修火属性之道。
<br/>　　“拜火陵神王为师？”
<br/>　　苏莫闻言，再次惊讶了，得到了这玩意，就能拜火陵神王为师了？
<br/>　　难怪其他人如此震惊，原来还有这种事情。
<br/>　　不过，现如今，他还不太敢去接近人王宫的神王境界的长老。
<br/>　　不怕一万，就怕万一！
<br/>　　万一，对方看出了自己的底细，万一人王宫的立场已经站在了帝一魂那边，他就完蛋了。
<br/>　　“嗯！”
<br/>　　中年执事点了点头，道：“这熔岩窟，不仅是火陵神王所建，也算是对于外宫弟子的考验，几十万年了，只有十二人进入了第十八层，你是第十三位，也是这三万年来的第一人，只要你愿意修炼火属性大道，便可前去内宫，拜火陵神王为师！”
<br/>　　“明白了！”
<br/>　　苏莫点了点头，心中不禁有些无语，一不小心，自己居然又创造了奇迹。
<br/>　　在下界之时，他一不留神，就会打破纪录，到了神域也是一样。
<br/>　　其他众弟子，个个沉默不语，但是，很多人均是面带羡慕之色，甚至有人眸中露出嫉妒之色。
<br/>　　拜神王境的长老为师啊，这可是天大的机缘。
<br/>　　在这个太古至强者不现身的时代，神王境的强者，就是神域的巅峰了。
<br/>　　褚彦面色变换，脸色要多难看有多难看。
<br/>　　稍一沉吟，他强笑一声，道：“段惊天，我观你气息，是五行气息，你应该修炼五行之力，并不适合火属性之道啊！”
<br/>　　苏莫对于褚彦的话，充耳不闻，心中在思索着利弊。
<br/>　　褚彦见此，以为苏莫听进了他的话，继续道：“你的天赋如此强大，战力也非常厉害，若是改修道统，怕是得不偿失！”
<br/>　　他不希望苏莫拜火陵神王为师，因为那样的话，苏莫就爬到了他的头上。
<br/>　　“执事大人，您还是先给我奖励吧！”苏莫手掌一番，收起了金色光团，向中年执事说道。
<br/>　　要不要拜火陵神王为师，不急于这一时，还是先拿到奖励再说。
<br/>　　“嗯，你闯到了第十八层，按照规矩，你将得到一百三十一万零七百一十五枚下品源石，不过，因数量太大，你要稍等片刻！”中年执事道。
<br/>　　这么庞大的一笔源石，他需要回人王宫调集。
<br/>　　其他人闻言，纷纷倒吸了一口凉气，即便人王宫资源充足，一百多万下品源石，也是一笔惊人的数字。
<br/>　　“那好，我在此等待！”
<br/>　　苏莫点头道，言罢，他想到了一件要事，急忙补充道：“我需要五行属性的源石！”。
<br/>　　中年执事颌首，随即，他立刻通过出口的传送阵，返回了人王宫。
<br/>　　“段惊天，你不准备去拜火陵神王为师吗？”褚彦试探性的问道。
<br/>　　因为，苏莫索要的奖励，是五行属性的源石，所以，他有这个猜测。
<br/>　　毕竟，若是苏莫确定拜火陵神王为师，那么，估计就只要火源石了。
<br/>　　当然，这也不绝对，只是有一丝可能而已。
<br/>　　据闻，当年的那十二位闯到第十八层的弟子，仅仅只有三人拜火陵神王为师了。
<br/>　　因为，其他人都各自见此自己的武道之路，不想改修火属性之道。
<br/>　　毕竟，有的人虽然天赋妖孽，但是，与火属性完全没有关系。
<br/>　　“我拜不拜火陵神王为师，与你何干？”苏莫冷冷的反问道。
<br/>　　呃~
<br/>　　褚彦闻言，顿时无言以对，说不出话来了。
<br/>　　“段兄，之前是在下鲁莽了，还请勿怪！”
<br/>　　这时，魏仑上前一步，向苏莫抱拳。
<br/>　　因为他清楚，苏莫一旦拜火陵神王为师，地位将直线拔高，不是他能得罪的。
<br/>　　所以，他要趁着苏莫还没拜师之前，赶紧认错，好化解恩怨。
<br/>　　此时此刻，魏仑心中后悔不已，暗骂自己没事找事，居然帮助褚彦对付段惊天。
<br/>　　“段兄弟，在下也有错，不过我并无恶意，还请见谅！”
<br/>　　高大青年鲁东，一见魏仑认错，急忙也向苏莫抱拳。
<br/>　　褚彦见此，脸色黑了下来，这局势变化之快，简直让他无法预料。
<br/>　　之前这些人，还信誓旦旦的，要帮他教训段惊天。
<br/>　　眨眼之间，居然纷纷反水了。
