　　龙墓之中，化龙池之内。
<br/>　　元始骨骼不断破碎重组，在这期间，真龙精血以及浓郁龙气不断地改造着他的全身，但要数龙气最多的还是他的骨骼，使他的身体达到真龙一般。
<br/>　　同时，元始的元神之中，《神元九转》第六转涅槃运转起来，借助此处纯净的龙灵之气，破碎元神，使之融入更多龙气，达到涅槃的效果。
<br/>　　化龙池，本身就有化龙的效果，元神涅槃，与化龙的效果一样。
<br/>　　更何况，元始本身的灵魂就有过一次涅槃，而且还有灵魂之火存在，这元神涅槃对于别人来说，要困难十倍百倍，但对于他来说，化龙池加上灵魂之火，却不是那么的困难。
<br/>　　《神元九转》第六转就这样水到渠成地成功了。
<br/>　　同时，元始的肉体开始最后的重组了，一道道肉体规则被元始捕捉到，化龙池的真意在于化龙，而并不是修炼，修炼只是顺带的。
<br/>　　随着元始修复运转《玄黄不灭》，他的肉体开始真正强大起来，堪比真龙一样，再加上五行四象之力的淬炼，他的肉体已经踏入半步混元了。
<br/>　　《玄黄不灭》正式突破到第四重龙脉融龙骨，本身更是成就真龙体。
<br/>　　同时，元始也接受到真龙精血之中的一些龙族传承，特别是龙祖的血脉传承。让元始知道，龙祖修炼的竟然是力量法则与金系法则，难怪龙祖如此强大。
<br/>　　龙祖为天龙，为龙族之祖，修炼力量法则，肉体强大，只是他还没有领悟更深层的力量罢了，要不然他还会更加强大的。
<br/>　　金系法则也是杀伐最重的法则之一，金系配力量，龙祖想不强大都不行。
<br/>　　通过血脉传承，让元始也了解到一些力量法则。正所谓一力降十会，力量达到极致，一拳暴星辰，一拳碎苍穹只是小意思。要知道，盘古就是主修力量法则的了。
<br/>　　“咔擦……”
<br/>　　就好像模具定型一般，元始全身骨骼在发出最后一声咔擦后，就彻底稳固下来，而此时的元始，《玄黄不灭》第四重也彻底成功了。
<br/>　　但元始并没有停下，依旧毫无顾忌的运转功法，肉体还在不断增强，化龙池之中的龙灵之气不断地融入到他的身体之中。
<br/>　　真龙体初期，中期，后期，大成……
<br/>　　直至第四重真龙体大成，元始才开始醒过来，此刻的他已经心满意足了。要是没有这化龙池，仅仅还是那百滴真龙精血，虽然能够让他的《玄黄不灭》达到第四重，但绝对不会像如今这样第四重大成。
<br/>　　看着已经缩水一大半的化龙池，元始没有丝毫的不好意思，就好像这不是他干出来的事一样。
<br/>　　元始确实没有丝毫不好意思，化龙池虽然被他用了近乎四分之三，但里面的龙珠等他并没有破坏，只有再过一段时间，它还是会恢复原样的。
<br/>　　他元始没有将这化龙池彻底炼垮，或者将之搬走，就是对得起他龙族了。万事过之不及，借人家化龙池修炼就行了，没有必要将它完全占有或者破坏。
<br/>　　既然目的达到了，元始便不再多留，闪身出了龙墓，出现在混乱虚空之中。
<br/>　　之后，元始随便找了一个方向，便向着远处遁去。
<br/>　　又在混乱虚空之中度过数千年，之前是为了寻找龙族龙墓，现如今目的既然已经达成，他是在此地却是为了参悟空间法则。
<br/>　　没有什么地方比混乱虚空之中更好参悟空间法则的了，此处空间混乱，不断分层，空间各种坍塌，裂缝，风暴不断上演，空间法则清晰显现在眼前。
<br/>　　之前就已经领悟些许空间法则的他，现在对于空间法则已经不是那么毫无头绪了。通过空间门的转换，空间风暴的形成，对元始参悟空间非常有用。
<br/>　　“咦？此处怎么会有洞府？”
<br/>　　这天，元始来到有一个空间之中，但此空间并不像其他空间一样荒芜空洞，此空间之中竟然有一座仙家洞府。
<br/>　　不错，就是仙家洞府。
<br/>　　洞府之外，杨柳遍地，郁郁青青，随风飘扬。满园尽是各种顶级灵根仙草，仙鹤唳飞，松鼠雀跃，仙音袅袅，云气缥缈……
<br/>　　确是一座洞天福地！
<br/>　　但元始此时却是没有时间欣赏，因为他感受到这座洞府是有主的，而且主人还在家中，从这其中传来的气息来看，其主人修为一点也不比元始差。
<br/>　　这让元始开始谨慎起来，自己突然闯进别人洞府，说不得待会儿还要做过一场才是，由不得他不小心。
<br/>　　不过，他也并不是太过担心，因为他自信就算自己不敌，但对方想要杀了自己，那也是不可能的。他也自信没人能够留住自己。
<br/>　　“道友临门，何不进来一叙？”
<br/>　　就在此时，洞府宫殿之中，传来一阵缥缈的声音。紧接着，宫殿大门在缓缓打开。
<br/>　　元始犹豫片刻，便迈着步子进去其中，虽说这里是在别人的地盘，君子不立危墙之下，但，他并没有感受到其主人丝毫的恶意。
<br/>　　既然对方没有恶意，那进去一趟又何妨。
<br/>　　随着元始进入宫殿，宫殿之中却是不同外面，宫殿之外尽显华丽，但宫殿内却是一片自然，仅仅几座茅草屋，充满大道道韵。
<br/>　　茅草屋之前，一棵巨大的杨柳树屹立不倒，杨柳青翠欲滴，枝条碧绿，随风摇曳，一举一动，充满道韵。
<br/>　　绿柳之下，一个白头老翁盘腿而坐，他一身灰衣道袍，毫不起眼，眉毛长得搭地，却是白色眉毛。
<br/>　　元始见此，心中对老者的身份已有猜测。
<br/>　　只见他坐在绿柳之下，一人独自下棋，黑白两路棋子渭泾分明。元始不懂棋道，但不妨碍他看人下棋。
<br/>　　“道友所谓何名，在这洪荒之中，大能无数，但贫道还是多少有些了解，但为何没有听过道友之名？似道友这般，不该如此无名才对。”
<br/>　　只见那老翁转头看了元始一眼，缓缓道。
<br/>　　“贫道乃道印天尊元始，自出世就一直在洞府修炼，不曾闯荡洪荒，今日不久才出得洞府，道友不知亦是正常，若是听得贫道名号，那才叫奇怪。”
<br/>　　元始不缓不急，淡言道。
<br/>　　“哦。”
<br/>　　那人不咸不淡应了一声，便不再废话。
<br/>　　之后，元始又再次问道：“看道友如此，想必就是扬眉道友吧？”
<br/>　　“咦？”
<br/>　　这次，那道人明显有些许惊讶了，当然，这只是些许而已，之后，又再次恢复淡然之态，淡言道：
<br/>　　“不错，贫道确实是扬眉，只是贫道不喜争斗，也并没有常在洪荒之中走动，道友又是如何得之？”
<br/>　　元始笑而不语，扬眉老道也没有再问。
<br/>　　“此次却是贫道唐突了，贸然闯进道友洞府。”
<br/>　　之后，元始向扬眉道了一声。
<br/>　　“无妨，既然能在这无尽虚空乱流之中遇到，那便是与贫道有缘，其他的都不重要了。”扬眉毫不在意道。
<br/>　　“既然如此，那我们论道一番，如何？”
<br/>　　“善！”
