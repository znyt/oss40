　　这七尊圣者，联手之力，打出无尽灵力，近乎化为了一条长河，在长河之中，凝聚出了一尊无上金身。
<br/>　　“镇狱金身！”
<br/>　　这无上金身出现，顿时那七人便是同时催动灵力，向着楚风眠轰击过来。
<br/>　　这一尊金身的实力，丝毫不比真正的巅峰圣者要弱，而且没有生死大劫的威胁，可以全力出手，甚至是比起一尊巅峰圣者都要恐怖的多。
<br/>　　“镇！”
<br/>　　金身出手，一拳朝着楚风眠打出，这一拳之下，蕴含无尽镇压之力，仿佛是要将楚风眠当做一尊邪魔来，直接永世镇压。
<br/>　　楚风眠看了一眼那七人联手，脸色平静道极点，他手心一动，那金云便是被直接捏死。
<br/>　　这样一尊金家圣者，便是被楚风眠像是捏死了一只蚂蚁一样，轻而易举的给捏死。
<br/>　　“七个废物，学习了合击之术又如何？都给我破！”
<br/>　　瞥了一眼那金身，楚风眠眼神冰冷，大手一挥，一股灵力便是爆发而出。
<br/>　　楚风眠虽然只是随手一动，但是其中的力量，却是骇人到了极点，轰然之间，那金身便是开始节节破碎，其中的无尽镇压之力，被楚风眠轻而易举的给打破。
<br/>　　同时楚风眠的大手一抓，直接便是抓在了那金身之上，猛然一用力，这金身直接便是被捏出无数裂痕来，开始节节破碎。
<br/>　　“不可能吧！”
<br/>　　那七尊金家圣者，看着楚风眠手一捏，那金身便是破碎，眼睛瞪的斗大。
<br/>　　这金身的力量，可是真正巅峰圣者的力量，他们金家的合击之术，也是赫赫有名。
<br/>　　这样一尊足以媲美巅峰圣者的金身，就算不是楚风眠的对手，在他们看来，也是足以跟楚风眠抗衡一二的。
<br/>　　但是任由是谁都想不到，这样一尊足以媲美巅峰圣者的金身，居然是在楚风眠的手中，被如此轻易的捏碎。
<br/>　　他们七人联合，甚至是足以暂时抗衡赵无敌。
<br/>　　可是如今在楚风眠的面前，却是如此脆弱到不堪一击，一招便是将那金身打破，这得是有着多么恐怖的实力才能做到。
<br/>　　随手一招打破了那金身，楚风眠表情平静到极点，看着那七尊金家圣者，冷笑一声道。
<br/>　　“你们金家的人，都是乱臣贼子，人人得意诛之！今日我便是杀了你们，也让无数人看看，敢做这乱臣贼子的下场！”
<br/>　　楚风眠一边说着，一边出声，无尽灵力，轰击而出，要将那七尊金家圣者，全部轰杀。
<br/>　　“老祖救我！”
<br/>　　“老祖宗！”
<br/>　　那七尊金家圣者，一个个被吓得脸色惨白，他们可都是绝代圣者，一方大人物。
<br/>　　可是在楚风眠的面前，却都仿佛如同是巨兽面前的蝼蚁一样，瑟瑟发抖，根本无法还有敢跟楚风眠抗衡的勇气，一个个不由的大声喊道。
<br/>　　“何妨小辈！敢来跟我金家为敌！”
<br/>　　一声轰鸣响声，突然响起，就在金玄古城之中，一道古老而又强大的气息，猛然爆发出来。
<br/>　　一道身形，从金玄古城之中出现，直接便是化为了一道遁光，飞了过来。
<br/>　　金家的老祖，那一直隐藏的巅峰圣者，终于是出现了。
<br/>　　“老祖宗苏醒了！”
<br/>　　“好！”
<br/>　　“老祖宗苏醒，这小子肯定死定了！”
<br/>　　一尊尊金家圣者，听到这声音传来，好像就是抓住了最后的救命稻草一般，纷纷开口道。
<br/>　　这金家老祖，也是一尊巅峰圣者中的无敌人物，跟那秦皇大帝，赵无敌一样，有着冲击生死境的人物，距离度过生死大劫，只差一步之遥。
<br/>　　他的出现，令在场无数金家圣者的心，终于是安定了许多。
<br/>　　“哦？这金家老祖的实力，倒是不弱，比起秦皇国度皇室中的几尊老家伙，还要强上几分，不过倒是还不如秦皇大帝。”
<br/>　　楚风眠灵识一扫，这金家老祖的实力，他便是一眼看了出来。
<br/>　　这金家老祖，的确是有着冲击生死境的潜力，不过还不如秦皇大帝。
<br/>　　秦皇大帝，如今都死在了楚风眠的手中，这区区一个金家老祖，他根本不需要去在意。
<br/>　　“小辈！我们金家无意与你们皇室为敌！现在退去，给我们金家赔礼道歉，献上无数奇物来，今日的事，便是还可以就此过去了。”
<br/>　　一声古老沧桑的声音，仿佛便是从天际之上传来的一样，正是这金家老祖开口了。
<br/>　　“敢跟我们金家作对，皇室的那几个老古董都不敢！现在你一个小辈，胆大包天，敢来挑衅我们金家，已经是犯了大罪过！你可清楚！”
<br/>　　这金家老祖开口，一时间便是戳戳逼人，语气高高在上，好像是在教训一个小辈一样。
<br/>　　“今日的事，秦皇国度的皇室也保不住你，现在你跪下，赔礼道歉，才是你最好的选择。”
<br/>　　“金家老祖？哪里来的老家伙，修为不高，口气不小，还想要楚某赔礼道歉？别说你们金家，就算是赵家，就算是北邙学院，七大宗门都不配，你算是一个什么东西！”

<br/>　　他的大手一挥，顿时突破无数空间，就在这无数空间的背后，一尊白发老者，突然被楚风眠直接从无数空间背后，给抓了出来。

<br/>　　金家老祖被楚风眠直接从无数空间的背后摄出，脸上也是露出十分惊讶的神情。
<br/>　　金家的隐秘之术，也是赫赫有名的，金家老祖的隐秘之术，号称圣者之境无人能破。

<br/>　　可这如今，一个照面，他就被楚风眠直接从无数空间的背后给摄了出来。

<br/>　　“老祖宗的身形，鬼神莫测，在强大的圣者都无法察觉，曾经秦皇大帝都寻找不到，这小子怎么可能找的出来！”
