　　第474章
<br/>　　“大当家，我可以干些什么呀？”高飞虎问。
<br/>　　高飞虎没有参与这次行动，颇有些不甘心。程大雷派他去青牛山取出宝藏，结果是空手而归，他更觉有些丢人。所以现在迫不及待向程大雷请令，想要去草原上扬名。
<br/>　　“哈，会有你的事做，你先随我来。”
<br/>　　程大雷还真有些不放心高飞虎去参加行动，他手下的兵是从江南带来的，不想蛤蟆城的队伍经过日夜操练，至于战斗力么……反正程大雷不报任何信心。
<br/>　　高飞虎随着程大雷来到黄沙城的城主府，这里已成为一个临时的指挥中心。
<br/>　　外面打得如火如荼，程大雷这几日也没有闲着。秦蛮等人在草原上攻城略地，程大雷坐镇黄沙城，由喜字队负责情报工作，每天源源不断的情报汇报到程大雷这里，都是帝国军的动向。
<br/>　　程大雷处理过后，再由快马将情报递给秦蛮五人，方便他们调兵遣将。
<br/>　　蛤蟆军可以神出鬼没，料敌在先，每次都能打敌人一个措手不及。一是因为神行技能，二就是因为这些情报，第三，是因为程大雷训练出来的蛤蟆军，的确有帝国军没有的战斗力……
<br/>　　情报配合，出色的组织能力，万众一心的士气……这些东西稍微拿出来些，便有资格给帝国军好好上一课。
<br/>　　千里草原成了讲台，十万帝国军是最听话的学生，而负责授课的是程大雷的一万蛤蟆军。这里没有人不听话，因为那些不听话的学生，已经死在蛤蟆军的进攻下。
<br/>　　高飞虎走入这『指挥中心』时完全惊呆了，他还没有见识过蛤蟆军进入战斗状态时是一个怎样的情形。这里挤满了人，敌人堆满用废的纸张，不断有人进出，给这些口干舌燥的人端水，也有新的情报汇聚到这里。
<br/>　　这还只是在并州临时组建的指挥中心。假若在蛤蟆城，将有一座军策府，上百名谋士为程大雷出谋划策，分析敌军的动向。
<br/>　　“大当家，我……”
<br/>　　高飞虎还准备说些什么，却见程大雷已忙碌起来。
<br/>　　“大当家，有新的情况？”阿喜来到程大雷身前：“相府的军队在调动，将军府的军队在调动，扬州、青州、冀州、豫州……他们都在调动部队，应该是冲着黄沙城而来。”
<br/>　　阿喜的话里，透漏出一个令人不安的事实，在蛤蟆军猖狂过一阵后，逼得各路诸侯同仇敌忾起来，他们联合起来，调动大军，要攻击程大雷的临时大本营。
<br/>　　诸人打得主意，定然是攻占黄沙城，如果能生擒程大雷，在千万人面前活剐了他，那就更加大快人心了。
<br/>　　程大雷倒是很镇定，将阿喜递过来的情报翻了一遍，轻轻扣在桌子在。
<br/>　　“应该能想到的，分散开来打不过，就只能将拳头合拢。不过，他们竟然到现在才达成合作，还真是出乎预料啊。”
<br/>　　蛤蟆军虽然在草原上兴风作浪，但大部分战斗都属于抽冷子，打闷棍，至于正面冲击敌人的大本营，则一次都没有，也同样是程大雷尽力避免的事情。
<br/>　　如此状况下，帝国军自然不服气，便会集结兵力，要与程大雷正面一战。
<br/>　　只不过，他们的执行力还是远远低于程大雷的估计。
<br/>　　“大当家……”高飞虎又开口。
<br/>　　“好啦，好啦。”程大雷抖着手中的情报：“瞧见没有，瞧见没有，事情到了。”
<br/>　　“大当家，下一步咱们该怎么应对？”刘悲。
<br/>　　“把人召回来，准备决战。”
<br/>　　那一日正午，黄沙城的城头焚起狼烟，黑色的烟柱直冲天际，方圆百里都清晰可见。
<br/>　　黄沙城五十里外，一场战斗正在进行，为首一员将，手持蛇矛在队伍中横冲直撞。敌方的指挥官早已被他挥矛砍掉脑袋，此时他杀戮的不过是一些宵小。这样的战斗，本不用大将亲自冲锋，但张三爷何须人也，天上的魔君降世，黄泉的杀胚出土，一杆长矛过处，敌军血肉分离。
<br/>　　张字军的战士也完美继承他的作战风格，打起仗来不要命，人人都敢顶着敌方的箭雨冲锋。
<br/>　　在这几日，毫无疑问，举张字旗的蛤蟆军是帝国军最不愿碰到的敌人。
<br/>　　这场战斗已接近结束，剩下的就只不过是打扫战场而已。张肥抹掉脸上的血，看到了远处的狼烟。
<br/>　　“走喽，走喽，该回去了。”
<br/>　　于此同时，关鱼、秦蛮、赵子龙、高飞豹都看到了这冲天的狼烟。
<br/>　　关鱼已经缀着一支敌军长达三天之久，慢慢耗掉对方的体力，令敌人的精神崩溃，此时也正是敌人最虚弱的时候。但在收到这信号后，他立刻转身，向手下下令。
<br/>　　“收兵。”
<br/>　　秦蛮将铁枪从一个敌人的尸体中抽出，道：“打扫战场，把该杀的人杀干净，回城。”
<br/>　　三千龙字骑奔腾如虎，此时已向蛤蟆城方向疾奔。
<br/>　　……
<br/>　　放出去的五个魔头，杀胚在收到信号后，同时选择返回大本营。
<br/>　　于此同时，相府、将军府、扬州兵、青州兵、豫州兵、冀州兵等等诸侯也看到了这道狼烟。
<br/>　　尉迟离面露困惑，猜不透敌人的动作，可不知怎地，心中已有了不好的预感。
<br/>　　在黄昏时分，五路军已经陆续回到黄沙城，缴获的战利品归入仓库，及时治疗伤员，另外清点各路军的伤亡情况。
<br/>　　秦蛮五人聚在一起，这算是制定报复行动后的第二次军事会议。
<br/>　　程大雷简单向他们说了一下情况：“帝国军出现大规模调动，不出预料的话，他们将会大举攻来，寻找决战的机会。”
<br/>　　这次帝国军集结的兵力在十万左右，兵力比在十比一，纵然蛤蟆军可以以一敌十，但情形也依旧不太乐观。
<br/>　　但也不必妄自菲薄，一万戎族骑兵可以横扫边境的帝国军，程大雷觉得，现在的蛤蟆军并不弱于帝国军。
<br/>　　“大当家，下一步咱们的计划怎样？”高飞豹迫不及待的问。
<br/>　　程大雷的手指扫过不久前才绘制的战略地图，最后停在一个地方，缓缓吐出四个字。
<br/>　　“此地决战。”
