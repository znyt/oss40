　　五行逆转大阵之中，无数古帝正在其中交锋，少帝突然一指轰击，不少的古帝都是被他所杀。
<br/>　　“滚！”
<br/>　　听到五行刀圣的话，少帝冷喝一声，丝毫不顾，身形猛然间冲了楚风眠冲了过去。
<br/>　　眼神之中蕴含这恐怖的杀意，丝毫不顾五行刀圣，向着楚风眠追杀而来。
<br/>　　上一次楚风眠从他的手中夺走了青铜天魔骸骨，让少帝想要利用青铜天魔骸骨，一举冲击天人境的计划被彻底打破。
<br/>　　这可是对少帝来说，冲击天人境最好的机会，可最终却是被楚风眠所破坏。
<br/>　　今日再度见到楚风眠，少帝心中的怒火猛然爆发而出，不顾一切也要杀了楚风眠。
<br/>　　少帝的遁光速度极快，刹那之间踏入到了五行逆转大阵之中，顿时五行逆转大阵之中力量爆发而出，向着少帝轰击而去。
<br/>　　“崩灭！”
<br/>　　少帝脸色一寒，长啸一声，在他的身边滚滚灵力爆发而出，这滚滚灵力接触的一刻，无数强者的身形当即崩灭。
<br/>　　天空震荡，少帝身上的力量化为了一片黑云，围绕在他的身边，一切种种力量在接触到这黑云的一刻，都被当即崩灭。
<br/>　　五行逆转大阵，都在这一股力量之下被打出了一个缺口，其中的无数裂痕破碎，几近崩塌。
<br/>　　这五行逆转大阵，足以抗衡数十位九劫古帝的同时出手轰击，可如今少帝一出手，这五行逆转大阵，已经是几近崩溃。
<br/>　　“不好，这五行逆转大阵，都拦不住少帝？”
<br/>　　楚风眠看着背后，五行逆转大阵的崩溃，也是脸色大变，他冲入到了这五行逆转大阵之中，也是想要借助这大阵之力，去阻拦少帝。
<br/>　　可这少帝的实力比起楚风眠想象之中的还要恐怖的多。
<br/>　　随手一击，五行逆转大阵，都被打碎。
<br/>　　少帝的身形，所向无敌，近乎没有任何人足以拦得住他。
<br/>　　楚风眠的心中也是露出了几分焦急，一个五行刀圣，已经是无比棘手了。
<br/>　　现在更是来了一个比起五行刀圣还要棘手的多的少帝。
<br/>　　楚风眠目光疯狂的闪烁着，在观察着周围的一举一动，想着对策。
<br/>　　突然之间，那佛寺入口之处的大门，猛然敞开。
<br/>　　之前这佛寺入口的大门，被五行逆转大阵所笼罩，现在随着少帝的出手，五行逆转大阵被破。
<br/>　　这佛寺入口，也已经是没有了任何的阻拦。
<br/>　　“冲！必须要冲进佛寺之中！”
<br/>　　楚风眠没有丝毫的犹豫，身形一闪顿时向着佛寺入口的方向飞去。
<br/>　　他现在距离这佛寺入口方向，不过只有两三里的距离。
<br/>　　想要从少帝，五行刀圣的手中逃走，几乎是不可能的事情了，唯有进入到了这佛寺之中，才有着一线生机。
<br/>　　“想要逃进去？天地封锁！”
<br/>　　看着楚风眠的身形向着佛寺入口的方向飞掠而去，少帝也明白了楚风眠的想法，顿时冷笑一声，猛然一掌打出。
<br/>　　这少帝的一掌，笼罩了这方圆百里，在这方圆百里之中的古帝，都被这一掌的力量所压制着，在空中，寸步难行，就连遁光都无法催动。
<br/>　　少帝的实力，已经是无限接近于天人境，他猛然出手，一招镇压，方圆百里的空间，都被这少帝所镇压。
<br/>　　楚风眠感觉到他的身躯被一股股巨力不断的挤压着，哪怕是楚风眠有着始祖天龙血脉之力，在这镇压之下，都是寸步难行。
<br/>　　两三里的距离，对古帝来说是弹指一挥间的事，可对现在的楚风眠来说，却是无比的遥远。
<br/>　　在场所有的强者之中，唯有五行刀圣一人，没有收到影响，但是他的目光，却是一直看着楚风眠，丝毫没有出手的打算。
<br/>　　这五行刀圣，也看的出来，少帝如今出手，是为了杀楚风眠，既然如此五行刀圣也不打算要插手其中了。
<br/>　　毕竟对五行刀圣来说，他的目的，也是杀了楚风眠，至于是少帝杀的，还是他所杀的，都一样。
<br/>　　“该死！”
<br/>　　楚风眠看到五行刀圣的举动，眼神之中露出了几分阴霾。
<br/>　　他本来也有着打算，要让少帝跟五行刀圣交手，好让他有着逃出生天的机会。
<br/>　　但是现在看来，五行刀圣宁愿是牺牲几名五行天弟子，现在也不愿意插手其中，坐山观虎斗，看着少帝来追杀楚风眠。
<br/>　　滚滚黑云，向着楚风眠笼罩而来。
<br/>　　情况危机到了极点，曾经楚风眠也挑战过无数强者，但是只有如今的少帝，让楚风眠感觉到，纵然是楚风眠拼劲一切，都不是这少帝的对手。
<br/>　　这种实力，已经完完全全的超越楚风眠，纵然是今日楚风眠奇遇在多，都难以全身而退。

<br/>　　少帝身形闪烁，来到楚风眠面前，双瞳看着楚风眠身躯之上，显现而出的血肉晶体，露出疯狂的杀意道。

<br/>　　少帝冷喝一声，滚滚灵力镇压而下，这少帝，仿佛便是万魔之帝，高高在上。

<br/>　　楚风眠脸色之中露出了几分疯狂，也不顾一切了，施展出他最大的底牌。
<br/>　　一枚古老破碎的瓦罐，突然间从楚风眠的身躯之中飞了出来，转动而出，爆发出一股白色的光芒。
<br/>　　“人皇之力！教化众生！”
<br/>　　从这古老瓦罐之上，爆发而出的白色光芒，疯狂的冲击这周围的魔意。
