　　“喂，那个什么联合行动司令部现在控制权归你了。”蛰龙对着罗昊说道：“那些人是生还是死，一切你说了算，我对他们不感兴趣。不用谢我，我会不好意思的。”
<br/>　　说完，顺着那名战略行动组队员放下的绳梯爬下了货轮，进入特种潜航器中。
<br/>　　特种潜航器充气囊被释放掉，特种潜航器再次沉下海面。蛰龙可不傻，他才不会驾驶着特种潜航器在海面上行驶，这样就是把自己的行踪暴露给美国人的间谍卫星，给自己找不痛快。
<br/>　　货轮上这么多人就这么眼睁睁的看着这艘特种潜航器沉下海面，消失在自己的眼前。罗昊、李然这些龙隐部队队员心中是无比畅快，而詹姆斯、麦克米兰这些人心中却是无比憋屈。
<br/>　　虽然詹姆斯用联合行动司令部中那些人来威胁罗昊交出P病毒，这样的手段算不上什么正大光明，但他是个军人，站在星条旗下宣过誓的军人，一切以美利坚的利益为上，所以即便詹姆斯的手段不怎么干净，但是麦克米兰也没觉得有什么不妥。
<br/>　　在执行那些得不到政府承认的“黑色行动”时，麦克米兰还有他手下那些队员的手段同样也不干净。
<br/>　　“将军，麻烦您跟我们走一趟吧。”罗昊把詹姆斯拉到自己面前，说道：“麻烦您让人把我们的人送到机场，并且保证他们安全的坐上回国的飞机。我这人胆子小，如果受到了惊吓，我自己都不知道会做出什么事情来。”
<br/>　　敢绑架美国太平洋舰队司令，就这样了还胆子小？詹姆斯冷哼一声，在心中把那个“F”开头和“S”开头的单词，美利坚的国骂骂了一遍又一遍。
<br/>　　詹姆斯被罗昊拽上了他过来时搭乘的直升机，詹姆斯离开前对那个代号“JR”的DEA卧底投去一个眼神，JR会意的点了下头。
<br/>　　罗昊让直升机返回那个军用机场，在那里换乘湾流商务机，利用卫星电话跟大使馆进行联系，并且跟国内把情况进行了详细的汇报，这样一来，詹姆斯就算是有想法，也不敢再轻举妄动。挑起两个超级军事大国之间的战争，这样的责任詹姆斯担当不起，他不傻，他知道自己应该怎么做。
<br/>　　所以，不管心中有多愤怒，有多憋屈，就算是把肝都气爆了，詹姆斯也都只能当一只忍者神龟，忍着！
<br/>　　机场上，龙隐部队和大使馆方面的人汇合。罗昊站在一架国航A340客机前，跟詹姆斯“友好”的握了握手，说道：“将军，希望我们有机会再见！”
<br/>　　詹姆斯皮笑肉不笑的咧了咧嘴，没有答话。
<br/>　　龙隐部队众人登上A340客机，登机梯撤离，客舱门关闭，飞机滑出停机位，进入跑道滑行，等待塔台的指令，准备起飞。
<br/>　　詹姆斯身旁的作战参谋对詹姆斯说道，“将军，我调查过这架飞机的航线，我们可以在途中对他们下手，让他们消失在太平洋上空。”
<br/>　　“你是没带脑子吗？还是脑子里塞得都是大便，是****！？”詹姆斯转身对着自己的作战参谋臭骂，“如果这架飞机凭空消失，谁都会知道是我们做的手脚，就算是没有证据，也会怀疑到我们头上，你想遭到幽灵刺客无休止的追杀吗？你想让你的老婆孩子成为他们的陪葬品吗？你想让中国人的洲际导弹落入美国本土吗！？现在中国已经不是十年前那个实力孱弱的中国了，现在他们完全有实力跟我们打一场新世纪的大战，你的愚蠢甚至会引发第三次世界大战，到那个时候你就会出现在历史课本上，被所有人认识。”
<br/>　　被詹姆斯一通筹码的作战参谋脸色惨白，无言以答，额头上满是冷汗。
<br/>　　在得到塔台方面的指令后，A340客机开始加速滑跑，然后起飞，消失在天空云层中。
<br/>　　等到飞机起飞后，罗昊才察觉到在客舱最后那排座位上似乎多了几个人，一回头，看到蛰龙然也在这架飞机上，而之前他们居然没有人发现他。
<br/>　　“我草！”罗昊冲着蛰龙问道：“你这货怎么也在这儿？”
<br/>　　“我靠，要不是我帮忙，你老婆，P病毒都危险了！”蛰龙把装着P病毒瓶子的背包放在他前排座椅上，说道：“老子帮了你这么大的忙，搭个顺风航班又怎么了？用得着这么大惊小怪吗？再说了，这飞机又不是你家的，至于吗？”
<br/>　　罗昊解开安全带，走到最后那排座椅上，对蛰龙问道：“回国？”
<br/>　　蛰龙轻轻摇头，说道：“你也知道我不会回国的。”
<br/>　　“还是因为你母亲家那边的原因？”罗昊说道：“他们也都已经接纳你和你父亲了，也对当年的事情表示出了愧疚，你总不能让他们拉下面子跟你这个小辈道歉吧？”
<br/>　　“我不是要他们跟我道歉。”蛰龙摇头，回答道：“我要他们跟我的父亲道歉，这是他们欠我父亲的，必须要还，这点没有任何商量！只要我的父亲没醒，他们没有道歉，我是不会回去的。”
<br/>　　“你就不怕他们……”罗昊挥了挥手，说道：“算了，当我没说，除了你自己愿意，不然没有人能抓住你。”
<br/>　　这架国航A340在东南亚某国做了一个短停，而在此期间，蛰龙则是在没有人知晓的情况下，留下那个装有P病毒瓶子的背包离开了飞机，没有人知道他是什么时候离开的。
<br/>　　蛰龙是很多政客、军火商、军阀、军事承包商的噩梦，但他却是孤独的，有家却不能回，就像刺客一样，因为刺客都是孤独的！
<br/>　　罗昊靠在椅子上轻轻叹了口气。
<br/>　　云熙轻轻拍了拍他的手背，说道：“蛰龙心中有心结，就跟当年的你一样，只是他的心结要比你深，等到什么时候他自己把心结解开了，他也就回去了。”
<br/>　　“在他父亲醒来之前，他的心结想要解开，很难！”罗昊说道：“他是幸运的，也是不幸的。他得到了很多人得不到的东西，但是也失去了很多东西。”
