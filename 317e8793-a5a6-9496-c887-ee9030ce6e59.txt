　　慕清霄额头上已经溢出许多冷汗，脸上肌肉不停的抽搐，整张俊逸的脸庞都扭曲起来，血迹隐隐自牙缝中渗出。
<br/>　　此时，体内的苍龙虚影，带着势如破竹之势，在隐藏的狭窄经脉中，横冲直撞，宛如冲破堤坝的河水狂涌而过，狭窄的经脉瞬间膨胀。
<br/>　　苍龙生前就是半步元婴境界，虽本体已死，但精神力量依旧是一股不可忽视的强大力量，哪怕慕清霄用上剑意，一时半会都无法抹杀。
<br/>　　狭窄的经脉膨胀，因为无法防御精神力量的冲击，经脉上居然出现极为细小的裂痕，裂痕密密麻麻，经脉不断的膨胀！
<br/>　　若非心法已经将慕清霄的经脉，改造的极其坚韧，否则继续这样膨胀下去，难免逃不出经脉寸断的下场。
<br/>　　就在这时，筋脉的墙壁上，忽然涌出无数缕细微的金色火焰，伴随着金色火焰的出现，裂痕缓缓收缩，最后以肉眼可见的速度，完全愈合。
<br/>　　待经脉愈合后，原本不能抵御苍龙精神力量冲击的经脉，居然变得比之前要坚韧的多，而且经脉也比之前宽阔许多........
<br/>　　观察着体内的一幕，虽然剧痛难忍，但慕清霄依旧咬牙坚持，发现自己的经脉变得宽阔不少，眼中却闪烁着兴奋的光芒。
<br/>　　他非常清楚，疼痛是暂时的，但疼痛的过后，却能给自己带来无尽的好处。
<br/>　　只要隐藏的筋脉打通，那他以后调动灵气的速度和量，必然会翻倍，如此一来，在战斗中就不用担心灵气问题，这样能占据绝对的优势。
<br/>　　经脉恢复，苍龙的精神力量依旧乱窜，但疼痛却减轻许多，最后无数道洞穿的一声音，如同河流汇聚到大海。
<br/>　　慕清霄躯体表面渗出许多的黑色污垢，污垢中甚至参杂这不少黑色血迹，这是他多年来所留下的暗疾，也随着经脉的畅通而愈合。
<br/>　　污垢顺着毛孔渗出，散发着一股恶臭，但慕清霄却仿若未觉，仔细的观察着体内的动静。
<br/>　　经脉畅通，污垢排出体外后，慕清霄顿时觉得神清气爽，思维比往日里更加的清晰。
<br/>　　只要他运转心法，打通的经脉就如同通风口一样，迅速的吸收外界的能量，周围稀薄的灵气都顺着毛孔钻进体内，在四肢百骸中流淌。
<br/>　　现在调动灵气的速度，简直是打通经脉前的四五倍，没想到苍龙虚影竟无意间帮了他的大忙。
<br/>　　经脉中的疼痛已经减轻，这让慕清霄稍稍松了口气，终于不用在忍耐剧烈的疼痛。
<br/>　　同时心中也非常的庆幸，好在自己没有立刻让紫嫣她们吞服龙元，这种剧痛连自己都难以忍耐，他不想让自己的女人，体会这种痛楚！
<br/>　　此时，龙元中的生命力，依旧源源不断的被牵引进体内，完全没有减少的趋势。
<br/>　　越是如此，慕清霄越是激动，如此磅礴浩荡的生命之力，长生不老绝对不是枉然，
<br/>　　只是让慕清霄好奇的是，刚刚经脉中所涌出的金色火焰，那分明就是凤凰的本命火焰，看来自己炼化的凤髓还有开发的空间。
<br/>　　因为，他能感觉到，自己与金色火焰之间，存在着某些联系，仿佛那就属于自己。
<br/>　　他可没忘记，当初在凌云窟最底部的岩浆领域中，凤凰所发射的金色火球，那种恐怖的温度与能量。
<br/>　　若非自己能释放出强大的剑气，想要与凤凰同归于尽，无异于痴心妄想！
<br/>　　自己将凤髓炼化，血液也进化成了金色，就是不知道能不能掌控那种火焰，若是能够将之掌控，绝对是一大杀招！
<br/>　　然而，让慕清霄失望的是，那种金色的火焰，只有在自己受伤的时候，才会冒出来帮自己治愈伤势，存在于身体内何处他都不知晓，谈何掌控。
<br/>　　想到这里，慕清霄收敛心神，将心思放在体内的苍龙虚影上，虚影依旧在体内乱窜，与剑意互相对峙，几乎势均力敌。
<br/>　　心法在炼化生命力，好在剑意强悍，几个循环下来，就将苍龙的精神力量抹杀近半，想要完全抹杀，还需要一些时间。
<br/>　　........
<br/>　　........
<br/>　　半个时辰后，龙元内部的生命之力，全部被牵引进慕清霄的体内，赤红色的光晕也逐渐消散，变成半个拳头大小的普通圆珠，有些类似夜明珠。
<br/>　　与此同时，苍龙虚影在体内不断肆虐，慕清霄全速运转心法，以最快的速度将生命之力炼化，体表上的污垢逐渐脱落，散发出乳白色的光晕。
<br/>　　“不愧是半步元婴，本体死亡后都拥有如此的精神力量........”
<br/>　　慕清霄嘴角勾起一丝邪异的笑容，虽然苍龙虚影在体内疯狂肆虐，但已经不能给他带来任何的伤害，最多也就是细微的疼痛感。
<br/>　　经脉扩张后宽阔许多，而且比之前坚韧数倍，任由苍龙虚影在里面乱窜，也无关紧要。
<br/>　　当然，谁也不会希望自己的体内，拥有自己以外的东西存在，尤其是这种凶戾之物。
<br/>　　此时，生命之力剩余不多，慕清霄终于能够运转心法，与剑意抹杀精神力量。
<br/>　　同时，他惊喜的发现，自己的境界在不知不觉中，已经抵达金丹八层巅峰的零界点，即将迈入金丹九层境界。
<br/>　　虽然金丹八层与金丹九层只有一步之遥，可想要破开壁障，还需要庞大的能量，好在生命之力还剩余不少........
<br/>　　“生命之力力量温性，但其中的力量也不可忽视，让自己突破境界应该是足够了。”
<br/>　　慕清霄在心中沉吟片刻，任由剩余的生命力在四肢百骸中流淌，开始全力运转心法，利用上剑意，两股力量融合直接朝苍龙虚影杀去。
<br/>　　这股精神力量极其强大，却已经不足以给自己带来任何伤害，就算如此，慕清霄也不允许自己体内，有其他的力量存在。
<br/>　　哪怕他愿意，恐怕魂剑也不会同意。
<br/>　　两股力量在体内相撞，互相交汇，慕清霄咬牙忍耐，不断运转心法，剑意与苍龙碰撞，不断抹杀着苍龙的意志。
<br/>　　“昂昂!”
<br/>　　半步元婴的意志力量确实可怕，可依旧逃脱不了被抹杀的命运！
