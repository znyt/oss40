　　第182章
<br/>　　戎族大帐。
<br/>　　各部落的头领进入大帐，离开时每个人都会发出一声长叹。有巫医端着绿色的药汤，递给榻上的男人。
<br/>　　“王子，药已经熬好，您把它喝了病就能好。”
<br/>　　赤眉拥着厚厚的毛毯，强撑着身体坐起来，药液咕咚咕咚灌入腹中，下巴挂着绿汁。
<br/>　　从回来后，赤眉就病倒了，身上冷一阵热一阵，乏力得连刀都握不紧。
<br/>　　自家人知道自家事，他知道自己的病什么药都治不好，自己患的是心病，心病还需心药医。
<br/>　　他们为何能识破自己的诱兵之计？
<br/>　　卧榻这几天，一直在思考这个问题。难道是自己的表演不够完美么，还是因为对方有千里眼，顺风耳？
<br/>　　这种敌人了解自己，自己却看不透对方的感觉实在糟糕。
<br/>　　部落中带出来的族人已经伤亡过半，有何面目回部落，有何面目去见那位从不正眼看自己的父王。
<br/>　　习惯了胜利的赤眉，终于发现失败原来是件这么讨厌的事情。
<br/>　　“弟弟，你真令人失望呐。”
<br/>　　人未到，声先至，刚听到这声音，赤眉就皱起眉头，冷冷的看着帐门。
<br/>　　一道人影缓步而入，穿着书生儒袍，手中折扇，腰间佩剑。看这幅打扮像极了江南或京州的才子，但他深陷的眼窝，金色的眸子还是暴露了他的身份。
<br/>　　北蛮部九王子，金问道。
<br/>　　金问道的确是北蛮部的九王子，至于他究竟是不是北蛮王的第九个儿子，怕是没人知道。不过能够封王，被人称作殿下，证明他在北蛮王眼里还是有些重要的。呼尔勒死后，他是极有力的王位竞争者。
<br/>　　此人虽生在北蛮部，但却挚爱帝国文化，服装饮食都以模仿帝国为荣，自己还给自己起了个帝国名字。
<br/>　　“九哥，你这么来了！”赤眉脸上的冰冷敛去，换上热切的笑容。
<br/>　　“若你中用些，我需要亲自来么。”
<br/>　　金问道用折扇敲在赤眉头上，像呵斥一只幼犬，赤眉眼底寒芒一闪而逝。
<br/>　　“父王要的东西，你拿到没有？”金问道。
<br/>　　“敌人十分狡诈，暂时还未……。”
<br/>　　“打了二十日，还拿不下一座破城，亏你还在父王面前夸下海口。”金问道摆摆手：“我已来了，现在由我接管一切。”
<br/>　　多么趾高气扬吧，这种习惯发号施令的语气还真是令人讨厌！
<br/>　　北蛮王有诸多孩子，本来呼尔勒最受宠爱，但呼尔勒死了，其他王子才有了争夺王位的机会。
<br/>　　有希望争夺王位的一共十人，大家都在想方设法拉拢草原上的部落，给自己称王的路上添砖加瓦。
<br/>　　像这次随赤眉出征的小山部、犬丘部等都是势力较弱的部落，所以选择投靠势力较弱的十王子赤眉。若这一战能打赢，那么这些部落将成为自己的争王的筹码。
<br/>　　可现在打输了。
<br/>　　金问道一来就要接手统帅的位置，那么，自己辛苦拉拢的势力，都将成为他的人。
<br/>　　“九哥，我的病并无大碍，依旧能够领兵，我必打破蛤蟆寨，取回父王要的东西。”
<br/>　　“有影响的不是你的病，而是你的脑子，你太笨了，根本不懂得打仗是怎么回事。”金问道又一次用折扇敲击赤眉的脑袋：“现在我将召集他们议事，如果你的病死不了，就睁大眼睛看着，我如何三日破城。”
<br/>　　……
<br/>　　“喂，那个南人把马粪清理一下！”
<br/>　　几个戎族聚在一起，用手中石块砸向孟子云，看着他惊慌失措的样子，几人哈哈大笑。
<br/>　　孟子云却不敢流露出任何不满，他用铁锹将马粪装在粪车上，忍着恶臭。
<br/>　　不是他没有羞耻心，而是他已习惯了这种侮辱。所以有时候习惯真是一件可怕的事情。
<br/>　　但这毕竟是一件很累的事情，孟子云站在太阳底下发了会儿呆，看着远处起伏的山峰。
<br/>　　随之，不知怎么回事，他看到山峰上一棵杨树动了动。他以为是自己眼花了，又看了一眼，发现不仅杨树在动，树下还有一个人影。
<br/>　　孟子云的心砰砰跳动起来，他一直等待一个机会可以令自己被重用，然后带兵杀进蛤蟆寨，将程大雷千刀万剐。
<br/>　　如今，自己等待的机会仿佛到了。
<br/>　　他在黄昏悄悄离开大营，爬近那座山峰往上望，树下的确有个人，蓬头垢面，可孟子云还是一眼认出对方是谁。
<br/>　　林少羽。
<br/>　　他几乎是飞奔着跑回大营，闯入了金问道的大帐。
<br/>　　“你是南人？”金问道。
<br/>　　“嗯。”
<br/>　　“为何要加入我们？”
<br/>　　“我于那贼人有夺妻之恨，刻骨难忘。”孟子云咬着牙道。
<br/>　　“好，以后你就做我的马夫吧。”
<br/>　　“马夫……”孟子云难以相信，自己禀报如此重要的信息，得到的只是一个马夫的职位。
<br/>　　“放心，等你真杀了那贼人，会有你更好的位置。”
<br/>　　金问道笑笑，看着远处的牛角峰，忽然呵呵笑道：“有意思，有点意思。”
<br/>　　……
<br/>　　牛角峰。
<br/>　　林少羽躺在大石上，缓缓活动着双腿，身边摆放着面包和净水。
<br/>　　这不是一件太好的差事，先不说发潮的面包，冰冷的山石，便是长久无人说话的孤独，就可以令一个人神经崩溃。
<br/>　　但林少羽不觉得痛苦，反而有几分荣耀。
<br/>　　从未向别人说过，他乃是世家之后，不然也置不起一套行走江湖的装备。从小到大，耳濡目染着那些英雄故事，他也想成为那种传说中的人物，在世人口中流传。
<br/>　　直到迈入江湖，才知江湖艰难。
<br/>　　直到现在，林少羽才觉得自己正在做一件好事。驱除蛮夷，救黎民于水火，这当然是一件好事，做一件好事很重要，难道不是么？
<br/>　　夜来，漫天星斗。
<br/>　　忽然听的有走路声，借着夜光看去，只见爬向山峰的道路上，铁矛发着瘆人的寒光。
<br/>　　来了！
<br/>　　既然敢来这里，便注定有被发现的危险，不过是时间长短而已。
<br/>　　林少羽精神一震，握紧自己的铁枪，夜里枪身挂着露水，摸上去湿漉漉的。
<br/>　　他毫不犹豫的出枪，向靠近山峰的一人狠狠扎去。
<br/>　　一点寒芒先到，随后枪出如龙。
<br/>　　这一枪的名字是：破浪枪。
<br/>
