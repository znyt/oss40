夏夕绾花了几秒才将视频里那个清瘦的少年跟眼前这个高大英挺的男人结合在一起，“陆…陆先生，你怎么来了？”
<br/>
<br/>        陆寒霆居高临下的看着她，扬了一下手里的手机，他眼里是笑的，但是笑意不达眼底，像蓄积着两个小风暴，一不小心就能将人给吞没下去，“这句话应该是我问你，你来这里干什么，你看到了什么？”
<br/>
<br/>        夏夕绾从来没有看过他这样生气的样子，哪怕以前他发病的时候都没有这么阴冷可怕，“陆先生，你听我解释…”
<br/>
<br/>        陆寒霆伸手，直接将手机砸到了墙壁上，“砰”一声，手机全碎了。
<br/>
<br/>        男人那股凌厉的力道带着巨大的声响在夏夕绾的耳畔炸开了，她吓得脸色全白，惊魂未定的时候一只大手伸了过来，一把拽住了她纤细的皓腕将她从座椅上拽了起来。
<br/>
<br/>        “够了，陆寒霆，”陆茵茵迅速起身，“放开夕绾，是我约她过来的，你现在是不是很生气，又要失控了，把夕绾放开，不要伤害她！”
<br/>
<br/>        陆寒霆拽着夏夕绾没有松手，他狭长的眼梢里染上了几分猩红，就这样冷鹜的盯着陆茵茵，精硕的胸膛上下喘动着，从喉头里发出森然的音节，“我早就警告过你，把嘴巴给我闭上，看在奶奶的面子上，今天就给我滚出海城，这是我对你最后的仁慈！”
<br/>
<br/>        “陆寒霆，你竟敢这样跟我说话，我可是你姑姑，还有你是怎么知道我和夕绾在这里见面的，你是不是派人…跟踪监视我？”陆茵茵震惊的看着陆寒霆，她无法想象她24小时的生活都在别人的监控里。
<br/>
<br/>        陆寒霆冷冷的睨了陆茵茵一眼，然后鹰隼般的眼风给到了一个黑衣保镖，掀了掀薄唇，“把她给我带走。”
<br/>
<br/>        简单的字眼，是不容置喙的强势。
<br/>
<br/>        “是，总裁。”黑衣保镖迅速上前。
<br/>
<br/>        陆寒霆再也没有看陆茵茵，他拽着夏夕绾就走了。
<br/>
<br/>        被抓住的陆茵茵一脸惊悚的看着陆寒霆消失的身影，“陆寒霆，你是不是疯了，你竟敢把我关起来，你就是不正常，是一个神经病！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾被拽着手腕，因为男人步伐跨的很大，所以她跌跌撞撞的跟在后面，她看着男人凌厉的后脑勺开口道，“陆先生，你不要生气，你听我解释。”
<br/>
<br/>        陆寒霆的脚步倏然一顿，转过身，猩红的狭眸落在了她的身上，然后大手一伸，捏住她的脸蛋将她给拽上前，“夏夕绾，我跟你说过什么，你是不是全忘了，你想听什么，只要我能说的我都告诉你了，还有一些不能说的你就这么好奇么，我那样求你不要去听陆茵茵跟你说什么，你为什么不听话？”
<br/>
<br/>        昨天晚上陆茵茵去了医院，她跟陆茵茵在走廊上碰面，这些他都知道，昨晚她就扯谎了，不过他没有拆穿她。
<br/>
<br/>        他想给她一次机会，可是她让他太失望了。
