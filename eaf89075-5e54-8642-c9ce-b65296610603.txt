　　第312章
<br/>　　程大雷立在峡谷旁，向对面望过去，之间两侧宽度在三丈左右，也就是大约十米。其下深度有二三百米，还缀着塌掉的索桥。
<br/>　　“取绳索来。”
<br/>　　程大雷将大斧卸下，活动了一下双臂。想要走下去，不修桥是不行了，眼前虽然麻烦些，但却是一劳永逸的事情。
<br/>　　有人把绳索取过来，程大雷看到对面有棵歪脖子枣树，他在绳子一端系上一个绳套，然后让周围人散开。
<br/>　　大家都看明白了程大雷的意思，是想把绳索套在峡谷另一端的枣树上，中间隔着十米距离，即便擅长套索的黑岩、律鸿也没有把握。
<br/>　　看程大雷脸上的表情，定是觉得十拿九稳了，黑岩心中也是暗暗佩服。
<br/>　　绳索在头顶抡开，越转越快，见劲头稳了，程大雷手中的套索脱手而出。
<br/>　　所有人都注视着绳索去的方向，准备欣赏程大当家的表演，程大雷自己的弟兄已经准备拍手叫好。
<br/>　　嗖……
<br/>　　没有套中。
<br/>　　“……”众人。
<br/>　　“呵。”
<br/>　　也唯有樊梨花发出一声嘲讽的笑，她走过来将套索接了过去，然后顺着绳子一点点拽过来。
<br/>　　程大雷灰溜溜给她退出位置，来到秦蛮身边。
<br/>　　“大当家，你如果不行就不要摆出一副高手风度，很丢人的。”秦蛮。
<br/>　　“我有说过我是高手么，我只是试验一下好不好。”
<br/>　　只见套索从樊梨花手中离开，然后在空中划过一道弧线，稳稳的落在对面枣树上。
<br/>　　樊梨花将绳索扯紧了拽在手中，示威般的看着程大雷。
<br/>　　程大雷又没有要和她比试的心理，这姑娘这么好战，的确有些难以应付。
<br/>　　“程当家，靠这条绳索也没办法把人和货运过去吧？”黑岩现在也知道了程大雷山贼的身份。
<br/>　　程大雷把佩剑也取下，交给银眸拿着，自己走在悬崖旁。
<br/>　　黑岩还没明白程大雷要做什么，正欲发问，忽然见程大雷脚尖一点地，身子跳了起来。
<br/>　　“程当家，不要寻短见……”
<br/>　　黑岩的话没说话，自己就顿住了。只见程大雷身子在空中如鹞子翻身，借着绳桥一点，又一次高高跃起，再借着一点，身子已经出现在悬崖的另一端。
<br/>　　黑岩张开的嘴巴久久合不拢，良久，方才发出一声感慨。
<br/>　　“程当家真神人也！”
<br/>　　“一般，也就一般啦。”
<br/>　　程大雷把绳子在枣树上系紧，随着又让人丢绳子过来，连续数次，搭起一座绳桥。
<br/>　　其实有这道绳桥，已经可以把人和货运过来，关键是马匹。所以，程大雷想要的也并不只是一道绳桥。
<br/>　　“开始修桥吧。”程大雷在对面挥挥手。
<br/>　　大家在山上伐来超过十米的大木，在中间部开，搭在峡谷两端。只伐了两根大树，便建成一座简易的木桥。
<br/>　　只可惜山寨的建筑精通技能带不到这里，不然可以更坚固些。不过有这简易木桥，已经可以把货物马匹运过去，等程大雷下次来的时候，带上工具，可以将桥修得更扎实坚固。
<br/>　　人和货都运了过去，黑岩在另一端向程大雷挥着手。
<br/>　　“程当家，莫忘了我的花布？”
<br/>　　“一定，一定，请回吧。”
<br/>　　之后，程大雷便带着队伍返回，一来一去，足花了二十天时间。大家先在落玉寨落脚，清点这次交易换来的货物。
<br/>　　骏马超过一百匹，兽皮三百张，草药、玉石无数，还有一些稀奇古怪的小玩意。
<br/>　　马匹都留在落玉寨，程大雷若想培养一支骑兵，现在已经有了基础条件。首先落玉寨有养马的草原，而又可以从戎族买到上好的骏马。
<br/>　　至于玉石、兽皮则交给游九楼，他有下货的渠道，然后再由他换成粮食运到蛤蟆寨，之后，再与樊梨花、云中龙、乔鹤、游九楼四家分。而换来的草药，还是由程大雷亲自处理，毕竟他也有下货的渠道。
<br/>　　如今程大雷一人要养五座山寨，家大业大，一睁开眼就有无数人张口要饭吃，程大雷也感觉到了莫名大的压力。
<br/>　　将云中龙三人送走后，程大雷也要和秦蛮、福德勒一起离开。如今落玉寨已经稳定下来，不再需要秦蛮压阵，秦蛮也就可以带着蛮字队回大本营了。
<br/>　　不过，程大雷还要等一个人，就是他带回来的银眸。
<br/>　　这小姑娘不知在戎族经历了什么，满身污垢，又伤痕累累，由樊梨花带着她去洗澡，换套干净衣服。
<br/>　　程大雷三人在落玉寨大门外等着，忽然秦蛮眼睛一亮，用手捅了捅程大雷。
<br/>　　“大当家，你看。”
<br/>　　程大雷扭头看去，表情也是一怔，只见樊梨花牵着一个小姑娘走来。
<br/>　　她刚洗过澡，头发湿漉漉的垂在脑后，身上穿的一件白袍，大概是樊梨花的，她穿在身上自然有些宽大。用一根带子松松垮垮的系在身上。宛若埋在泥里的璞玉，洗干净污垢后露出了本来面目。只不过一双银眸还是十分冷漠，冷得像冰。
<br/>　　程大雷、秦蛮、福德勒都表达了惊叹，只有樊梨花表情有些不善，也不知为何。
<br/>　　所以樊梨花根本没有靠近，只是拍了拍银眸的肩膀，让她过来。
<br/>　　这银眸过走进程大雷后，便将她的大斧提在手中。
<br/>　　“嗳，你别碰这个，很重的。”
<br/>　　程大雷话还没说完，就见银眸已举起了斧头，一百单八斤重的鬼面斧，就连程大雷曾经也觉得沉重。可这小女孩手臂纤细，提在手中却如若无物。
<br/>　　看来铁力这隐藏属性果然不是白给的。
<br/>　　程大雷心中想着，日后有这样一个冷漠小姑娘为自己捧剑背斧，却也是一件很酷的事情。
<br/>　　心里想着，程大雷伸出手捏了捏她的鼻尖，这次银眸倒是没有张口咬程大雷，只是表情依旧很冰冷，像一座冰雕。
<br/>　　程大雷又在落玉寨拣了一匹小马，由她骑着，这姑娘毕竟是在戎族长大的，对于骑马并不陌生，甚至骑术比程大雷还要好。
<br/>　　程大雷也骑上马，众人踏上了返回大本营的路。
<br/>　　而此刻蛤蟆寨内，工程仍在如火如荼的进行，程大雷回来得及时，不然山寨的经济已经撑不下去这建设了。
