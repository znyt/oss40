<br/>				两人就着夜明珠喝着酒，聊着一些趣事，一边看着随着夜色渐深，夜空中隐隐闪烁着光芒的星星，云层之后，月牙也隐隐浮现，依稀将月光洒落。<br/>　　<br/>　　良辰美景，月下美人，本应是很惬意而愉悦的，然而，当一声惊慌带着哭腔的呼救声隐隐传入两人耳中时，正在说笑的两人微顿了一下，端在手中的酒杯也随着放了下来，不约而同的往那黑漆漆的山下看去。<br/>　　<br/>　　上山看日落的人原本就并不多，随着日落之后，那些人大多都没有在这山上停留，只有依稀的一些亮光在那山中若隐若现，仿佛是有人手中持着灯具一般。<br/>　　<br/>　　而那带着哭腔的呼救声，正正是从那山下隐隐浮现的亮光处传来。<br/>　　<br/>　　“这大晚上的，这地方怎么会有呼救的声音？而且似乎是女子？”凤九说着，目光落在那半山呼救处。<br/>　　<br/>　　“让吞云去看下便可。”轩辕墨泽说着，淡漠的收回目光。<br/>　　<br/>　　“嗯。”<br/>　　<br/>　　凤九点了下头，唤了一声：“吞云。”声音一落，吞云从空间中跃了出来，看了两人一眼后目光落在凤九身上：“主人。”<br/>　　<br/>　　“吞云，半山腰那里有人在呼救，你去看看是怎么回事？”她轻声吩咐着。<br/>　　<br/>　　“是。”吞云应了一声，健壮的身影如闪电般掠出，只听呼的一声风刃划过，下一刻便没入了漆黑的夜色之中，消失不见。<br/>　　<br/>　　夜风拂过，送来的不仅仅是那一丝丝的凉意，还有那半山腰处近乎绝望的呼救声，她一手托着脸颊，听着那声音，道：“这地方偏僻少人，尤其是夜色之下更是人烟稀少，若真遇到危险，只怕呼救也没人去救，真不知连自保的能力都没有，怎么会来这么偏僻的地方？”<br/>　　<br/>　　听着这话，轩辕墨泽轻笑着：“也许是有男子陪伴，只不过，并非每个男子都有我这般的实力，也并非每一名女子都如你。”<br/>　　<br/>　　闻言，凤九轻笑：“这倒也是。”<br/>　　<br/>　　两人说话间，听着那呼叫声还没停止，不免有些奇怪：“这距离，吞云应该是到了的，怎么那女子还在呼救？”<br/>　　<br/>　　“也许是吞云觉得不可救。”轩辕墨泽抿了口气，目光一扫，唇角微勾：“它回来了。”<br/>　　<br/>　　凤九侧头看去，见那密林之中，吞云窜了出来，没一会就来到她的面前。她看了有些奇怪，问：“吞云，你怎么回来了？那呼救的人莫不是没找到？”<br/>　　<br/>　　吞云呆了一会，道：“主人，找到了，可是，你不是让我去看看吗？所以我就去看看是怎么回事后就回来告诉你了。”<br/>　　<br/>　　听见这话，凤九也呆了一下，仿佛是没想到它竟会理解错一般。缓了半响才缓过神来，她神色古怪的看着它，问：“那你看到了什么？是什么个情况？”<br/>　　<br/>　　旁边坐着的轩辕墨泽见此微露出笑意，并没理会那一人一兽，而是再倒着酒喝着。<br/>　　<br/>　　吞云想了想，才道：“有几个散修将一男子打晕了绑在树上，正追着与男子同行的女子，找着她的衣服。”<br/>
