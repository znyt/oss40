　　“我们……一分钱没有了？”叶少阳望着翠云说道。
<br/>　　翠云紧咬嘴唇，一边哭一边摇头，“只剩下两块大洋了。都怪姐没给你看好，这可是你娶媳妇的钱……”
<br/>　　叶少阳拍着她的肩膀，不住安慰她，自己的心情也很是不好。
<br/>　　他当然不想在这个世界长住，娶媳妇什么的，他需要钱，只是为了解决这一路上的盘缠。
<br/>　　俗话说一分钱憋死英雄汉，从这里到龙虎山还有不少路程，没钱寸步难行，自己跟翠云总不可能一路要饭下山吧？
<br/>　　“好了，钱不算什么，没钱咱们想办法再赚，我有这一身本事，还怕赚不到钱吗。”叶少阳各种安翠云，翠云怕叶少阳担心，最后也不敢再哭，慢慢冷静下来。
<br/>　　“破财消灾，人没事就好，人没事就好。”老万也缓过神来，佝偻着脑袋，喃喃自语。
<br/>　　叶少阳转头看去，见老万和陈毛子一副垂头丧气的样子，心中也有些不忍。
<br/>　　老万长叹一口气，站起来，望着面前的几具尸体，想起之前几具尸体轮流诈尸、扑杀那些士兵的场面，顿时有点紧张起来，颤声说道：“我们还是快走吧，这些尸体万一再起来，我们手上可没枪……”
<br/>　　他当然不知道之前诈尸是叶少阳捣的鬼。
<br/>　　正好外面这时候雨也停了。
<br/>　　几人各自收拾行李，离开山神庙之前，翠云问怎么处理二蛋的尸体。
<br/>　　老万和陈毛子对二蛋的人品嗤之以鼻，根本就不想管他的尸体，翠云却是觉得人死为大，让叶少阳帮着收拾一下。
<br/>　　这荒山野岭的，叶少阳也没办法处理尸体，总不能带下山去，最后也只能在外头找了根树枝，在被雨浸湿的土地上掘了个浅浅的坑，把二蛋的尸体埋进去了事。
<br/>　　四人一路下山，因为担心遇到那几个逃兵，山路都没敢走，而是从泥泞的丛林里穿过，走的很艰难，最后来到了山脚下一个荒村里，这个村子的房屋倒得都差不多了，大部分只剩下一些地基。叶少阳一行人找了个干净的地方坐下来，讨论接下来怎么办。
<br/>　　老万和陈毛子来江西都是做生意的，哪里想到会遇到这种事，真是欲哭无泪，按照老万的理论，真要是遇到土匪，被抢走钱也就算了，因为土匪就是干这个的，但是被一帮还没当上土匪的逃兵抢走了所有，老万和陈毛子真的有点难以我接受，一头撞死的心都有。
<br/>　　做买卖的本钱没了，再往江西腹地走，对两人来说也没意义了，还好带来的货物还在，两人打算下山后就近找一座县城，把带来的货物卖了，至少够回去的盘缠，这一趟虽然没赚到钱，倒也不算太亏，毕竟他们来办货，早就地方着土匪，没有带太多的钱。
<br/>　　叶少阳要去龙虎山，当然只有继续往前走。翠玉跟着他一起。于是四个人在山脚下分道扬镳了。老万把自己的地图拿出来，送给叶少阳，给他好生解释了一番怎么赶路，“我给你画的这几个地方，坚决不能走，这几个地方，都是闹过土匪的。”
<br/>　　叶少阳仔细辨认了一下，指着地图上离自己不远的一处地方说道：“这个地方为什么要圈起来，也是闹土匪？”
<br/>　　“这里不是闹土匪，而是公堂义庄。”老万说道，“这个地方，你可也不能去，一定要绕路走，那些赶尸匠规矩很多的，可不要冲撞了他们。”
<br/>　　“赶尸匠？”叶少阳怔住，“赶尸不是只有湘西一带才有吗，怎么这里也有？”
<br/>　　老万道：“那不知道，反正都说那里是公堂义庄，有几个赶尸匠在那里住，专门负责往南方赶尸，你可千万要绕路走，别跟他们撞见！”
<br/>　　叶少阳道了谢，询问赶尸的详情。老万也不太清楚，只是听说有几个赶尸匠长期在那个义庄里住，去周围几个县城揽生意额——所谓的生意，也就是死人，跟死者家属谈好价钱，然后赶回老家下葬。
<br/>　　关于赶尸，陈毛子也听说过一二，告诉叶少阳。叶少阳加上自己的理解，总算了解清楚了大致的情况：
<br/>　　从晚清开始，以胡雪岩为代表的徽商崛起，皖南十分富庶，李合肥在政治上对安徽也很有照顾，因此周边地区去安徽做工的人很多，一直到了民国，段总理就是皖系，安徽是他的大本营，重视程度自然不必说，大力发展工业。
<br/>　　有一些行业是很危险的，经常会死人，那个年代也没什么安全意识，反正打仗也是要死人的，只要有一碗饭吃，再危险的职业也会有争抢。于是周边地区去安徽做工的人只多不少，经常都会有人客死他乡。
<br/>　　这些做工的都是苦命人，死亡之后，家人也雇不起马车，长途跋涉地将尸体运回去，而且往南很多地方没有官道，就算有马车也过不去。但是华夏的传统是叶落归根，人死之后，肯定是要葬回老家的。
<br/>　　赶尸，就成了最佳的运送尸体回乡的办法。赶尸匠是用赶的，成本比较低，而且一次至少也会赶几具尸体，所有每家收的钱其实不多，在这些死者家属的接受范围之内。
<br/>　　因为常年都有人克死他乡，有几个赶尸匠就聚在一起，在一个山区荒村建了个义庄，所谓义庄，就是停放死人的地方，皖南和赣北的人大多知道，如果附近一带有死尸需要赶，赶尸匠会将尸体接到义庄暂放，将同个方向的凑三五具尸体，然后一起赶回去……
<br/>　　陈毛子说，之前没上山的时候，他在茶馆里听人聊起过义庄的事，说是最近因为打仗的缘故，道路不通，就算是有钱人家，也没法通过大路送尸体回家，只能找赶尸匠，因此最近义庄的生意非常好，赶尸匠都不够用了。
<br/>　　叶少阳起初只是随便了解一下，听陈毛子说到赶尸匠不够用的时候，心中一动，想到了什么。
<br/>　　老万和陈毛子好生嘱咐了叶少阳二人一番，然后四人就地告别，老万和陈毛子下山去附近的县城，去卖货物。
