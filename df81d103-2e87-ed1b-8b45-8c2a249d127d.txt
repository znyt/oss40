　　离开掌门宫殿后，慕清霄就原路返回住处。
<br/>　　吩咐一声后，便迅速前往后山，盘坐于山崖之巅，雾气缭绕间，一袭白衣，发丝随风飘动，仙风道骨。
<br/>　　空明剑与太虚剑，乃是剑仙紫胤的成名剑法，想要修炼此剑法，自身修为必须迈入元婴期，且需要领悟剑意。
<br/>　　否则，难以入门。
<br/>　　将空明剑剑诀仔细阅览了一遍后，慕清霄盘坐于山崖，陡然睁开星目，中指与食指并于胸前：“凝！”
<br/>　　“铮铮...!”
<br/>　　话语刚落，他的周围就开始出现一柄柄由灵气虚幻而成的剑影。
<br/>　　片刻后，抬头望着空中，数十柄微鸣的剑影，每一柄虚幻剑影上都附着恐怖的剑意，锐利至极。
<br/>　　慕清霄微微一笑，道：“空明剑，这只能算是入门吧？不知修炼至圆满，此剑法该有多强。”
<br/>　　天墉城东面的剑塔之上，一名蓝色道袍中年，负手而立，看向西侧方向，深邃的眼中满是震惊。
<br/>　　片刻后，紫胤眼中的震惊化作唏嘘，轻声道：“空明剑，短短半刻钟就入门了吗？果真是妖孽之才。”
<br/>　　...
<br/>　　...
<br/>　　光阴似箭日月如梭，眨眼间，五年岁月一晃而过。
<br/>　　天墉城西侧，山崖之巅，青年盘坐于上，如同老僧入定，浑身没有一丝的气息流露而出。
<br/>　　陡然，青年睁开双眼，星眸中光泽流转，‘呼’嘴中吐出一口浊气，道：“也是时候，在去一趟妙法堂了。”
<br/>　　五年的时间里，天墉城并未发生过大事，琐事倒是不少。
<br/>　　期间，慕清霄每个月按时帮天墉城炼制好规定的丹药，其余的时间都在修炼空明剑与太虚剑。
<br/>　　因此，他对剑道上的领悟更深了一些，修为也从练虚一重提升至练虚四重初期。
<br/>　　此时，他已经将空明剑与太虚剑修炼至圆满，一剑斩下，威力击杀练虚六重强者不在话下。
<br/>　　拍了拍长袍上积攒的灰尘，慕清霄站起身来，化作一道剑光向天墉城中央掠去。
<br/>　　来到妙法堂，慕清霄收敛气息，并没有引起任何人的注意，径直来到第三层，走向中央的凝虚长老。
<br/>　　当慕清霄来到身前，凝虚真人才睁开双眼，上下打量了他一番，道：“几年不见，凝丹长老的修为，估计不比老夫差了吧？”
<br/>　　闻言，慕清霄微微一笑，道：“紫胤真人剑诀高深，在下颇得要领，修为提升一些实属正常。”
<br/>　　话语刚落，凝虚真人眼角抽了抽，恨不得大嘴巴子抽他。
<br/>　　正常个屁，老夫修炼了上百年，都只有练虚八重的修为，你特么短短几年，直接从化神窜山练虚。
<br/>　　现在还在老夫面前装逼，能不能要一点脸啊！
<br/>　　“凝丹长老来此，是要前往第四层吧？”凝虚真人似乎猜到了慕清霄的来意，直接开口问道。
<br/>　　慕清霄微微点头，道：“不错，空明剑与太虚剑在下已经在参悟，所以想去第四层瞧一瞧。”
<br/>　　“哎，老夫早就知道，这一天不会远，没想到来的如此之快。”
<br/>　　凝虚真人挥了挥手，有些不耐烦的说道：“快点上去吧，看完赶紧走，别再来气老夫了。”
<br/>　　慕清霄报以微笑也不在意，转身向第四层走去。
<br/>　　走上阶梯几秒钟后，正当慕清霄准备进入第四层的时候，脚步陡然一顿，停在了最后一层阶梯上。
<br/>　　因为，在他的面前，出现一道透明的壁障，在壁障的上方，刻着一道透明的凹槽，凹槽呈现剑印形状。
<br/>　　看着其上的剑印凹槽，慕清霄低喃道：“难怪涵素真人敢将最珍贵的秘籍放在这里，原来设下了空间禁制，剑印便是开启禁制的钥匙吗？”
<br/>　　虽然只是初步猜想，但他却能从壁障上，感受到一股毁灭性的气息。
<br/>　　若是有修士想要强行破开空间禁制，那么下场恐怕会无比的凄惨。
<br/>　　所以，唯有将空明剑与太虚剑参悟的弟子，才有资格进入其中，与凝虚长老所说完全一致。
<br/>　　想到此处，慕清霄调动灵气，在空中凝聚出一道空明剑印，空明剑印与壁障上的凹槽相互重合......
<br/>　　果然，在剑印莫入剑印凹槽后，壁障明显松弛了许多。
<br/>　　于是，慕清霄再一次调动灵气，凝聚出一道太虚剑印，紧接着太虚剑印也莫入剑印凹槽中。
<br/>　　在太虚剑印莫入剑印凹槽后，整个壁障都波动了一下，旋即如同水波一般荡漾起来。
<br/>　　壁障荡漾的同时，仿佛被一双无形的大手给扯开，露出一个能容纳一人通过的门扉，其中的景象依旧只是倒影。
<br/>　　看到门扉的同时，慕清霄脑海中不禁响起，在斗破位面，进入迦南学院时的情况。
<br/>　　场景似乎如出一辙，但此处的空间禁制更加的恐怖，绝对不是斗破位面能比拟的。
<br/>　　望着门扉，慕清霄也不再犹豫，一脚踏进其中，随着前脚莫入其中，完全就像消失了一般。
<br/>　　随着壁障一阵阵涟漪荡起，慕清霄整个身体都莫入其中，最后消失的无影无踪。
<br/>　　在他的身体完全进入其中后，门扉也开始缩小，渐渐融合，最后消失，再次显现出一个剑印凹槽。
<br/>　　...
<br/>　　...
<br/>　　妙法堂第四层，与下面几层没有任何区别，至少在慕清霄看来，没有任何的不同。
<br/>　　其中的空间不小，但四周皆是空荡荡一片，唯有中央处摆放着一个书架。
<br/>　　唯一的书架上，摆放着不少的秘籍，由下至上摆放的非常整齐，这倒是令得慕清霄有些感到意外。
<br/>　　真是没想到，第四层的秘籍，居然比第三层要多上不少，要知道这些可都是掠夺积分，自然是多多益善。
<br/>　　想到此处，慕清霄面露微笑的走上前去，拿起最底层的一本秘籍，脑海中再次响起系统的提示声。
<br/>　　“叮，《五行封印》收录中，价值10000掠夺积分。”
<br/>　　微微一愣，慕清霄翻开秘籍，其中记载的是一种封印秘法，而且是颇为高级的封印秘法，由上古时期流传至今。
<br/>　　金木水火土，五行为印，甚至比天火尊者的‘天都火印’都要高深许多。
<br/>　　望着手中的《五行封印》，慕清霄叹息一声放回原处，再次抽出第二本秘籍。
<br/>　　“叮，《阴阳八卦印》收录中，价值10000掠夺积分。”
