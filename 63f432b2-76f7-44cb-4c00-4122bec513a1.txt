　　“噼里啪啦！”
<br/>　　一阵脆响，元始口冒赤火，鼻若生雷，身体微响轻颤。不一会儿，这神火就消失不见了。
<br/>　　“不错，不错！这火味道还行！虽然比不上太阳真火、太阴真火之类的，但也只稍差一筹。”元始张嘴，赞叹道。
<br/>　　“轰！”
<br/>　　朱雀听了这话，浑身气势更加狂暴了，风云变色，岩浆怒啸。你这话意思是我的火焰不如太阳真火、太阴真火了。士可忍，孰不可忍？今天就让你尝尝我的厉害。
<br/>　　“啾！…”
<br/>　　“轰隆隆！”
<br/>　　“朱雀六击！”
<br/>　　朱雀振翅，双翅舒展，遥指九天，浑身赤金色火焰沸腾，犹如火中神灵，沐浴神火。双目怒睁，鸟嘴微勾，全身气势凌天，睥睨苍生。
<br/>　　朱雀双翅如剑，猛地下斩，剑气纵横，烈焰翻滚，如一尊绝世剑仙斩向元始，又如火神降临，威压天地！
<br/>　　“哼！”元始冷哼一声，双目微怒。竟然比我还会输耍帅，你是神兽又如何，今天本座偏偏要揍你一顿，谁让你在我面前威风的？
<br/>　　元始高举大手，对着向他攻来的朱雀遥遥一指，轰然击下。如天帝凌天一指，霸道绝伦，又如天柱镇世，神威盖世。
<br/>　　天地万物无声，神火剑气瞬间就被冻结，轰然破碎，岩浆结冰，空间破碎。
<br/>　　寂灭指！
<br/>　　一指之下，万物尽归寂灭！
<br/>　　“唧唧唧唧！”
<br/>　　朱雀倒飞，悲啼鸣叫，声声刺耳。但更多的是愤怒，不甘，隐隐有一丝忌惮。
<br/>　　元始一步踏出，追到朱雀身边，一脚踏在它身上，一拳又一拳的轰击在朱雀真身上，拳打脚踢，拳脚相加。
<br/>　　朱雀怒吼，怒火升腾，神火变得更加炙热恐怖，双爪如勾，撕裂虚空，双翅若剑，斩尽星辰，天神临凡，神威如狱。各种攻击，全部对着元始攻来。
<br/>　　就算是一尊准圣强者，面对这些攻击，亦会身受重创，心有不及。
<br/>　　但，元始却怡然不惧，面对那令人色变的神火攻击，张开大口，将朱雀神火尽皆吞噬，至于其余攻击，也尽在元始的狂暴攻击之下消散一空。
<br/>　　元始不停，继续拳脚相加，尽皆加在朱雀身上。
<br/>　　如此长时间攻击，朱雀不敌，想要逃跑，但不管它去到那里，元始总能第一时间出现在它身上，继续打它。
<br/>　　这样的状态持续很久，直至——
<br/>　　“停——”朱雀再次躲开元始攻击，喊道，“停！停！停！不要再打了！”
<br/>　　“嗯？”元始听到这话，停了下来，看向朱雀，那朱雀看似凄惨无比，实则并没有受重创，只因元始每次都控制好了自己的劲道，虽然打是打到了朱雀身上，但那些只是些皮外伤，并没有重创它。
<br/>　　元始轻声呢喃道：“还是拳拳到肉的打架方式爽啊，好久没有如此打过了，还真是让人怀念！”
<br/>　　接着，又对着神兽朱雀冷冷道：“不打了？”
<br/>　　“不打了！不打了！”朱雀连忙摇头，心说，我又不是傻子，我才不打呢？既然不是你的对手，谁还要和你这样的疯子打架，你虽然没有重创我，但，你每次都跑到人家身体上打人家，身为神兽的我感觉疼痛难忍，这样的攻击，谁能受得了啊。
<br/>　　淡淡地看了一眼神兽朱雀，好似知道他的想法一般，元始淡淡道：“既然不打了，那么我们就来好好谈谈。”
<br/>　　“谈谈？谈……谈什么？”朱雀有些畏惧道。
<br/>　　“本座欲与你做一场交易。”
<br/>　　“是何交易？”
<br/>　　“予本座一滴神兽精血！”元始面无表情，淡淡道。那神态，那语气，好似给我你的精血，你不会吃亏一样。
<br/>　　“不可能！”朱雀神兽想都没想，一口否决道。
<br/>　　取神兽精血，就是等于断其未来。神兽精血，每一滴都需要上千万年时间来凝聚，其中蕴含着神兽本源之力，本源受损或者不足，就不能够突破，未来若是没有巨大造化，更不可能有进步，要是一个处理不好，修为甚至会倒退下降。
<br/>　　“嗯？”元始眉头微皱，神情微冷，淡淡的看着朱雀神兽。
<br/>　　“呃？！”朱雀神兽被看了心里直发毛，但就算这样，他仍然坚决不松口，毕竟这关系到自己的未来成就。
<br/>　　直至良久，元始似乎是看到了朱雀内心的想法，微微道：
<br/>　　“本座又不白拿你的精血，这是交易，既然是交易，你自然有得有失。”
<br/>　　“你用什么来换？”尽管不想交易，但摄于此人的强大实力，自己还是决定先听一听。
<br/>　　“与我一点精血，本座送你一个让你气运大涨、实力大涨、与天同寿，与地同存，能够快速突破的机缘，如何？”
<br/>　　“什么？！”朱雀一惊，“竟有如此机缘，那——，你怎么不用。”朱雀显然是有些不信。
<br/>　　“以本座之实力，有必要骗你吗？本座完全可以强行提取汝之精血，根本没必要与你废话。只是这机缘只有像汝这样的神兽可以拥有。”
<br/>　　“也是。”朱雀想了想，点头道。
<br/>　　“交换与否？”元始道。
<br/>　　“交换！”神兽朱雀想了想，还是决定交换，自己已经有很长一段时间没有突破了，现如今有这等机缘，自己一定要珍惜，将之抓紧。何况，眼前这人实力强于自己，自己也没有力量反抗，就算不交换，此人也一定会强取自己的本源精血的。
<br/>　　“好，算你识相。既然如此，那本座就慷慨一次，先将此机缘送与你。”元始点头，他早已料想到如今这种情况，神兽镇压天地四极，想要突破极难，如今有此等机缘，他断然不会放过。
<br/>　　元始心念一动，一道信息便传到朱雀脑海之中。
<br/>　　朱雀心神沉入脑海，瞬间就知道了这道信息，内容大概是太古星辰之中，有二十八宿，其中，南方七宿（井、贵、柳、星、张、翼、轸），其形象鸟，位于南方，属火，色赤，为朱雀……
<br/>　　此乃神兽朱雀星域，其中一座朱雀星宫，神兽朱雀若入其中，则为星主，统御朱雀星域，镇天之四极，与地之四象相应，功德无量，气运加身……
<br/>　　星辰不灭，朱雀不死，必能与天地同存。
