　　星月奴肉身在北斗观？
<br/>　　这一点，连他们也都不知道。
<br/>　　“不好！道风这是调虎离山，把清长风拴着这，他的元神去了北斗观！”黄衫道士第一个回过神来，几个大佬登时心头一颤，虽说他们都不喜欢星月奴，但按照董事会的意思，法术公会的买卖，是他们这些分公司都要支持的，一旦黄了，大伙都不好看，再说清长风已经求救了，当然不能装死，赶紧的热闹也不看了，纵身朝北斗观飞去。
<br/>　　“好啊道风，你这一手，可真是毒辣！”清长风气得浑身哆嗦，他倒是没走，既然那几个大佬去了，他再去也没什么意思。
<br/>　　现在最关键的，就是看他们能不能赶得及。到这一步，他自己再担心也没用了。
<br/>　　道风已斩却两尸，肉身和魂魄，各自灌入一缕精气，模拟神魂意识，各自都可以独当一面，在来的路上，他的元神化身就已经从本尊分裂出去，去了北斗观，一路杀上山去……
<br/>　　这本来就是一个调虎离山的计划，清长风以为自己在这里设下埋伏，伏击他们几个，实际早就被道风看穿，将计就计，目标是斩杀星月奴的肉身。
<br/>　　“道风，你怎么知道，星月奴肉身在我北斗观？”清长风厉声喝问，他之所以会中计，除了道风谋划得太过周全，还有一个重要因素是：除了极少数几个人，就连法术公会的核心领导层，都没几个人知道她肉身在自己那里，就这个这个，他才一点也没往这方面想。
<br/>　　道风没有回答，低头看着地上的上古邪神——他的元神也快融化没了，只剩下一道淡淡的虚影，还在望着自己，在临近毁灭的时候，他的眼神中居然带着笑意。
<br/>　　“希望……为你争取到了时间……”上古邪神的残念，在神识中对道风说道。
<br/>　　“为什么不走？”
<br/>　　“你有十二门徒，另外十一个，早就不知道换了几批……只剩下我一个，我也早就该死了，没有什么，能死在这里，我也值了……”
<br/>　　他的元神开始燃烧，只剩下最后一点灵智。
<br/>　　“道风，我当初为鬼母所杀，是你为我聚灵重生……从那一刻，你便是我主人，为你而死，我并无遗憾，只是……等不到你斩三尸证道了……”
<br/>　　没有什么大义凌然视死如归的豪言，他也不觉得自己这么做有多伟大，无非就是当奴仆的，为主人牺牲而已，天经地义。
<br/>　　灯枯油尽的他，总算燃烧尽了最后一点灵智，元神爆裂，一团团的精魄粒子，在空中散开，朝着轩辕之门的方向飞去。
<br/>　　世上再没有上古邪神了，总有一天，他精魄聚生，却要经受六道轮回之苦，不过，那已经是几百年之后的事了。
<br/>　　道风在心底叹了一声。
<br/>　　“阿弥陀佛……”
<br/>　　山山禅师双手合十，朗诵往生咒，为上古邪神超度，消除他生前的业障，这样将来他再轮回的时候，或能减轻一些孽缘。
<br/>　　因果机缘，是最神妙的事，就算证道的大能，也无法把握，但超度，会带来一些妙不可言的改变。
<br/>　　道风微微颔首，抬头望着清长风，神色平和，没任何表情。
<br/>　　“是的，道风，你这一手计谋用得极好，但你为自己想过没有？你的魂魄，却是走不脱了。”
<br/>　　“想过。”
<br/>　　清长风还想听下文，却冲他招了招手，这意思……是让自己动手？
<br/>　　“你不怕死？”
<br/>　　“已致绝境，怕有何用。”
<br/>　　“好，我就看你还有什么手段！”
<br/>　　对道风所说，清长风是不信的：他身外化身已经毁了，元神又在自己山门那边，眼前这是他的魂魄，如果再挂了，就只剩下元神了。
<br/>　　普通生灵到这彻底挂了，因为元神是虚无之物，只能依靠肉身和魂魄而存在，一旦这两样都没了，元神要不了多久也就会消散了。
<br/>　　就算道风斩了两尸，肉身和魂魄能各自独立，靠着一口精气形成化身，元神没有他们也能独立存在，但终归是元气大伤，想要再修炼出魂魄和血肉，至少得上百年工夫，道风当然等不起。
<br/>　　因此，清长风猜测他必定还有伏笔，只是压根想不到他怎么才能翻盘，当下大吼一声，将要动手之际，突然想到道风之前所为：当时自己不也认定他绝对要挂了，想不到任何翻身的机会，结果人家早就金蝉脱壳了……难不成他真的还有什么后手，不然怎么看上去这么气定神闲的？
<br/>　　不管了，试一试！
<br/>　　清长风一咬牙，将大洞真经的法阵发挥到了极致，汉字融化组合，形成了六十四卦，从四面八方将道风裹住，看着像个巨大的魔方，不断翻转着，道风被困在中间，随着“模仿”上下翻飞，竟然没有反抗。
<br/>　　为何会这样？
<br/>　　清长风心中怀疑，但是不敢怠慢，继续操控法阵，消耗着道风的神魂之力。
<br/>　　之前与山山禅师斗法，道风神魂也受了伤，再加上化身被灭，元神又在另一个地方，只剩下魂身，实力大大削弱，根本无法对抗法阵的威力，一会儿的工夫，魂力就耗尽了，魂魄逐渐变得透明起来。
<br/>　　真的……没有后手了？
<br/>　　清长风怔怔地看着叶少阳，看着他魂魄立刻就要被撕碎，知道他再没什么后手了，喃喃道：“道风，你为何要这么做？”
<br/>　　“打神鞭和番天印你可拿去，将来，自会有人来找你取。”
<br/>　　清长风笑道：“这等神器，贫道自然不会放过。”
<br/>　　道风魂魄燃烧，三魂七魄都被撕碎，在众目睽睽之下化作了精魄，朝着轩辕之门方向飞走了。
<br/>　　山山禅师又念起了往生咒，为道风超度。
<br/>　　在场的人，包括清长风和他那八个弟子，怔怔地望着道风精魄飞走的方向，难以相信道风的魂魄就这么死了。
<br/>　　虽然，他之前已经做了太多不可能做到的事。
<br/>　　“走，回山！”清长风回过神来，箭步离去，突然又折回身子，冲山山禅师拱手道谢，说了几句客气话，这才飞身回山。
