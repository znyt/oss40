　　辽阔的太平洋之中，一座荒芜的小岛上，
<br/>　　“嘿嘿！华夏战神叶天歌，若非亲眼所见，我还真不相信，你居然还活着！”
<br/>　　说话的，是一名一头金色长发的白人中年男子。
<br/>　　而在白人中年男子旁边，还有一名犹如铁塔般的黑人男子，以及一名风情万种的混血女子。
<br/>　　“海神波塞冬！力量之神沙奎尔！还有梦幻之神苏珊！”
<br/>　　叶天歌扫了一眼三人，便将三人的名字一一喊了出来。
<br/>　　“听说你们十二主神，如今只剩下你们三个了，竟然还不消停，难道真的想全部覆灭不成？”
<br/>　　听到这话，三人顿时纷纷脸色全都一变，
<br/>　　“还不都是你儿子干的好事！今天就拿你的命，来祭奠我死去的兄弟们！”
<br/>　　三人之中，明显以力量之神沙奎尔，脾气最为火爆，大喝一声后，便直接朝叶天歌扑了过来！
<br/>　　“叶天歌！你可敢接我一拳？”
<br/>　　轰！
<br/>　　这沙奎尔，不愧是力量之神，一拳砸出，空中竟然响起一串串气爆声，那如同砂锅大的拳头，瞬间便已经攻到叶天歌的跟前。
<br/>　　“哼！我有何不敢？”
<br/>　　面对沙奎尔这一拳打爆一头大象的一拳，叶天歌不闪也不避，直接扬起拳头便迎了上去。
<br/>　　沙奎尔顿时一喜，
<br/>　　他原本只是随口一激，没想到对方竟然真的敢和他对拼力量！
<br/>　　‘在神境之中，单单比拼力量的话，老子可是无敌的！’
<br/>　　沙奎尔自信满满。
<br/>　　轰隆隆！
<br/>　　两人的拳头，在空中狠狠相撞在一处！
<br/>　　似乎连周围的空间都扭曲起来，两人脚下更是出现了一个巨大的深坑！
<br/>　　蹬蹬蹬！
<br/>　　两个大小相差悬殊的拳头，一撞之后，便旋即分开。
<br/>　　然而，
<br/>　　结果却是，
<br/>　　身材无比壮硕的沙奎尔连连倒退，而叶天歌却纹丝未动。
<br/>　　“什么！”
<br/>　　不但沙奎尔满脸震惊之色，身后的海神波塞冬和梦幻之神苏珊，也全都脸色大变。
<br/>　　要知道，沙奎尔是神境级别的强者，而且又以力量强悍著称，单单比拼力量，便是他们昔日的老大，雷神宙斯，也不是他的对手！
<br/>　　作为老对手，他们对叶天歌的实力，还是比较清楚的，
<br/>　　顶多也就是神境巅峰，比雷神宙斯还要弱上一筹。
<br/>　　可是现在，他居然一拳便将沙奎尔击退！
<br/>　　此时此刻，他们不得不重新估量叶天歌的实力了。
<br/>　　他们又哪里知道，被叶尘用神龙之血和七叶回仙草复活，叶天歌如今已经具有人仙级的实力，而且受到神龙之血的影响，力量也比之前有了极大的进步！
<br/>　　“哈哈哈！”
<br/>　　叶天歌哈哈一笑，
<br/>　　他一拳击退了沙奎尔，同样微微有些意外，虽然他知道自己如今的实力，已经今非昔比，可是却也没有想到，竟然进步如此之大！
<br/>　　“沙奎尔！看来这半年多不见，你也没有太大长进啊！”
<br/>　　“你也来接我一招试试！”
<br/>　　说完这话，
<br/>　　叶天歌忽然纵身跃起，以手臂做刀，狠狠劈落下来！
<br/>　　“流云刀！”
<br/>　　轰！
<br/>　　一道长达近百米的青色刀芒，在空中划过一道完美的弧线，瞬间便斩落斩落到沙奎尔的头顶。
<br/>　　“哼！”
<br/>　　沙奎尔不慌不忙，当即举起双臂，直接挡了上去。
<br/>　　铿！
<br/>　　青色刀芒斩在沙奎尔的手臂之上，竟然发出一道金属撞击的清脆声音。
<br/>　　原来，沙奎尔的手臂之上，竟然带着一个金灿灿的护臂。
<br/>　　那护臂，和当初叶无伤佩戴的一模一样，居然是由振金合金打造。
<br/>　　沙奎尔虽然挡住了这一记刀芒，整个人也不由得又倒退了数步。
<br/>　　不过很显然，他并没有受伤。
<br/>　　叶天歌看到这一幕，顿时眉头一皱，不由得低声感叹了一句，
<br/>　　“看来米国超能局的科技，又进步了不少！”
<br/>　　在过去的二十年间，叶天歌没少和米国超能局的高手过招。
<br/>　　他深刻清楚，米国的强者，完全是靠先进的科技堆出来的，基因药水强大改造身体，再加上各种先进的武器和装备，足以让普通人在很短的时间内，和华夏修炼了几十年的武道高手相媲美。
<br/>　　就拿眼前的沙奎尔来说，方才他若是没有这振金打造的护臂，此刻那条胳膊，早就被他一刀砍下来了。
<br/>　　“沙奎尔一个人对付不了，大家一起出手！”
<br/>　　海神波塞冬，看到沙奎尔不是叶天歌的对手，当即大喝一声，跃至高空，双手朝远处海面上一抓！
<br/>　　哗啦啦！
<br/>　　海面顿时爆炸开来，两道水柱从海中窜出，犹如两头水龙，朝叶天歌冲了过去。
<br/>　　刷！刷！
<br/>　　叶天歌当即挥出两刀，将那两条水龙斩破。
<br/>　　然而下一刻，叶天歌又察觉到一股强大的精神攻击！
<br/>　　却是梦幻之神苏珊也跟着出手了。
<br/>　　三大主神同时出手，而且配合默契，叶天歌顿时便落入了下风。
<br/>　　尤其是那波塞冬，原本在十二主神之中，就排名第二，实力仅在当初的雷神宙斯之下，而且此时又是在海上，相当于是他的主场，能量几乎可以取之不尽，用之不竭。
<br/>　　……
<br/>　　一晃眼，四大强者已经交手上千招，叶天歌以一敌三，此时已经颓势尽显。
<br/>　　数百海里之外，华夏海军的一艘军舰之上，海军上将林震南，死死的盯着眼前的屏幕，满脸的凝重之色，
<br/>　　“唉！我早就说过，叶将军不该以身犯险，这下麻烦大了！”
<br/>　　旁边一名中年副官，满脸焦急之色，
<br/>　　“首长，请下命令吧！我们若是再不去支援叶将军，他恐怕不是那三人的对手啊！”
<br/>　　林震南又重重叹了一口气道：
<br/>　　“此事我又未尝不知？可是那里，已经属于米国的势力范围，我们若是贸然出兵，便等同于主动挑衅米国，到时候可就给了米国出兵的正当借口！”
<br/>　　”米国和华夏都是当世大国，一旦战争打响，后果将会极其可怕！“
<br/>　　“更何况，我们如今的海军实力，和米国还有不小的差距，上头给出的命令，也是尽量克制……”
<br/>　　众人听到这话，顿时个个脸色无比难看，
<br/>　　“可是，首长，难道我们就这么干看着？要不派一只神龙卫精英小队，前去支援叶将军……”
<br/>　　还没等那人说完，林震南直接摆了摆道：
<br/>　　“没用的！达到他们四人的境界，寻常高手根本帮不上什么忙，至少也得是神境以上的实力，才有插手的资格！”
<br/>(本章完)
