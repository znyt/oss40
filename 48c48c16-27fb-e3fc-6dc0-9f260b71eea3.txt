　　“不是每个人都能打开那扇禁锢之门，进入那个领域。只要你儿子进入不了那个领域，就注定不可能超越蛰龙，即便是打开了那扇门，也未必能超越蛰龙。”
<br/>　　方牧南看向战术训练场的方向，看着在战术训练场上不断翻腾跳跃，做着各种战术动作通过障碍物的年纪大概也就十岁左右的学员，说道：“我们就算缔造过传奇，可是我们终究会老去，甚至会死在战场上，这些小家伙还有他们身后更加年轻的小家伙，他们才是国家未来的希望，我们应该做的，是留下些什么，尽可能的帮助他们成长。”
<br/>　　“我的身体我自己清楚，说不准哪天睡着了，就再也醒不过来了。在我剩下的有限时间里，想要再培养出一个学生是做不到了，你家小子的天赋和底子都不错，能学到多少，就看他自己了。老了——”
<br/>　　方牧南长叹一声，拄着拐杖转身离开。
<br/>　　看着方牧南在夕阳下拖得长长的影子，罗昊能从他孱弱的背影和那声长叹中，感觉到他心中的不甘，如果可以，他恨不得像诸葛亮那样摆下七星灯，向天再借一纪（12年）的寿命。
<br/>　　……
<br/>　　晚上，罗昊、云熙、齐凡三个人就离开了红旗学校，虽然云熙真的很想留下陪云悠，但红旗学校有红旗学校的规矩，不能坏！
<br/>　　有方牧南和沈琛的关照，罗昊也相信，类似的事情不会再发生，而且诸磊那小子也没有可能再接近云悠到身边。
<br/>　　考虑到方牧南身体的关系，罗昊一个星期中有三天时间去方牧南的小院子里学习。
<br/>　　罗战去方牧南那个小院子的时候，看到云悠正拿着从蓝若依那里弄来的小鱼干正在喂方牧南样的那只猫，这不是什么有品种的名猫，只是一只流浪的小野猫被方牧南遇见，就捡了回来。
<br/>　　“小悠儿，方爷爷呢？”罗战对云悠问道。
<br/>　　云悠扭头看着那间被方牧南锁了有很长时间的工作室，说道：“在工作室呢。”
<br/>　　罗战跑过去逗了一下那只猫，惹得那只猫冲着罗战大叫了一声，在云悠不满的抱怨声，罗战快速跑开，朝着工作室的方向跑去。
<br/>　　罗战扒着工作室的墙壁，探头朝着工作室里看了一眼，看到方牧南正在工作台前正在忙碌着什么，迈步走进工作室，对着方牧南打招呼道：“方爷爷！”
<br/>　　“你这捣蛋鬼又去惹老猫了是吧？”方牧南手指对着罗战虚点了一下，笑着问道。
<br/>　　“您怎么知道是我？”罗战反问道。
<br/>　　“能进到这个院子里来的人只有你跟小悠儿，但是小悠儿可不像你这么调皮。”方牧南回答道。
<br/>　　罗战嘿嘿一笑，也没有反驳。走到工作台前，想要弄明白方牧南到底在干什么？凑近一看，发现方牧南正在工作台前制作着一颗子弹，而且已经制作完成。
<br/>　　“您是在制作狙击步枪的子弹吗？”罗战对方牧南问道。
<br/>　　方牧南有些好奇地看着罗战问道：“你应该还没有学习到武器制作，你怎么知道我是在做狙击步枪子弹？”
<br/>　　“我在龙穴的时候，看狐狸叔叔做过。”罗战回答道：“狐狸叔叔说，每个狙击手都有自己各自的射击习惯，常规生产流水线上制造出来的子弹只能狙杀一定距离内的目标，如果想要狙杀超远距离的目标，或者是一些特定的狙杀任务，自己做得子弹更顺手。”
<br/>　　“没想到你这小子还知道的挺多。”
<br/>　　方牧南拿起刚制作好的子弹，放在眼前仔细地打量了一番，随后满意的点了点头，叹了口气，说道：“老了，就连子弹都做不动了。”
<br/>　　“方爷爷，这颗子弹是给我做的吗？”罗战盯着方牧南手中的子弹，问道。
<br/>　　方牧南先是一愣，他没想到罗战居然会问自己这个问题，但想到他是被赵天宇和罗昊训练出来的，猜想到这小子可能知道点什么，便点头回答道：“每个我亲手教过的学生，我都会给他们留下一颗子弹。”
<br/>　　方牧南拿起工作台上那个垫着带有子弹轮廓凹槽的黑色阻燃材料的盒子，指着盒子里那颗子弹说道：“这颗子弹是蛰龙的，剩下两颗被拿走的，一颗是你师公的，另外一颗是飞扬，也就是你妈妈老师的。如果他们两个人的子弹还在，我还要重新制作一个盒子，但现在不需要了。”
<br/>　　方牧南把那颗刚做好的子弹，放入其中一个凹槽中，对罗战说道：“你这颗子弹会一直留在这个盒子里，等到你拿到‘龙刺’徽章，或者是到了合适的时机会有人把这颗子弹还给你。我离开了，我会把这个盒子交给天宇保存，等到天宇老了，会把这个盒子交给你爸爸，如果他们认为你达不到拿走这颗子弹的标准，你这颗子弹就会一直留在这里，如果有一天你背叛了国家，这颗子弹就会出现在战场上，会有人把它送进你的身体。”
<br/>　　虽然罗战从罗昊和赵天宇那里听闻过老师给学生制作子弹的说法，也看到过当年自己师公给自己爸爸制作的那颗子弹，但是现在亲耳听到方牧南谈起这件事情，罗战突然觉得自己后背凉飕飕的，就像是有人藏在暗处拿着一把枪瞄准着自己一样，仿佛随时都有可能都会有子弹打中自己，击杀自己。
<br/>　　方牧南希望自己放进这个盒子里的这颗子弹永远没有被装进抢滩的那一天，因为等到它被装进枪膛的时候，那么扣动扳机的人很有可能就是罗昊！
<br/>　　孩子未来的发展，跟他小时候成长环境有很大的关系，方牧南也相信，罗战这个身上流淌着一半云家血液的小子，有云熙、罗昊、赵天宇这些人的督促和教育，不会走到无法挽回的那一步！
<br/>　　云悠藏在工作室外，听到了工作室中自己弟弟和方牧南之间的谈话，两只小手握的紧紧的，还显得稚嫩的眼睛中闪烁着坚定之色，她一定不会让自己弟弟走到国家对立面的！
