　　叶少阳对照着她的脸，在记忆中搜索起来，最后确定，这大姐肯定是认错人了！自己父亲就没有亲姐妹，母亲那边……更不用说了，她都死了十几年了，就算有娘家人，就算自己去认亲，人家能不见得能认识自己。
<br/>　　绝无可能。
<br/>　　叶少阳刚要辩解，“表姐”问道：“你感觉怎么样？”
<br/>　　“暂时死不了吧，那个……”
<br/>　　“没事就行。那个，张妈，您先回吧，我来给阿牛弄点吃的。”
<br/>　　阿牛……叶少阳听到这个差点喷出来。
<br/>　　“中中，先喂他喝点粥，他昏了几天，不能吃硬东西哦。有啥事去我家叫我。”老女人喋喋不休的吩咐了一番，让年轻女送出门去。
<br/>　　过了一会，年轻女子进屋，冲叶少阳笑道：“你醒了就好，饿不饿，我去煮点粥给你喝。”
<br/>　　自己确实感到饿坏了，点了点头，随即又说道：“大姐，我真不是你表弟，不是什么阿牛，你认错人了。”
<br/>　　“没错的，你就是我表弟。行了你刚醒，先躺着，一会喝粥。”
<br/>　　年轻女子说完就去了外屋，留下一脸懵逼的叶少阳。
<br/>　　叶少阳听见外屋有生火做饭的声音，好像用的是柴火灶？
<br/>　　叶少阳咧了咧嘴，做出一个非常无奈的表情，然后朝自己身上看去，肋下和左边肩膀都用棉布包扎上了，似乎在提示他，之前发生的一切，并不是梦。但是……这究竟是什么地方啊！
<br/>　　想不透的事情，叶少阳还是不去想了，靠在床头，之前的经历浮上心头，叶少阳猛然想起什么，忍着疼痛，在床上翻找起来，结果没找到背包和腰带，顿时吓的大汗淋漓，正好这时候那年轻女子进来了，手里端着一碗热腾腾的粥。
<br/>　　“大姐，我的东西呢？”
<br/>　　“我都给你收起来了，在床底下呢。”
<br/>　　在叶少阳强烈要求下，年轻女子把叶少阳的背包和腰带都找出来，叶少阳急忙翻找起来，东西都还在。
<br/>　　“这把剑也是你的吧？”
<br/>　　叶少阳从她手中接过七星龙泉剑，内心是颤抖的，这把剑，还有阴阳镜、天风雷火旗等等一些法器，都是自己的命根子，幸亏没丢……当下对年轻女子投去感激的一瞥，道：“多谢大姐了。”
<br/>　　“莫客套，你是我表弟嘛。”
<br/>　　叶少阳一头黑线。
<br/>　　“那个，你真的认错人了，我不是你表弟，我叫叶少……”
<br/>　　年轻女子捂嘴笑道：“大兄弟，我不逗你了，我知道你不是我表弟，你是逃难来的吧？”
<br/>　　“嗯？”
<br/>　　“三天前的晚上，我在院门口看到你，你躺在我院门外，一身都是血，我怕你死了，就把你弄家里来了，我跟人说你是我表弟，家破了来投奔我的，在山里遇到野兽被抓伤了。你身上的伤是被野兽抓的吧？”
<br/>　　叶少阳想了一下肋下被芮冷玉用手指插出的伤口，确实被像是什么东西抓出来的，冲年轻女子点了点头，“真是多谢大姐了。”
<br/>　　年轻女子道：“我叫翠云，我比你大，你叫我一声翠云姐就好，我说你是我表弟，是为了避嫌，我家当家的肺痨病死了几年了，我现在一个人寡居。”
<br/>　　叶少阳缓缓点头，还想再问，翠云把粥端上来，要喂叶少阳吃。叶少阳一直胳膊能弄，自己端着喝了，是粗米煮的红薯粥，看上去黑乎乎的，叶少阳趁热喝了，胃里有了点热气，感觉舒服多了。
<br/>　　翠云说道：“大兄弟你身体真棒，我以为你要躺个十天半个月呢，没想到你才几天就醒了能吃东西了。”
<br/>　　叶少阳坐在床上，缓了缓，想着翠云说的救下自己的经过，感觉有点匪夷所思，倒不是怀疑她的话，而是事情实在是有点离奇：自己之前还在空界，就算是昏过去，自己那些小伙伴也不可能随便就把自己丢在一家人门口等死吧？
<br/>　　这中间一定发生了什么事！
<br/>　　“翠云姐，你当时除了我，还看到什么人没有？或者说，还有没有什么别的情况？”
<br/>　　翠云歪头想了半天，摇头说道：“没有，我就只是看到你一个人，我生怕你是被土匪赶的，等了小半宿，没啥动静，再看你身上的伤，好像是野兽抓的，这才找邻居把你弄进屋的。”
<br/>　　“土……土匪？”叶少阳傻傻地看着她。
<br/>　　翠云点了点头，“对啊，有一些是土匪盯上的人，或者从土匪窝里跑出来的，那真是不敢救的。”
<br/>　　叶少阳感觉像在听天书，怔了半晌，说道：“这都什么时代了，还有……土匪？”
<br/>　　翠云听他这么一说，也是愣住了，说道：“就是这时节，土匪才多啊，不过我们这个镇子挨着县城，倒还好的，听说山里的土匪可多了。”
<br/>　　叶少阳直勾勾地盯着她，看了足足有十几秒钟，猛然想起了什么，深吸一口气，声音发颤地说道：“现在……是哪一年？”
<br/>　　翠云听他这么问，更是吃惊，反问道：“你不知道？”
<br/>　　“请你先告诉我。”
<br/>　　“民国十一年啊，四月初八。”
<br/>　　民国……叶少阳大脑卡壳了一下，随后才想到民国这两个字的涵义，抬头问翠云：“翠云姐，你没跟我开玩笑？”
<br/>　　“这有啥子开玩笑的？”翠云比他还吃惊。“今年是民国十一年，这是谁都知道的啊。大兄弟你真失忆了？”
<br/>　　叶少阳点点头，“很多事情都记不起来了，民国十一年，是……公元哪一年？”
<br/>　　“公元？是什么？”
<br/>　　叶少阳这才想起，在民国时候八成还没有公元这个说法，就算有，一般老百姓也未必知道，当下对翠云勉强一笑，说道：“翠云姐，你为什么要救我？”
<br/>　　“为什么？”翠云吃惊地反问，“你倒在我家门口，还有一口气，我总不能见死不救吧。”
<br/>　　“你一个寡……嗯，自己住的，不怕传什么闲话？”
<br/>　　翠云说道：“怕啊，所以我说你是我表弟，逃难来的，现在兵荒马乱，到处都是逃难的流民，这是没人怀疑的，反正我也是外地嫁过来的，没人知道我底细。”
<br/>(本章完)
