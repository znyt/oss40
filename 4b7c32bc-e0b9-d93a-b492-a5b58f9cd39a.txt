　　后门是一个小的拱门，也是木头的，陈悦过去推了一下就开了，三人走了进去。
<br/>　　道观只有一重院子，中间是个道场，四面都是瓦房，接壤在一起，看上去像个四合院，道场中间有一只巨大的香炉，边上有一副香火架，里面全是蜡油，说明这里香火还不错。
<br/>　　陈悦四下张望了一番，大声叫起来：“师父！”
<br/>　　一脸叫了几声，没人答应。
<br/>　　叶少阳提议找一找，于是三人分开，一人负责一排房舍，挨个去找，遇到门锁上的就去敲门，趴在窗户上看。
<br/>　　叶少阳找的这一排，清一色全是卧室，想到这道观就他们师徒两个，这些房间应该是给那些香客住的，有些比较远的，或者遇到暴雨之类的，必须在山上留宿，这些厢房就是为他们准备的。
<br/>　　叶少阳趴在窗户上一间间看进去，里面设施简陋，只有一张床和一个柜子，一眼就能看个大概，叶少阳很快就找完了，来到陈悦那边。
<br/>　　“没人，除非有人藏在柜子里。”叶少阳说道。
<br/>　　瓜瓜也找了一遍，回来告诉他们，那边是厨房和仓库，里头也没人。
<br/>　　陈悦在一间房门外站了许久，用力推了一下门，没开，她伸手在门锁的位置摸索了一番，抓住门把手用力一提，门开了。
<br/>　　“这么熟悉？”叶少阳道。
<br/>　　“这是我的卧室，看到这扇门，我就想起来了。”陈悦迫不及待走了进去。
<br/>　　房间里家具比那些厢房里要多，但还是很简约，墙有点黑，地面还是水泥地，靠墙摆着一架木床，床上摆着一只很大的海绵宝宝的布偶，边上叠放着几件衣服，最上面是几件内衣。
<br/>　　陈悦红着脸走过去，把内衣塞到下面的柜子里。
<br/>　　叶少阳假装没看见，在房间里转悠了一圈，道：“这就是你住的地方，怎么连个电视都没有。”
<br/>　　陈悦走到对面的矮柜前，指着上面说道：“这里原本有个电视，因为山顶总停电，放着一段时间受潮坏了，就再也没买，我也不太喜欢看电视。”
<br/>　　“那你干什么，看书？”叶少阳发现了一个书架，走过去，一眼看过去，大部分都是经书，还是手抄本，只在书脊上用毛笔写了名字，有道德经、清静经、太上感应篇、逍遥游等等，都是道家的典籍，边上那个书架上倒是有一些出版书籍。
<br/>　　不是世界名著，就是梦溪笔谈、山海经、阅微草堂笔记这样的古典名著，最下面一排，还有一些诗集和当代小说。
<br/>　　“你真的有文化……”叶少阳表示佩服。
<br/>　　“山里没什么可以做的，就是看书了。”
<br/>　　“不不不，不要谦虚，跟山里没关系，我也在山里住过十几年，我就没你这么好学。”叶少阳不好意思地挠了挠头。
<br/>　　陈悦在房间里慢吞吞地走着，最后来到窗前，打开窗户看外面，外面就是山体岩壁，与窗户中间有一小段缝隙，种着一丛花，还有一棵芭蕉树。
<br/>　　“你们不要打扰我，让我静下来想一想。”
<br/>　　陈悦说着，趴在窗户上，望着芭蕉树发起呆来，过了好一会，她又起身走到外面院子里，慢吞吞地走着，半晌，她突然深吸一口气，转头看着叶少阳，道：“我全想起来了……”
<br/>　　不等叶少阳搭话，她快步走向一个房间，房间锁着门，陈悦站在门前说道：“这就是我师父的房间。”
<br/>　　“人不在？”
<br/>　　门是木头的，因为时间太久，中间有一些很大的缝隙。陈悦趴在门缝上看了一会，突然浑身一颤，用力一脚提开门，冲了进去。
<br/>　　叶少阳不知道发生了什么，跟着进去，一眼看到床上坐着一个人，再一看，当场惊呆了：床上的这个人，是一具尸体。
<br/>　　一个身穿长衫的妇人，头发半白，看上去得有六七十岁，盘膝坐在床上，可能已经死了很久，血肉都风干了，一张脸皮紧贴着骨头，闭着眼睛，却诡异地保持着一动不动的姿势。
<br/>　　因为是冬天，尸体没腐烂，而是直接被风干了，成了一具干尸。
<br/>　　“师父——”
<br/>　　陈悦扑倒在尸体面前，放声大哭起来。
<br/>　　瓜瓜默默在一边陪着，安慰她。
<br/>　　陈悦放声哭了好一会，逐渐冷静下来，哽咽道：“她是自杀的。”
<br/>　　“看样子是。”叶少阳道，如果是他杀，没道理凶手还要把她弄成这种坐化一般的姿势，而且她身上也没明显外伤。
<br/>　　“但是她没理由自杀……我师父清修几十年，早就看破一切，她不可能自杀的，不可能的……”
<br/>　　叶少阳不认识石道人，不好搭话，想了一下说道：“你节哀顺变，看她死得这么从容，估计会留下点什么遗言之类的，你可以找一找。”
<br/>　　这句话提醒了陈悦，当下在房间里翻找了一遍，还是没任何发现。
<br/>　　叶少阳劝她别着急，今晚可以住在这里，慢慢找一下。
<br/>　　两人商量了一下，决定应该让死者入土为安，这时候陈悦已经恢复记忆了，自己去到库房，拿来一把铁锨，让叶少阳帮忙背着尸体——一般人是不敢背尸体的，主要是怕晦气，但叶少阳一个上仙（前几章说叶少阳是灵仙，实在是记错了，特此更正），任何晦气也沾不到他身上去。
<br/>　　三人出了后门，陈悦想了一下，决定把她安葬到山顶的一个地方，石道人生前喜欢在那里清修，至于风水什么的，她也没后人，就不讲究这个了。
<br/>　　从后门绕过去，有一条小路，延伸到绝顶。
<br/>　　叶少阳背着尸体，一边思索整件事，越发觉得诡异：之前他检查过了尸体，死亡时间至少有一个月以上，而根据陈悦的记忆，她是一个多月前失踪的（关于如何失踪的，她全无记忆，印象就头天还好好地躺下睡觉，之后的记忆就没了，被切换到了虚幻世界），那时候她师父还是好好的。
<br/>　　也就是说，她师父是在她失踪之后死的，而且不会相隔太久……这两件事之间，是不是有什么因果关系？
