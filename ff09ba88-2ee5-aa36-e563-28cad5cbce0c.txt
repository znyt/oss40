
<br/>　　<br/>　　    寻常人眼中的厉鬼，基本都是面目狰狞神情恐怖，呲着獠牙七窍流血什么的，但其实殊不知若论鬼的表情和人也差不多，它们多数时候展露出来的一面只是为了吓唬人而已，七情六欲是也会有的。<br/>　　<br/>　　    绣花鞋女鬼神情焦急的飞舞而去，眼中满是惶恐和愤怒，一下子就扑到了第二具尸体的脚下，那是个生前大概二十岁左右的青年，可能原本长相比较清秀，但死后却变得脸色铁青没有一点血色了。<br/>　　<br/>　　    一行血泪从女鬼的眼中滑落，随后散成雾气悄然消逝了，女人流泪多数为情。<br/>　　<br/>　　    此时再见，不是阴阳两隔也不是人鬼殊途，更不是隔了一个世纪，而是我就在你的身边，但你却永远都触摸不到了。<br/>　　<br/>　　    “阿郎，你还记得你被拉去当兵时对我说的那番话么·····”女鬼抱着尸体的双脚，轻声呢喃着说道：“姑娘往北走，阿郎向南行，今生就此别过，不再打搅，望翠儿姑娘日后，善其身，遇贵人，与君欢笑，暖色度余生。”<br/>　　<br/>　　    女鬼叫秦翠儿，当年她青梅竹马的阿朗哥被拉去征兵，临行前对她说过的一番话，大概的意思就是，此时我离你而去可能就永无相见之日了，上沙场生死两不知，希望你能够再寻郎君碰到个好人，无忧虑的度过余生。<br/>　　<br/>　　    阿郎的尸体又轻微的颤了一颤，额头上贴的那道符纸又轻飘飘的飘了一下，但最终仍然贴在他的额头上没有飞去。<br/>　　<br/>　　    “咦？”东干看着这一幕，说道：“真是怪了，他的魂魄明明都已经被封住了，怎么还能够蠢蠢欲动呢？想来是这女人让他产生了极大的触动，哼哼，可莫要坏了我的好事才行，这几具尸体我们足足孕养了百年之久，中途眼看着要收割回去了，但却莫名的被烧了一具已经够我心疼的了，这具不能再有失了，文华你且把这女鬼收了，让她安生一点”<br/>　　<br/>　　    “知道了，东干叔”文华答应了一声，拎着那盏阴魂灯朝着秦翠儿走了过去。<br/>　　<br/>　　    站在不远处的茅小草在心底叹了口气，开口说道：“几位，能否商量一下？”<br/>　　<br/>　　    东干，文华全都望了过来，小草姑娘说道：“我答应过她，为她寻到夫君然后安葬在一起，所以特意千里迢迢的赶到了长安，还算幸运，寻到了她夫君的尸身，应该就是她此时抱着的那具尸体了吧？几位可否成鬼之美，这对有情人生前无法白头偕老，死后能否让他们葬在一起？”<br/>　　<br/>　　    “那管我什么事？这几具尸体我们养了百年，他生前算是个有血有肉的人，死后就什么都不是了，连个收尸的人都没有，那就是无主之物了，我们捡到了那自然就归属我们了，你和我讲的那个凄美的爱情故事，没有一点能够打动我的，看你不惧不怕的想来也是有点见识的人，识相的就把这女鬼带走，我当没见过你，至于这五具尸体我则要马上赶走了”<br/>　　<br/>　　    茅小草皱着两道细细的好看的美貌，不瘟不火的说道：“我只要那一具，送他和这女子去安葬就行了，其余的我也管不着”<br/>　　<br/>　　    东干被气笑了：“你听不明白我的话是么？再和我磨叽下去我可就没有耐心了，到时候说不得收了这女鬼之后，也得把你给收了”<br/>　　<br/>　　    “唉·······”小草叹了口气，挺认真的说道：“你也听不明白我的话是么？再和我磨叽下去，我也没有耐心了，到时可能我就不只是要这一具尸体了，剩下那几具我也未必想要给你留下了”<br/>　　<br/>　　    两个人说了几乎相同的一番话，茅小草和东干之间就悄然寂静下来，只有微弱的喘息声。<br/>　　<br/>　　    东干愣了半天，文华和另一个中年一脸怒气，小草姑娘轻声和秦翠儿说道：“在渭河边上我就答应过你，寻到你夫君后让他和你一同安葬，话都说回去了哪有收回来的道理？你先回来吧，明年的今天就是你跟他的忌日了，我答应的就肯定会办的到”<br/>　　<br/>　　    秦翠儿听闻后，不舍的松开阿郎的两腿，抹了把眼角的血泪飘到了茅小草的身旁，她也不知道自己为什么会对这个看似好像是一株普通小草一样的女人这么信任。<br/>　　<br/>　　    小草姑娘说完以后，眼神就飘向了东干，对方从她的眼中看到了一抹警告的意味，其中更有一种你若敢动手，我就敢还手的意思。<br/>　　<br/>　　    “真是好笑，你还敢对我们三个下手不成？呵呵，好，好，好，有多少年了我们都没有活祭过生尸了，今天你撞了过来，这里正好四下无人，那我就拿你填补上之前被烧掉的那具养尸好了”东干朝着两个中年说道：“文华，秋生，杀了他吧！”<br/>　　<br/>　　    小草昂着脑袋说道：“湘西老苗养尸宗，专善孕养死尸，可将尸体养成一具僵尸，然后再
