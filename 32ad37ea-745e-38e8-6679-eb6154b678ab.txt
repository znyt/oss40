　　“什么！！”
<br/>　　绝无敌看到叶尘突然窜了出现，而且气息浑厚，似乎完全没有受伤，顿时两眼瞪得老大，满脸的不可思议，
<br/>　　“不！这不可能！！”
<br/>　　他方才这一招万佛千影，乃是擎苍宗的无上拳法，凭借这一招，他可是曾经斩杀过不止一位返虚真君。
<br/>　　然而现在，竟然未能伤到眼前这个小子分毫？
<br/>　　他真的只有元婴境吗？
<br/>　　不但绝无敌懵了，就连其他众天骄，一时间也全都愣住，包括龙辰在内。
<br/>　　虽然他早已经猜到，叶尘不会这么轻易，被绝无敌杀死，可是两人的修为差距实在太大！
<br/>　　在他看来，叶尘纵然不死，至少也要受不轻的伤势，却怎么也没有想到，他不但抗下了绝无敌的强悍一击，而且还像没事人一样！
<br/>　　即便这并非绝无敌的最强战力，可是能做到如此程度，这恐怕，至少也是天骄榜排名前二十的实力了吧？
<br/>　　而且看他的年龄，似乎只有二十岁上下，而且修为也只有元婴境，修真界何时出了这么一位天骄，为何之前从来没有听说过？
<br/>　　就在龙辰低头沉思之时，那绝无敌仿佛受到了巨大刺激，
<br/>　　“我不信！我绝不相信！今天你必须给我死！！”
<br/>　　绝无敌无法接受这一事实，顿时大吼了一声，右手猛然一翻，黄金升龙枪已经再次出现在他的手掌之中，
<br/>　　轰！
<br/>　　强大的杀气，自绝无敌的身上涌出，瞬间直冲云霄！
<br/>　　绝无敌高高举起黄金圣龙枪，朝叶尘狠狠刺落下来！
<br/>　　很显然，这一次，绝无敌是真的对叶尘动了杀心！
<br/>　　“不可！！”
<br/>　　地面上的龙辰，反应过来之后，想要阻止却已经有些来不及了。
<br/>　　叶尘见状，不由得眉头微微一皱，
<br/>　　动用了仙器法宝，而且还是暴怒状态下的绝无敌，已经足以威胁到他的性命！
<br/>　　“看来只能动用灭世魔刀了……”
<br/>　　就在叶尘也准备动用自己的最强底牌之时。
<br/>　　轰！
<br/>　　天空中，突然一道冰蓝色的剑芒，划破苍穹后，竟然替叶尘挡下了那杆金色长枪！
<br/>　　铿！
<br/>　　轰隆隆！！
<br/>　　冰蓝色长剑和那金色长枪，在空中相撞之后，周围空间都被震的破碎开来。
<br/>　　而接下来的一幕，就连叶尘也不由得两眼一瞪，脸上浮现出一抹惊愕之色。
<br/>　　两件法宝相撞之后，被震飞出去的，竟然是绝无敌的金色长枪！
<br/>　　非但如此，连同绝无敌本人，也连连倒退，足足退出百米之外，方才勉强站稳身形，同样满脸骇然之色。
<br/>　　这一幕，说时慢，其实不过就在电光火石之间发生。
<br/>　　等到周围空间，再次稳定下来，
<br/>　　九天之上，叶尘和绝无敌之中，不知道何时，多出了一道风华绝代的女子身影。
<br/>　　这女子，一身白衣，身上几乎看不到一件饰品，只有那一头长发，被一根细绳束着，披落在后肩，普普通通的打扮，却反而更加凸显她那远超常人的绝世风姿，
<br/>　　不过，她那精致到至极的脸庞之上，却布满寒霜，让人远远的望上一眼，都觉得寒气逼人，不敢直视。
<br/>　　“是她！”
<br/>　　叶尘看到这女子出现，不由得眼眸微微一亮。
<br/>　　“绝情剑！是洛冰玄！”
<br/>　　下方人群之中，顿时已经有人开始惊呼起来。
<br/>　　绝无敌稳定住身形之后，看到洛冰玄，脸上也不由得浮现出，一抹深深的忌惮之色。
<br/>　　前十大天骄之中，排名第一的龙辰，和排名第二的洛冰玄，实力是要远超其他八人的，这几乎是一个公认的事实。
<br/>　　绝无敌这些年，一直刻苦修炼，修为提升飞快，本以为可以拉近和洛冰玄的距离，但是方才交手那一下，让他瞬间就意识到，自己和洛冰玄之前的差距，不但没有拉近，恐怕反而变大了！
<br/>　　“呵呵！一向以绝情著称的冰玄仙子，没想到也会出手救人，莫非这小子，是仙子你的人？这可真是罕见啊！”
<br/>　　绝无敌自知不是洛冰玄的对手，不敢再继续出手，于是便忍不住调侃了起来。
<br/>　　洛玄冰却连看都没有看绝无敌一样，当然同样也没有看叶尘，语气冰冷无比，道：
<br/>　　“我救他，是因为他现在还不能死！等进入迷罗星海，你们谁死谁活，我才懒得管！”
<br/>　　绝无敌还想多说，这时下方的龙辰，忽然又跟着开口了，声音中透着一抹怒意，
<br/>　　“绝师弟，三招已过，难道你想反悔不成？你若是没过瘾，我来陪你玩玩如何？”
<br/>　　绝无敌自然也听出了龙辰语气中的怒意，顿时脸色微微一变，不敢再多说什么了，乖乖退回到那高台之上。
<br/>　　开玩笑，连洛冰玄都有如此实力，那排名第一的龙辰，他自然更加不是对手，哪里还敢再继续挑事？
<br/>　　“多谢冰玄仙子相救！”
<br/>　　叶尘身体一晃，来到洛冰玄跟前，冲着她拱手行了一礼道。
<br/>　　“你想多了！我从未想过要救你！”
<br/>　　洛冰玄一脸淡漠，依旧没有正眼看叶尘，撂下一句话，也飞身落回高台之上。
<br/>　　叶尘知道洛冰玄就是这个脾气，倒也不以为意，不过已经开始寻思着，准备找个机会，和这女人套套近乎，打听一下曦月的情况。
<br/>　　随着洛冰玄的到来，巨石碑上的那一百个人，也已经全部到齐。
<br/>　　轰隆隆！
<br/>　　那巨大石碑似乎真的可以，感应到所有通过考核者的气息，一阵剧烈晃动之后，在巨碑的中央位置，突然裂开了一条巨缝，犹如一扇门，而在门后，则是无尽的虚空，这应该是一个类似于传送阵之类的东西。
<br/>　　嗖嗖嗖！
<br/>　　从那扇空间门之后，突然窜出一道道流光，继而飞到每个人的面前。
<br/>　　“这个是传送玉简，大家务必要保管好此物，一旦里面遇到威胁到生命的危险，只要捏碎此物，就会被传送回初始星，是你们的救生符！”
<br/>　　龙辰朗声向众人介绍了一番，将自己跟前的传送玉简收入乾坤镯之中，便率先昂首踏入那扇空间传送门。
<br/>　　紧接着，便是洛冰玄、云破天、燕狂生、风华舞、绝无敌……
<br/>　　就在绝无敌，踏入空间传送门的时候，忽然回头望了一眼人群后端的叶尘，冷冷来了一句
<br/>　　“小子，你最好祈祷，不要在里面遇到我！”
