　　第282章
<br/>　　大河之上，船分浪花，几艘大船向前方驶去。
<br/>　　几艘船来自不同地方，却有一个共同的目的地。
<br/>　　“祁兄，一向可好”
<br/>　　“原来是朱兄，幸会幸会。”
<br/>　　两艘船上的人拱手打着招呼。祁珲坡，朱五苍都是莫鸣米封的山贼，在琴川一带各霸一地。今天是得到窦竹仝传来的消息，同时去往窦家寨。
<br/>　　一路上，还碰到诸多人，都是各家的首领。如果花皮虎本身，还没有如此大的能量，但大家都知道，这件事情背后真正发号施令者其实是莫鸣米。
<br/>　　大约在中午大家赶到窦家寨，花皮虎亲自在寨门口迎接。
<br/>　　“祁老大，您来得可早，快快里面请，严军师已经在里面等着了。”
<br/>　　“严军师也到了？”祁珲坡一惊。
<br/>　　“是，严军师上午就到了。”
<br/>　　祁珲坡快步走入山寨，知道这次琴川关会派人来，却是没想到出马的是严笛。在此地，严笛可是不能得罪的大人物。
<br/>　　大家在山寨大厅聚齐，祁珲坡看到，除了云中龙外，十八家校尉都已经到了。云中龙那里，也派了周差海来，只不过，这里是窦竹仝的地盘，今日窦竹仝却没有现身。
<br/>　　“窦家嫂子，窦大哥呢，我们兄弟可是好久没见面了。”朱五苍。
<br/>　　“喔，我家男人得了小病，现在在内院养病呢，今日的事，我在场也是一样的。”花皮虎笑着说，可心里却只想骂娘。知道吕春花的尸体被烧后，窦竹仝便一病不起，现在全靠一口参汤吊着。
<br/>　　对于花皮虎，大家也不敢不给面子。虽然花皮虎是个女人，可一身武艺当真不弱，而且论心狠手辣之处，比窦竹仝有过之而无不及。
<br/>　　更何况，她背后的人可是莫鸣米。
<br/>　　十八家都到了，这些人都是逞凶斗勇的凶恶之徒，互相见着面，污言秽语响成一片，直到严笛出现后，大家才安静下来。
<br/>　　“诸位，不用说大家也明白，今天大家都是为同一件事来，为的，就是那程大雷。”花皮虎开口道：“程大雷的名声，我想大家都听说过，说实话，是个人物。可这里，是我们的地盘，大家兄弟在一条河上吃饭，他来这里，就要抢大家的饭，我问各位兄弟，咱们能答应么？”
<br/>　　“既然他敢来这儿，就得听莫老大的，是龙他得盘着，是虎他得给我卧着，敢不听话，兄弟们就教教他做人。”
<br/>　　“要大家来，为的就是这个。琴川一地，没有程大雷吃饭的碗。”花皮虎斩钉截铁。
<br/>　　严笛此刻才开口：“程大雷手下不过一二百人，上次窦竹仝吃了憋，也主要吃亏在水中的毒蛇上。这倒也好解决，用雄黄将蛇驱走，大家乘小筏子上岛。现在的关键是，各家能够派出来多少人？”
<br/>　　大厅内的声音安静下来，这些人虽然是一帮大老粗，但其实个个都鬼精的很。这种出白功的事情，他们是不会做的。
<br/>　　“只要这次平了程大雷，莫将军免大家三月的税。”严笛。
<br/>　　一听到这话，诸人的眼睛都亮了。虽然十八家各守一方，垄断着私盐，又想过路的商人索要保护费，可他们剥削附近的，莫鸣米同样剥削他们的，说白了，他们也不过是给莫鸣米打工而已。如果能有三个月不用上供，这可不是一笔小数目。
<br/>　　“严军师放心，我早就看姓程的不顺眼了，我们出三百人。”
<br/>　　“那姓程的不管在别的地方多狂，到了这里都不好使，我们出二百人。”
<br/>　　诸人你一言，我一语，大厅内又乱做一团。你出二百人，我出三百人，最后十八家一共凑了三千人。
<br/>　　等声音慢慢静下来，严笛从怀里掏出一份地图。
<br/>　　“这就是蛇岛的地图，大家一起看看，我们该当从何处进攻。”
<br/>　　“严军师，几千人打一百人，还从哪里进攻，杀上去便是了。”
<br/>　　……
<br/>　　蛤蟆岛上，工程仍然在热火朝天进行，三班倒做了十天，再加上有建筑精通技能的加成，目前已经接近尾声。
<br/>　　本来岛上郁郁葱葱的大树被伐倒一半，聚义厅，医馆，点将台，瞭望台……几乎是拔地而起，蛤蟆岛外围修了明暗两处码头，还有一座待完工的船坞，在水地，船比马管用。
<br/>　　初此之外，还有防御设施，高处的箭楼，林地中的陷阱，水里埋着暗礁。
<br/>　　等到这一天，所有工程已全部竣工，蛤蟆寨从被遗弃的破落山寨，成为二级山寨。
<br/>　　“所有人往一块站，一块儿站，前后对齐，不要左摇右晃。”
<br/>　　演武场，徐神机搬了张桌子过来，把抢来的壮丁聚在一起。
<br/>　　“三哥，他们这是干什么？”人群中有人悄悄嘀咕着。
<br/>　　“估计是卸磨杀驴，过河拆桥吧，现在活忙完了，就用不着我们了。”
<br/>　　“放我们走……”
<br/>　　“也有可能是让我们入伙当山贼。”
<br/>　　徐神机和刘悲坐在一张桌子后，张肥高飞豹等人指挥着队伍，一个个上前。
<br/>　　“姓名？”徐神机
<br/>　　“赵三壮。”
<br/>　　“从哪儿来的？”
<br/>　　“青叶镇。”
<br/>　　“职业？”
<br/>　　“瓦匠。”
<br/>　　徐神机问，刘悲登记，笔落在字上。刘悲从箱子里拿出一吊钱。
<br/>　　“瓦匠的工钱是一吊，拿钱在旁边排队。”
<br/>　　赵三壮拿着钱，一时有些摸不着头脑，听说过杀人前管顿饭的，可没听说谁杀人前给钱的。
<br/>　　“这个排队，排队干嘛啊？”
<br/>　　“凑够一船人送你们上岸，你去旁边等着吧。”
<br/>　　“真的放我们回去？”
<br/>　　“怎么，你不想回去。”徐神机道：“我们现在也在招兵买马，来来，想报名的往那边站。”
<br/>　　诸人都拿到了钱，站在一起如坠云里屋里，有拿到一吊的，也有几百文的，如木匠石匠则能达到三四吊。
<br/>　　这些人面朝黄土背朝天，奔波一年也就求个肚子不饿而已。像这次忙碌十天，早晚管饭，最后还能拿到真金白银，这样的机会可不多。
<br/>　　“军师，那个……我想问一下，咱们这儿以后还有活嘛？”赵三壮道：“您要是还有活儿，我还可以来。”
<br/>　　“这不把大家都登记在这里嘛，以后我们或许还会有用得着你们的，到时候按着花名册招人。对了，你不是青叶镇的么，你就是青叶镇的负责人。”徐神机顿了顿：“当然，谁要是出去乱说话，你们妻儿老小我们也不是找不到。”
<br/>　　
<br/>(本章完)
