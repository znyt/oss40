　　第566章
<br/>　　程大雷也是一路提心吊胆过来的。他按照楚云生指点的方向寻到大牢，但谁知道大牢中有没有埋伏，万一自己走进去之后，牢门一关，冲出三五百埋伏的高手，自己岂非插翅难逃。
<br/>　　小心翼翼的走入牢房，一路胆战心惊，倒是没看到有埋伏的人。刚靠过来，就听见刘发财的话，程大雷心中感动，心道：瞧瞧，一见如故，相逢就是兄弟，真兄弟也就这样了。
<br/>　　当然，刘发财之前的话他就没听见了。
<br/>　　李亚子腾身而起，抓住椅子丢了出去，程大雷一脚踹飞，砸在墙上碎成稀巴烂。
<br/>　　“聊聊？”程大雷冲他挑挑眉毛。
<br/>　　在经过最初的慌乱后，李亚子已镇定下来。毕竟是书院出来的高徒，他也并非草包。程大雷突然出现在这里，的确出乎自己预料，但自己若能在此地杀了他，岂非也可以令所有人大吃一惊。
<br/>　　只要你敢要，有时候这世道就会给你。
<br/>　　他目光转了一圈，寻到墙角一根铁刺，这是给犯人动刑用的。其实书院也没有别人爱用，李亚子倒是颇爱此道。
<br/>　　“来得好。”
<br/>　　李亚子抓起铁刺，身子腾起，将铁刺做剑使，一剑向程大雷袭来。书生意气剑中的一往无前，招式犀利，动作干净，却也是个剑道高手。
<br/>　　程大雷撇撇嘴：“这是我的台词呐。”
<br/>　　面对袭来的一剑，程大雷眯起眼睛，身子微微一侧，便避过这一剑。随之匹夫剑出鞘，李亚子眼前一花，只感觉剑光一闪，程大雷整个人已经消失。
<br/>　　“啊！”
<br/>　　一只血淋淋的手掉在地上，李亚子发出一声哭爹喊娘的惨叫，身子倒在地上。
<br/>　　程大雷的剑已经收回鞘中，若论对剑道的理解，程大雷其实并没有多少。但若论快剑，程大雷不敢自称天下第一，但的确也没遇上比他更快的。
<br/>　　李亚子想要在程大雷面前动剑，颇有种螳臂当车，不自量力的感觉。直到现在，他才约莫明白自己和程大雷的差距。
<br/>　　在他眼里，程大雷简直深不可测。若在游戏中，程大雷就是头顶一大堆问号的大boss。
<br/>　　大boss没有搭理他，一剑斩断吊着刘发财的绳索，刘发财身子一歪倒在地上，程大雷忙扶住了他。
<br/>　　“大当家。”刘发财用虚弱的声音道：“从今往后，我这条命是你的了。”
<br/>　　他受尽酷刑，全凭一口气撑着，也就是见到程大雷，这口气才松下来。
<br/>　　“来，我给你个报仇的机会。”程大雷指了指地上的李亚子。
<br/>　　“我实在……不行了。”刘发财两眼一黑，晕了过去。
<br/>　　李亚子怎么不知道，死神已经冲他露出了獠牙。他忍着剧痛站起来，捧着断手向外逃去。
<br/>　　程大雷要顾忌刘发财，就没办法追他，如此，也只能任他走脱。
<br/>　　他将刘发财扛在肩上，迈步往大牢外面走，匹夫剑已经出鞘，心中也明白，接下来的路怕是不会很太平。
<br/>　　李亚子哇呀呀跑出大牢，口中没命的大叫：“他来了，他来了……”
<br/>　　张明理遇见了他，也是吓了一跳，看李亚子此刻的样子，像是看到狼的孩子。
<br/>　　“谁，谁来了？”
<br/>　　“程……那魔头到了。”李亚子在说出程大雷名字时，心底一阵颤栗，仿佛那三个字有莫名的忌讳。
<br/>　　“他……怎么到了？”张明理也是一楞，随之立刻道：“你去请夫子，我先带人拦住他，不得已，就在此地杀了他。”
<br/>　　等程大雷背着刘发财走出大牢时，就见自己面前呈扇字形围满了人。书生长袍，三尺青锋，说实话，模样还挺俊。
<br/>　　张明理持剑而立，冷声道：“程贼，你还想往哪里走。”
<br/>　　程大雷轻叹口气，冲张明理招招手：“来来，我看有那个不怕死。”
<br/>　　山上大战即将开始，山门前的战斗已经打得血流成河。
<br/>　　燕不归一柄重剑抡开，仗着就是一力降十会五个字。重剑无锋大巧不工，碰上了就与你拼力气，你如果力气没有他大，便会被一剑斩成两段。
<br/>　　打到现在，燕不归戎族人的身份人人都看得出来。戎族生长在草原上，天生就会不要命的打法。偏偏这三十六柄剑组成的书生意气阵，讲究的也是一往无前，不死不休。
<br/>　　双方拼起命来，剑阵不断被消耗，山门前倒下数具尸体。而燕不归的情况也很糟糕，他面对三十六人的缠斗，每一次都要面临七八柄剑的攻击，伤敌一千，自损一千一。
<br/>　　现在他整个人已经伤痕累累，虽然仗着战斗本能避过了要害，可身上细细密密的伤痕，令他宛若一个血人。
<br/>　　在这样打下去，没人可以杀死他，他也会因流血过多而耗死。
<br/>　　但是，他不在乎。黑贞死了，他的生命也失去了最在意的一件东西，心怀泼天恨，却无处消解。如今多杀一个人，心中就少一份郁闷，多一份痛快。
<br/>　　至于死，死又何惧。
<br/>　　宋寒之面沉似水，他不是求死的燕不归，他是个正常人。只要是个正常人，就不会觉得这样的牺牲是值得的。书院培养出的弟子，每一个都倾注了大量心血，怎么可以白白牺牲在此地。
<br/>　　他不得不下令，指挥剑阵调动，从攻击转向防守。
<br/>　　燕不归一次次向外冲撞，却撞不开重围，越打越憋屈，一腔恨无处发泄。
<br/>　　“来呀，杀呀。”
<br/>　　宋寒之不会再和他拼命，杀不死你，我就耗死你。
<br/>　　然而，这时候山上又有消息传来，程大雷那贼人现身了。
<br/>　　这正是屋漏偏逢连夜雨，两个魔头怎么同时出现了。张明理要他调些人上去，别让程大雷走脱。宋寒之气得够呛：调人，我这里人还不够用呢。
<br/>　　山上程大雷的情况要比燕不归好很多，他又没疯，不会让书院的剑阵有机会成型。背起刘发财，左冲右撞，在山上和张明理一帮人拼脚力。
<br/>　　程大雷除了快剑绝世外，轻身功夫也是相当不错。干净的起落，近身便杀人，轻巧灵便，此刻他也不管书院是不是有文化的地方，一心一意往外闯。
