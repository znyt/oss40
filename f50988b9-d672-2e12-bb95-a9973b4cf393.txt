　　一千一百亿财富，对其他人也许是一个天文数字，但是对楚风眠来说，却不一定算得了什么。
<br/>　　他拥有血海，这血海中的全部财富，足以以亿万来计。
<br/>　　楚风眠只需要将血海中的灵力抽出，将其中的血气全部精华，便是就足以换成最为精粹的灵石财富来。
<br/>　　“一千一百亿，这财富，此人真的拿得出来？”
<br/>　　在场许多武者，都不由的喃喃自语起来。
<br/>　　一千一百亿的财富，比起许多一流势力的底蕴还要多，如此巨大的财富，一个人真的足以拿得出来？
<br/>　　“乾元师兄，不要着急，此人肯定是在乱喊价，一千一百亿的财富，他怎么可能拿得出来。”
<br/>　　一号包厢之中，一尊圣王宗弟子急忙的说道。
<br/>　　“等他拿不出来财富，不用乾元师兄出手，这小子就会被通天阁毁灭了。”
<br/>　　“不错！”
<br/>　　乾元脸色，在听到这圣王宗弟子的话后，也变得舒缓了许多，开口道。
<br/>　　“小子，这次算你赢了，不错我倒是要看看，你到底能不能拿出这么多的财富来！”
<br/>　　“财富拿不出来的话，我现在就会将你斩杀！”
<br/>　　乾元的话中，尽是杀意。
<br/>　　无数眼神，看向楚风眠所在的十一号包厢。
<br/>　　的确，如此庞大的财富，楚风眠到底能不能拿的出来，都是一个未知数。
<br/>　　“拿不出来？你以为我跟你一样，是个穷光蛋，这么点财富，都拿不出来？”
<br/>　　楚风眠冷笑的声音，从十一号包厢中响起。
<br/>　　“好好看着！”
<br/>　　“轰隆！”
<br/>　　十一号包厢，猛然打开。
<br/>　　从其中，一条长河，一下子流淌了出来，从这长河中，乃是有着无数王品灵石，组成而成的。
<br/>　　甚至是在其中，都有些比起王品灵石，还要精粹的灵石。
<br/>　　“皇品灵石！”
<br/>　　一尊尊武者，看到这灵石长河的一刻，几乎是要陷入到了疯狂之中。
<br/>　　“不错，正是一千一百亿灵石！”
<br/>　　灵尊真人数了一下，便是数清楚，这其中正是一千一百亿灵石。
<br/>　　“这极道仙兵的残片，便是属于这位阁下了。”
<br/>　　灵尊真人手一抬，在那人皇罐碎片上的禁制，便是全然破碎，楚风眠大手一挥，直接就被楚风眠卷走。
<br/>　　“李长老，楚某就先走一步了。”
<br/>　　收起这人皇罐碎片，楚风眠直接对着李长老说了一句，便是带着白萱，身形一动，直接离开。
<br/>　　“此人离开了！”
<br/>　　“追上去！”
<br/>　　“此人的身上有着无尽财富，我们谁可以得到，就都足以一步登天！”
<br/>　　一道道身影，看着楚风眠离去的遁光，也都是纷纷追了上去。
<br/>　　楚风眠跟白萱，两道身影，几乎是在瞬息之间，便是离开了拍卖会场中，一下子离开了通天阁，在混元大陆中穿行。
<br/>　　“楚风眠，后面好像还有不少人在追，那一号包厢中的人，跟十号包厢中的人，都追了上来。”
<br/>　　一边飞速遁逃，一边白萱看着身后。
<br/>　　就在身后，几道遁光，便是一路追了上来，杀气腾腾，显然没有想要放过楚风眠的打算。
<br/>　　尤其是那乾元，跟古林二人，楚风眠在拍卖会上，抢了他们二人那么多的东西，打了他们二人的脸面。
<br/>　　这二人，便是丝毫没有想要放过楚风眠的心思，一个个追逐上来。
<br/>　　“无妨，就让他们追，这些人不知死活，我正好想要杀了他们，七大宗门的人，一个个都得死！”
<br/>　　楚风眠眼神中露出几分冰冷杀意。
<br/>　　这些人，既然是过来送死了，那楚风眠也就不客气了。
<br/>　　反正七大宗门的人，都是该死之人，楚风眠一个都不会放过。
<br/>　　两人的遁光，一路飞行，直接便是向着混元大陆最为偏僻的南方飞去。
<br/>　　“乾元师兄，此人的遁光速度好快，我们施展出的遁光，居然只能勉强跟上。”
<br/>　　就在楚风眠的后方，一群圣王宗弟子，正接连催动遁光，一直追逐这楚风眠的身影。
<br/>　　他们的遁光，都是玄妙无比，隐约之间，有着上古圣贤的气息，号称圣王，正道之王。
<br/>　　“哼，此人逃不走，他的身上，已经被我种下了禁制，不管是逃到天涯海角去，都无法逃出去。”
<br/>　　乾元冷声说道。
<br/>　　“他现在想要拼劲一切逃走，去炼化那极道仙兵碎片，哪有那么容易，此人敢跟我争夺，跟我为敌，便是死路一条，他的所有机遇，身上的无数财富，那极道仙兵的残片，便是都属于我的！”
<br/>　　“走，我们追上去！”
<br/>　　无数圣王宗弟子，便是都化为了一道道遁光，全部追了上去。
<br/>　　“古林师兄，圣王宗的人，好像都在前方？”
<br/>　　在这遁光之后，便是还有着一群人，也在追逐。
<br/>　　这一群人，正是十号包厢之内，荒古门的弟子。
<br/>　　“哼，圣王宗的人，据说那乾元，在古洞府中得到了大机缘，看他身上的财富，显然是得到了不小的机缘。”
<br/>　　古林看着前方，冷冷的开口道。
<br/>　　“我们在后面跟着，让那小子，跟圣王宗的人拼，按小子身上显然也得到了不小的机缘，他们争斗，受伤之后，我们在上，将他们一网打尽。”

<br/>　　“什么圣王宗的人，还有那小子，统统都会成为我的垫脚石，他们身上的一切机缘财富，都会属于我的！”

<br/>　　“乾元师兄，看样子此人停下来了。”
<br/>　　不远处，那几尊圣王宗弟子，也是察觉到了楚风眠停了下来。
<br/>　　“哼，肯定是这小子身上的灵力消耗不少了，所以才停了下来，我们直接追上去，将此人斩杀！”
<br/>　　乾元冷哼一声。
<br/>　　几道遁光，唰的一声，一下子冲了过来。
<br/>　　瞬息之间，几道身影，在楚风眠的身边，猛然降临。
