　　叶尘沉默了片刻，
<br/>　　以他如今的修为，的确还不是叶家家主的对手，可是真是因为如此，才更能激发出他的潜能！
<br/>　　一念及此，叶尘嘴角微微一扬，抬头望向不可一世的叶家家主，
<br/>　　“如果我拒绝呢？”
<br/>　　叶家家主眉头一皱，
<br/>　　“那你就是在找死！”
<br/>　　说完这话，似乎已经没有耐心的叶家家主，手掌猛然一握，
<br/>　　轰隆！
<br/>　　空中陡然出现了一道青色巨掌，一把便将叶尘死死握住，令他无法动弹，
<br/>　　“小子！看到了吗？这就你我之间的差距！蝼蚁终究只是蝼蚁，现在的你，只有两个选择，臣服！或者，死！”
<br/>　　“是吗？”
<br/>　　叶尘冷冷一笑，
<br/>　　“可惜，这两个选择，我都不想选！”
<br/>　　“青帝长生体，吞噬！”
<br/>　　呼呼呼！
<br/>　　随着叶尘一声暴喝，那青色巨掌上面的造化之力，顿时便开始疯狂的融入叶尘的体内。
<br/>　　原来，这三个月以来，叶尘不但将青帝长生体修炼到了小成境界，而且还借鉴了吞天神功，在青帝长生体的基础上，开发出来吞噬能力！
<br/>　　“咦！！”
<br/>　　叶家家主顿时便感觉到，自己的造化之力，居然在迅速的流失，不由得发出一声惊呼，当即第一时间收回了神通。
<br/>　　“好小子！居然还修炼了如此邪功，看来老夫还真是小瞧你了！不过，就凭你这点旁门左道，也想和老夫作对，未免太天真了！”
<br/>　　叶家家主说着，直接一掌拍落了下来，
<br/>　　轰隆！
<br/>　　一道硕大的巨掌，犹如一座小山般，朝下方的叶尘狠狠砸落下来！
<br/>　　而且周围，都是青藤围绕而成的巨大牢笼，令叶尘根本无处可逃。
<br/>　　叶尘只好举起双掌，奋力迎了上去，
<br/>　　砰！
<br/>　　叶家家主的修为，实在高出叶尘太多，即便对方并没有使出全力，也远远不是叶尘可以抵挡的。
<br/>　　叶尘只感觉自己的双臂，犹如托住了一座巨山，
<br/>　　咔嚓嚓！
<br/>　　双臂的骨头，在强大的力量冲击之下，竟然瞬间全部碎裂，连手臂上的血肉，也猛然炸裂开来！
<br/>　　整个人更是犹如炮弹般，狠狠砸落在下方的地面之上！
<br/>　　轰隆隆！
<br/>　　地面一阵剧烈的晃动之后，出现了一个近百米的深坑。
<br/>　　而叶尘深陷在巨坑之中，简直惨不忍睹。
<br/>　　“青帝长生体！造化修复！”
<br/>　　叶尘当即一声低喝，身上顿时青色大盛，造化之力开始传送到手臂之上，开始迅速的修复着伤势。
<br/>　　滋滋滋~
<br/>　　一转眼的功夫，叶尘手臂上的伤势，便已经好了七七八八。
<br/>　　“嗯？”
<br/>　　叶家家主看到这一幕，顿时眉头微微一挑，显然颇为感到意外，
<br/>　　他刚刚那一掌，虽然并没有打算要了叶尘的性命，可是按理说，废掉他应该绰绰有余了。
<br/>　　怎么也没有想到，叶尘居然这么快就恢复了过来！
<br/>　　“还真是一个邪门的小子！”
<br/>　　叶家家主既觉得震惊，又觉得好奇，于是再次一掌拍落了下去，而且比之前加大了几分力道。
<br/>　　轰隆！
<br/>　　随着地面再次一阵剧烈的颤动，叶尘再次遭到重击，整个人顿时又往下陷入数十米，在强大的力量冲击之下，全身上下的骨头，似乎全都要散架了一般。
<br/>　　“造化之力，修复！”
<br/>　　叶尘当即又催动青帝长生体，开始迅速的修复体内的伤势。
<br/>　　“啧啧！还真是耐打呀！”
<br/>　　叶家家主看到叶尘，在这么短的时间里，又将身上的伤势修复好了大半，不由得啧啧称奇，
<br/>　　“老夫倒要看看，你还能撑到什么时候！”
<br/>　　轰轰轰！
<br/>　　就这样，叶家家主每落下一掌，叶尘就身受重伤，但是很快便又会修复过来。
<br/>　　一转眼，叶家家主已经打了十几掌，叶尘虽然凭借着青帝长生体，以及对造化规则的理解，每次都能修复回来，但时间一久，体内的伤势还是越来越严重，
<br/>　　轰隆！
<br/>　　叶家家主再次一掌拍落下来，
<br/>　　“噗！”
<br/>　　叶尘顿时狂喷出一大口鲜血，再也支撑不住了，
<br/>　　“已经到了极限了吗？”
<br/>　　叶尘牟足了力气，终究还是没能再次站起身来，重重栽倒在地上，浑身都是血迹，简直凄惨无比。
<br/>　　叶家家主看到这一幕，也不由得松了一口气，
<br/>　　他还想从叶尘口中逼问出功法，所以自然不会让他就这么死了，可是这小子实在是太能抗了，他还真怕一不小心把这家伙给打死了。
<br/>　　“去！把这小子给我擒上来！”
<br/>　　叶家家主随手撤掉了周围的囚牢，冲着已经追赶过来的叶家众人，开口命令道。
<br/>　　叶家众人听到这话，顿时纷纷脸色煞白，谁也没有动身。
<br/>　　“这小子已经被我废了，怕什么？一群废物！”
<br/>　　叶家家主怒骂了一声，指了指叶家三大长老中一位，
<br/>　　“三长老，你去！”
<br/>　　三长老只好应了一声，纵身跃入那数百米深的深坑之中，朝已经犹如血人般的叶尘走了过去。
<br/>　　而此时，叶尘也感应到有人靠近，心中顿时升起一股强大的危机感，
<br/>　　奈何身上伤势实在太重，他现在已经动弹不得！
<br/>　　“我若是在死这造化大陆，多半就会是真的死了，永远也回不到真实世界……”
<br/>　　“不行！我绝不能就这么死了！绝对不能！”
<br/>　　“造化……造化……到底这造化法则是什么？”
<br/>　　“造化之道，便是天之道，天之道，损有余而补不足，所以才能衍生出世间万物，所以道法自然，修道便是顺天而为……”
<br/>　　“不对！错了！全错了！若是顺天而为，人又如何成为万物主宰？修炼之道，应该是逆天而为才对！”
<br/>　　“所以就要反其道而行之，损不足而补有余……”
<br/>　　轰隆！
<br/>　　这些念头在电光火石之间，从叶尘的脑海之中闪过，顿时犹如一道神雷，一下子就劈开了一直萦绕在叶尘头顶上的乌云，让他拨云见日！
<br/>　　而就在这个时候，叶家三长老，也已经缓步来到叶尘的跟前，
<br/>　　“受了这么重的伤，就算救活了也是废物一个！”
<br/>　　三长老自言自语了一句，之前无比凝重的脸庞之上，也不由得露出一抹轻蔑之色，然后便伸手朝叶尘抓去，
<br/>　　就在这时，
<br/>　　霍！
<br/>　　叶尘猛然睁开双眼！
