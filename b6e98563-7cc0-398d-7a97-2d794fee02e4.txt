
<br/>　　<br/>　　    “血祭之后，当时的情况很突兀，忽然之间地下的祭坛就开裂了，然后出现一道漩涡，当时我人正好在祭坛的正上方，漩涡出现以后我没来得及有任何的反应就被吸了进去”<br/>　　<br/>　　    林汶骐和李菲儿的口供一样，他俩说由于不知道水下有漩涡，从金字塔上往下游去的时候就正好撞上了，王令歌说的是他当时就在漩涡旁边，同样也没来得及有任何的反应。<br/>　　<br/>　　    四人讲述了下进入金字塔的经过后，在交代问题的时候，就已经到了金字塔的底部，身处一处水道中，等稳定下来就开始寻找出路，一直顺着水道来到了上一层，也就是金字塔的第一层。<br/>　　<br/>　　    这时候，四个人轮流叙述，从金字塔的下方开始讲，一直讲述到后来金字塔上放湖水倒灌进来他们这才找到机会冲了出去，期间并未有任何的隐瞒，唯独把林汶骐割下的那四十九张人皮的事给略过了，其他的都实话实说了。<br/>　　<br/>　　    撒谎这种事，并不需要你来胡编乱造，你只需要说九句真话，其中有一句是假的，这就会显得特别真实了，真要是撒谎的地方太多，你可能接下来就得需要无数句的谎话去圆了，到最后完全有可能把自己给绕死了，所以王惊蛰他们讲述的一点问题都没有，是经得起任何推敲的，毕竟湖下一百多米深的金字塔已经彻底被水给填满了，除非你把整个抚仙湖都给抽空了，不然金字塔内部的秘密是永远都没办法被揭露出来了。<br/>　　<br/>　　    他们几人在讲述的时候，旁听和记录的人都没有插嘴，一直仔细的倾听，等他们几人说完后，黄詹和张生利还有来自国安调查部门的人，就交头接耳的看着电脑轻声交谈了起来，一共两个笔记本电脑，其中一个播放的是水下古城的画面，另外一个是记录着他们刚才交代的经过。<br/>　　<br/>　　    二十多分钟后，对方似乎商量完了，黄九郎先是问道：“湖底的古尸有一部分进入了金字塔，那剩余的你们觉得，还会不会出现什么意外了，比如突然伤人，或者浮出水面之后再有什么别的异变？”<br/>　　<br/>　　    王惊蛰想了想，摇头说道：“应该不会了，我们提到过这个金字塔和祭坛都是由当时的滇国巫师设下了一个法阵，可能金字塔破了以后这个法阵也消失了，那些古尸不会再有任何的变化了”<br/>　　<br/>　　    来自国安的何平就发问道：“据你们推测的话，古滇国被沉到了水下，首先是出于什么原因？”<br/>　　<br/>　　    王惊蛰指了指林汶骐，说道：“这一点得让他来说了，对于探古墓，我是门外汉，他才是行家”<br/>　　<br/>　　    “好，你来分析下”<br/>　　<br/>　　    林汶骐略微思索了下，说道：“以我的经验和角度来看，应该是党羌为了保护他的墓，这才打造了一个庞大的工程，引水进入了古滇国，把整座城池都给葬在了水下，天下防盗墓的手段有太多了，无疑这么做是最保险的，因为党羌觉得没有任何人可能会深潜入一百多米深的水下，当然了，他也绝对不会想到两千多年以后，会有人研究出潜水装备来，把整个古滇国埋在抚仙湖下面，他就有可能重生或者是复活了，不然要是若干年后有人去了他的墓葬，搅了他的好事，就得不偿失了”<br/>　　<br/>　　    张生利顿时诧异的叫道：“这不可能，他这是为了一己之私把整个滇国都拉去垫背了？”<br/>　　<br/>　　    林汶骐斜了着眼睛冷笑道：“当年秦始皇打造始皇陵，最后建造的劳工死了多少你知道么？七十万人，七十万人全都被秦始皇下杀了埋葬着自己的陵寝旁边，两千多年的古滇国里有多少人，黄教授，有三十万么？”<br/>　　<br/>　　    黄詹摇头说道：“根据史料记载，滇国的人口应该是在二十万上下，还有我们现在也可以根据湖下古城的规模来推断，滇国当时的人口最多不会超出二十五万，古代的殉葬制度比我们想象的还要残忍，几乎每个朝代的帝王死后都有被殉葬的，少了几百过千人，多了上万十万的也不稀奇”<br/>　　<br/>　　    林汶骐摊着手说道：“秦始皇都能坑杀了七十万的劳工，党羌让这二十多万的滇国人给他陪葬，也不过就是一般而已了，反正他都要死了，谁还会管其他人的死活啊”<br/>　　<br/>　　    张生利顿时无言以对，那个国安的何平则接着谨慎的问道：“你们说党羌是为了复活或者重生，但为什么他的棺材却是空的，这意味着什么，是他成功了，还是他根本就没有被送进墓室中”<br/>　　<br/>　　    王惊蛰接过话，摇头说道：“这也是我们没有想通的地方，按理来讲既然党羌已经把整个滇国都拉下来给自己垫背，那他没有理由不进入到墓中，要说成功了的话也不太可能，他真要是成了，古城里的祭坛还有金字塔中，他用来逃生的布置也许就会失效，甚至压根就不存在什么抚仙湖古城了”<br/>　　<br/>　　    “这个解释，全靠你推断出来的？”何平皱眉问道。<br/>　　<br/>　　    “猜的！”<br/>　　<br/>　　    何平眼睛忽然紧盯着王惊蛰，审视的味道非常浓，但王惊蛰老神在在的坐在那完全没有任何的反应，林汶骐和王令歌也都是老油子了，表现的都是无懈可击。<br/>　　<br/>　　    随即，何平忽然朝着李菲儿问道：“李小姐你是如何认为的？”<br/>　　<br/>　　    “啊……”李菲儿下意识的看了林汶骐一眼，然后回过头说道：“我，我跟他们想的一样”<br/>　　<br/>　　    那位来自某办公室的邱秘书，靠向了何平，轻声说道：“不用问这些了，他们在水下一天一夜，你想问的问题，可能他们早就商量好了，不管你如何发问，都是在浪费时间”<br/>　　<br/>　　    何平小声回道：“就这么算了？”<br/>　　<br/>　　    “不然呢？他们怎么说就怎么是吧，要不然你觉得还有可能再去组织人把古城再探一遍？领队的蛙人不是已经汇报过了么，整栋金字塔里全都被湖水给淹了，里面情况复杂进都进不去，你又如何去判断他们说的真假？”<br/>　　<br/>　　    何平顿时哑口无言，他就是心有不甘，此时也拿出任何的章程了。
