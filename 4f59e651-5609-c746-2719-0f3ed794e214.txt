　　第602章
<br/>　　野原火需要一场胜利，告诉草原上的所有部落：他来了，他将统治这里。
<br/>　　钟牧山的能力在这次行动发挥了很大作用。他挥毫泼墨，发挥自己所有文采，向北蛮部下了一封战书。用典繁复，佶屈聱牙，大意是天下至宝有德者居之，而他辅佐的野原火便是一位有德之主，目的就是取天下。
<br/>　　哪怕金问道酷爱帝国文化，平素里也爱舞文弄墨，但这次也是费了好大力气，才读懂战书的大概意思。
<br/>　　战书送到北蛮部后，野原火亲自率领大军，集结兵力，出征北蛮部大本营。
<br/>　　这件事，引起草原上许多部落的注意。戎族各部落之间也并非一片和睦。他们攻打帝国时，自然同舟共济，不过在他们内部，也是打来打去。
<br/>　　金问道早就在召集各部落，以防御野原火的攻击。从见到他的第一天开始，金问道心里就明白，野原火壮大之后，第一个目标肯定是自己。
<br/>　　但在这件事上，他收获不大，响应他号召的部落并无多少。向呼延力求援，也是肉包子打狗，一点回音也无。所以，金问道在兵力上虽然占优，但并没有占有绝对优势。
<br/>　　冬天到了，草原上越来越冷。西北风刮过，只见荒草翻滚，不见一点生机，除了饿的饥肠辘辘的草原狼，什么都看不见。
<br/>　　金问道走出大帐，结结实实打个寒颤。他向远方看去，只见不远处的山坡上，敌军已经列阵完毕。六千人排成大队，战士身下的烈马不安分的刨着蹄子。中间一杆大旗，上书一个鲜艳的火字。
<br/>　　两军未曾交锋，空气中已经有浓烈的血腥味，就连从极北之地卷来的风都刮不走。
<br/>　　“大王。”有人将兵器送到金问道面前。
<br/>　　一杆铁枪，这金问道本也是一位用枪的好手，只不过，他也不记得有多少年没握过这柄枪了。
<br/>　　他挂上佩剑，拿住铁枪，翻身跨上自己的黑鬃马。自从青牛山之后，倒也不是没上过战场。只不过这一次，平白心中有些激动。
<br/>　　无论如何，噩梦总有结束的时候。今天要么是结束，要么是一个新的开始。说不得过了今天，自己就可以从尸山血海中挣脱出来。
<br/>　　北风猎猎，金问道跨马走在队伍最前方，身后上万北蛮部儿郎发出阵阵吼声。
<br/>　　毕竟是草原儿郎，既然大军压境，也绝没有怕的道理。无非是挺身一战，最多将这条命舍了出去。
<br/>　　两军相互对峙，各自发出阵阵吼声，一时间士气如虹。
<br/>　　钟牧山出现在两军阵前，第一次参与这种场合，他心中也是有些紧张。深吸一口气，冲着对面喊道：“我家大王也是草原的子孙，不忍同胞弟兄战死沙场。愿与阁下单对单较量，以胜负论成败。”
<br/>　　面对对方的挑衅，北蛮部的战士气得咬牙切齿，一员提大棒的汉子吼道：“大王莫慌，我去会他，取他的脑袋过来。”
<br/>　　多年未见，金问道也想看看，自己这位弟弟，实力到了什么地步。回到草原没多长时间，便搞出这么大动静。
<br/>　　这名请战的汉子，也是部落中有数的好手，却也可以拿他，试一试今天这野原火的深浅。
<br/>　　“好，许你出战，许胜不许败，莫坠了我北蛮部的威风。”
<br/>　　唿律律一声马鸣，这员名叫赤狼的汉子，拎着大棒出阵。对方的主帅野原火亲自出战，他手中一柄大刀，两匹马捉对冲锋。
<br/>　　各自都是来势汹汹，裹夹着肺腑中的杀意。在两马对头的光景，野原火扬起了刀，金问道眯眼看着。
<br/>　　对于他，金问道也并不是一点了解都没有。曾经的他用一柄弯刀，武艺也并不如何。只是不知何时，他从何地学会了这大刀的用法。
<br/>　　两马相错，只是一个刹那的光景，野原火的大刀狠狠挥下，咔嚓一声脆响，赤狼的人头滚落在地上，只有一具无头的尸体还骑在马上狂奔。
<br/>　　野原火一方，发出震天的吼声，士气大阵。如这样的两军对垒，大将捉对单挑也并不是完全没有的事情。获胜一方，可以提升己方士气，击溃敌人军心。当然，战场之上也不缺乏无有缚鸡之力的主帅，谈笑间杀敌于千里之外。
<br/>　　赤狼被斩之后，北蛮部一片静默，他也算部落中有数的好手，却在对方刀下连一个回合都没走过去。
<br/>　　金问道表情冷漠，心中却翻江倒海，只有一个念头不停重复：原来他已这般了得。
<br/>　　野原火拨转马头回来，再次稳住身形，冲对方战阵喝道：“谁来。”
<br/>　　北蛮部不缺不怕死的儿郎，更不缺觉得自己可以击败敌人的勇士。赤狼死后，连续又有几人出阵挑战野原火。在赤狼之后，野原火连败四人，杀机越来越旺盛，若烧起来的燎原大火，已经没有办法再扑灭。
<br/>　　最后，野原火的刀指向了金问道，口中冷冷喝道：“这次你来。”
<br/>　　金问道明白，迟早要轮到自己，事情总有个结束的时候。他拍马出阵，立在野原火对面。
<br/>　　时隔经年，许多事情都已改变，二人各自也有了许多变化。阴差阳错，金问道成了北蛮部的王，而野原火背后经历了什么，却是无人知晓。
<br/>　　“你活着，我很高兴。”金问道长叹一声：“我们也算是兄弟，能死在你的刀下，我不委屈。”
<br/>　　金问道和野原火的确算是兄弟，只不过，当初金问道不会把野原火当兄弟，今天，野原火没打算把金问道当兄弟。
<br/>　　野原火的目光很冷，他只是抬起刀，道：“你来。”
<br/>　　当触及到这目光时，金问道便知道自己已经败了。他还被往事纠缠着，当初人从尸山血海中走了出来，魂却留在了那座山谷中。但野原火不一样，他已经将那噩梦般的记忆嚼烂，化作身体最强壮的部分。从他走出山谷的那一刻，便已知道自己要做什么。
<br/>　　而有些事情一旦决定了，就再无法更改。
<br/>　　“来。”
<br/>　　金问道仰天长啸一声，扑向野原火。
<br/>　　草原上的儿郎，的确没有怕的道理。
<br/>(本章完)
