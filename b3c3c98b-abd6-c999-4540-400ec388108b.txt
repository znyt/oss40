        &lt;h3&gt;第三卷第一百零一章  西城分公司纪实&lt;/h3&gt;
<br/>
<br/>        陈子霖、呼延寰宇、李凌菲等西城的几个高层开了视频会议，就各自准备开始交接和行程。
<br/>
<br/>        李凌菲没有想到公司直接就调她到北京，心里也有些小惊喜。
<br/>
<br/>        “凌菲，你单身，去哪里都方便着，陈总调你过去肯定是看重你的能力，她是一个爱才之人，放心过去吧。”呼延寰宇对李凌菲说。
<br/>
<br/>        “嗯，我就怕新的工作环境有点不适应。”其实李凌菲还是很高兴的，多少人想去北京啊。
<br/>
<br/>        “没事，公司的同事都是很好相处，你过去是负责运营的，而且是新成立的部门，好好干你的工作就好。”呼延寰宇安慰李凌菲。
<br/>
<br/>        “你在西城也呆不了多久呢。”李凌菲笑着说。
<br/>
<br/>        “嗯，我可能要年后会跟着陈总一起甸城吧，其实我也很向往去一个新的工作环境，调整一下状态，要不然会温水煮青蛙的。”
<br/>
<br/>        “你对你目前的工作还不满意啊，还要挑战？”
<br/>
<br/>        “新的起点更能体现自己的价值。”呼延寰宇豪气的说，作为程序员，目前搬砖码字写程序他很快乐，在这个公司体现了自己的价值。
<br/>
<br/>        其实吧，想一想如果说为社会做贡献，添砖加瓦不在乎那个行业，梦想是一个人的，但也是大家的，只要大家奔着一个富国强民的目的，拧成一股绳，梦想就会实现。
<br/>
<br/>        “聊啥呢？你俩开心的。”陈子霖走过来了。
<br/>
<br/>        “陈总，呼延说他要重新奋斗。”
<br/>
<br/>        “重新奋斗？”陈子霖还没有弄明白。
<br/>
<br/>        “羡慕你换一个工作岗位。”呼延寰宇说。
<br/>
<br/>        “其实我还纳闷呢，为什么要让我去北京？我才去了三次北京，对北京的情况都不熟悉。应该让你过去。”陈子霖闷闷的说。
<br/>
<br/>        “我是要和陈总一起去甸城的。”呼延寰宇翻了一眼陈子霖。
<br/>
<br/>        “还是你受重用。”陈子霖酸酸的说，他都求了好多次陈子昂，要陈子昂去甸城的时候带上他，陈子昂从来都没有答应过。
<br/>
<br/>        “我咱感觉满屋子的酸味呢？”李凌菲捂住嘴偷笑。
<br/>
<br/>        “有吗，有吗？我怎么没有闻到。”陈子霖吸了吸鼻子对李凌菲“净瞎说。”
<br/>
<br/>        “呼延你下午和我去一趟集采中心，见一下乔总。”陈子霖是来安排任务的。
<br/>
<br/>        “好的。”
<br/>
<br/>        陈子霖算是带着他们去和乔总告个别，顺便将公司的状况和乔总说一下，还要带着另外的一个人过去介绍一下。
<br/>
<br/>        谭九儿和李凌菲搭档了很久，李凌菲调到北京，和客户商务对接的只有她了，她正在收拾东西，陈子霖和呼延及李凌菲一起走出来。
<br/>
<br/>        “九儿，收拾好了吗？”陈子霖问谭九儿。
<br/>
<br/>        “收拾好了。”谭九儿脆生生的回答。
<br/>
<br/>        “收拾好了我们就走吧。”陈子霖叫上呼延和李凌菲。
<br/>
<br/>        呼延寰宇开着车，四个人一起到了乔小东的办公室，但是乔小东开会去了，办公室里面的小姑娘招呼他们坐下，一人倒了一杯水，便忙自己的事情去了，等了大概有半个小时，乔小东才开会结束。
<br/>
<br/>        “咦，这是怎么回事？集体出动？”乔小东都认识他们几个人。
<br/>
<br/>        “乔总，我妹妹调我和李凌菲去北京了，这里后期是江俞轩和谭九儿负责，九儿以后主要负责集采的了。”陈子霖笑着介绍。
<br/>
<br/>        “嗯，小姑娘也很能干的。”乔小东也认识谭九儿的，经常来他们这里，那脆生生的声音像黄鹂一样。
<br/>
<br/>        “怎么了，这是要全部去北京发展了？”乔小东问陈子霖。
<br/>
<br/>        “是这样的，那边耗材已经准备投向市场了，让我们先过去帮着点，后期还是要回来的。”陈子霖后期肯定是要回西城的，老婆孩子都在这边。
<br/>
<br/>        “寰宇不动吧？”乔小东看着呼延寰宇。
<br/>
<br/>        “我目前不动，但是明年也会去外地。”呼延寰宇和乔小东也是老朋友了，他们同在西城，以前知名点的人都认识，何况现在呢。
<br/>
<br/>        “你们公司发展得是越来越大了啊。”乔小东赞许的点点头，陈子霖他们的发展他可是看着的。
<br/>
<br/>        “一般一般。”陈子霖谦虚的说。
<br/>
<br/>        “江俞轩负责西城分公司？”乔小东接着问陈子霖。
<br/>
<br/>        “是的。”一问一答，像是例行公事又像是老朋友间的日常。
<br/>
<br/>        “我还一直以为江俞轩只是你们公司的，原来还是江家大少。你们这公司还真是卧虎藏龙啊。”乔小东就是吃饭的那一次才真正的知道江俞轩的身份。
<br/>
<br/>        “江俞轩是我们公司的股东，我们公司还有几个一层着的大股东都是厉害人物。”陈子霖笑着说，那种骄傲溢于言表。
<br/>
<br/>        “所以说，你们公司卧虎藏龙嘛。对了等你们的耗材推向市场了，可以参加集采。”乔小东说，明年的工作已经在部署了。
<br/>
<br/>        其实大家等的就是乔小东的这句话，想了解一下后期的走向。
<br/>
<br/>        “乔总，后期还望你多多提携，九儿年龄小，如果有什么不到之处，请您以后多多指导。”谭九儿接话。
<br/>
<br/>        “你现在流程什么的都熟悉了，前几次的工作也做很好嘛！“乔小东赞许的看着谭九儿，别看这姑娘年龄小，但是做事情很沉稳。
<br/>
<br/>        .........
<br/>
<br/>        陈子霖也是要和江俞轩进行工作交接的，虽然江俞轩接了陈子霖的工作，但是江俞轩只是兼顾，后期还是要同事们担起责任的。
<br/>
<br/>        陈子霖将刘清影、李秀叫到办公室。
<br/>
<br/>        “清影，我去北京之后，江总会来接手我的工作，但是江总只能是兼顾，公司的事情就得你操心了。”
<br/>
<br/>        原本刘清影就是做管理的。只是他走之后，刘清影就要更辛苦一些，毕竟江俞轩还得管理江家的事情。
<br/>
<br/>        刘清影笑着说：”再辛苦也是给自己做事情，给自己做事情就不要说辛苦不辛苦的的了。“
<br/>
<br/>        ”李秀，你继续你的公关啊，这几年公司要是没有你们公关部也不会这么生机勃勃。“陈子霖话里充满了感激。
<br/>
<br/>        这些人都是和他一起进的公司，他对他们对这个公司充满了感情，突然调走，心里竟然万分不舍。
<br/>
<br/>        “放心吧，有你没有你我都会干好工作的，刘姐说的对，咱是给咱自己干事情呢。”李秀其实也有点不舍陈子霖走，他们搭档得很好，一唱一和，这么多年了还没有分开过呢。
<br/>
<br/>        “我和凌菲一走，你就得和九儿俩个搭档了，九儿年轻，你得指导着她，多带带，要多点耐心。”陈子霖就像是一个大哥哥，絮絮叨叨的叮咛自己的弟弟妹妹。
<br/>
<br/>        “好，一切都听你的，九儿虽然年轻，但是九儿做事情很稳妥，你放一百二十个心吧。”李秀为了让陈子霖安心便出言安慰。
<br/>
<br/>        “不，九儿活泼爱贪玩，又爱说笑，有些正式的场合你还是得看着她。”陈子霖对公司每个人的个性都摸得透透的。
<br/>
<br/>        “陈总，你又背着我对李秀哥说我的坏话。”谭九儿刚好要来找李秀。
<br/>
<br/>        “我哪有说你坏话啊？我不是让你李秀哥多带带你吗？”
<br/>
<br/>        “李秀哥，是真的吗？”谭九儿问，按照公司的职位，谭九儿应该叫李秀一声“李总“，可是谭九儿儿自从见到李秀之后就一直”哥哥、哥哥“的叫着。
<br/>
<br/>        “是真的。”李秀看了一眼谭九儿，谭九儿那双细长的秀气的眼睛调皮的盯着他，让他一阵阵的心慌。
<br/>
<br/>        每次见到这个小妮子都有这种心慌慌的感觉。
<br/>
<br/>        “李秀哥，你看看这个事情怎么办？”谭九儿直接挤开陈子霖站在了李秀的身边，拿出文件，俩个人头挨着头开始讨论工作了。
<br/>
<br/>        陈子霖突然就觉得他是多余的了。
