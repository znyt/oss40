　　第二步成为人仙，体内灵气转化为仙元，人仙之后为地仙，地仙者，阴神化为阳神，仙元练成仙丹，永镇下田，炼形往世而得长生不死，以作陆地神仙。
<br/>　　地仙之后为天仙，天仙者，胸中五气齐聚，“五气”，通常便是人体之五脏，心、肝、脾、肺、肾，修炼五脏的五气为金、木、水、火、土五行之气，直至体中五行圆满不缺。
<br/>　　天仙之后为玄仙，玄仙者，精、气、神具现，化为顶上三花，有着玄之又玄之意。
<br/>　　玄仙之后为金仙，金仙者，金者，不朽也。金仙，顶上三花齐放，凝聚无上庆云。金仙的修为，就要开始悟道，有句修道名言，观天之道，执天之行，说道就是金仙以后的修行。金仙，跳出三界外，不再五行中。只要不作死，寿命无穷。大罗金仙、混元大罗金仙本质也是金仙。
<br/>　　金仙之后为太乙金仙，太乙者，逆反先天，开始从所修之道中走出自己的道，以自己的道凝聚道果。当然，也可以不凝道果，直接进入大罗，那么，大罗金仙也就是你之极限，一生不能再有进步。
<br/>　　太乙之后为大罗金仙，大罗金仙，证得大罗果位，法力无穷，神通自现，种种不可思议的玄妙呈现。
<br/>　　大罗之后为混元大罗金仙、圣人（圣人为混元大罗金仙，但混元大罗金仙不为圣人），跳脱命运长河，不死不灭。
<br/>　　或许，在之后还要多出一个境界，那就是准圣，准圣介于大罗金仙与混元大罗金仙之间，实质还是大罗金仙，只是更进一步而已。
<br/>　　修道本就是逆天而行。这个逆天不是真正的逆天道，或者天道意识这种人要么身死道消，要么只是凡人喊喊而已。天道不会怪罪于一个什么都不懂的凡人，只有真正的大神通者，才知道天道有多强。
<br/>　　修行所谓的逆天指的是逆着天地演化而修行，先聚灵气，再返后天为先天，再聚仙元、五行返玄黄，玄黄逆混沌。
<br/>　　所以，元始夺舍混沌魔神，生而具有混元大罗金仙是多么的幸运，只因那时他一出生吃的就是混沌之气。。
<br/>　　元始现在就是需要修炼元神，阴神化阳，化而为人体，之后胸藏五气，汇聚精、气、神，顶开三花，之后找到自己的道，凝聚道果，再化先天道体，参悟法则，证道不朽。
<br/>　　现在元始身为元神清气，需要整理一部修炼元神的功法，这个简单，当初身为混沌魔神之时，修炼的也是元神，只不过，那时是以大道法则锻炼元神，现在以自己弱小的元神，哪能经得起法则的锻炼。

<br/>　　所以，如今元始需要自己参悟一步功法，这个简单，一元始的眼界与领悟，就算如今弱小，也能创出一步功法，总好过这样慢慢炼化灵气来壮大元神。
<br/>　　整理之中，元始根据自己的想法与经验见识，历经数万载，终创一专修元神之功法，名为《神元九转》。

<br/>　　神元第四转融术，可将自己最强大的术法融入识海之中，不单单只是神识化刃，别的五行系风雷冰系等任何强大术法神通一样可以融入识海，都可瞬发攻击……
<br/>　　一旦修炼到第四转，简直可以大罗金仙修为斩杀准圣强者，至于神元第五转界成，更是可以形成自己的识海世界。
