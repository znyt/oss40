        现在，他还是为了筱颖，才答应爷爷结婚。
<br/>
<br/>        但他会弃商去当兵。
<br/>
<br/>        等婚礼过后，傅夜沉决定跟爷爷傅炎廷坦白，让弟弟傅昼景带着筱颖和孩子留在滨江城里长住。
<br/>
<br/>        然后，他会把傅氏集团的总裁之位让给弟弟傅昼景，自己回归部队继续去当兵。
<br/>
<br/>        傅氏集团现在的“版图”很大，他会在走之前，把所有的事情都安排妥当，让阿景顺顺利利地上位，无后顾之忧。
<br/>
<br/>        既然筱颖给阿景生了个儿子，那么等阿景将来退休了以后，筱颖的儿子还可以继任傅氏集团。
<br/>
<br/>        这样一来，筱颖以后的日子一定会过得很幸福。
<br/>
<br/>        筱颖会是个好妻子、好妈妈，将来也会是个好婆婆、好奶奶……
<br/>
<br/>        这次，他想去边境，带着对筱颖的爱去，然后离他们越远越好，一个人孤独终老。
<br/>
<br/>        而他左腕上的这条青丝手链，只怕会跟随他一辈子，直到入土为安。
<br/>
<br/>        翌日。
<br/>
<br/>        傅氏集团大厦。
<br/>
<br/>        此刻时间，九点半。
<br/>
<br/>        傅夜沉已经看完了十份合同，眼瞧着快要到开会的时间了，而他的“万能”秘书，却到现在还未到办公室。
<br/>
<br/>        而且，落以柔今天也未打电话给他，告知她要休假。
<br/>
<br/>        傅夜沉本想着无所谓，但是当他想自己解决一些事情时，才发现有些资料在落以柔手中，于是他拿出自己的手机，马上按下数字“2”的快速拨号键。
<br/>
<br/>        数字“1”是助理郑忠威的，现在郑忠威还在请长假中，为了方便联系落以柔，他把落以柔的手机号码设置成了数字“2”的快速拨号键。
<br/>
<br/>        电话通了，落以柔也接了。
<br/>
<br/>        但是，傅夜沉却很不客气地质问：“你在哪里？”
<br/>
<br/>        “大、大哥……”落以柔的声音听起来有些气若游丝。
<br/>
<br/>        傅夜沉误以为她还没睡醒，瞬间不满地反问：“知道现在几点钟了吗？”
<br/>
<br/>        “现在？现在几点呢？”落以柔迷迷糊糊地说。
<br/>
<br/>        “九点半。”
<br/>
<br/>        “九、九点半？”落以柔虽然如梦初醒，但整个人仍旧没什么力气，声音虚弱地说道，“对不起啊！大哥！我马上就过来上班！”
<br/>
<br/>        “记得把工作U盘带过来！”傅夜沉命令道，说完后，直接将电话给挂了。
<br/>
<br/>        落以柔连忙从床上爬了起来，以最快的速度换衣洗漱，收拾好手提包，带上了傅夜沉要的那个工作U盘，立马出了门。
<br/>
<br/>        她火急火燎地赶到总裁办里时，傅夜沉已经开会去了。
<br/>
<br/>        落以柔下意识地拍了拍自己的额头，才发现自己的额头烫得很厉害。
<br/>
<br/>        是发烧了吗？
<br/>
<br/>        但是现在，她不是顾及自己的时候，她得赶在大哥下会前，把大哥要的资料全部整理出来。
<br/>
<br/>        她昨天大白天得陪着那个李忆莎去安排婚礼上的事情，到了晚上才去整理大哥所要的资料。
<br/>
<br/>        其实，她昨晚已经快整理完了，只是突然间觉得很犯困，就迷迷糊糊地爬床上去睡了，而这一睡，直接睡到了大哥打电话把她给叫醒了才起床。
<br/>
<br/>        为了不让自己跑上跑下，落以柔直接坐在了傅夜沉的总裁办里的辅办公桌上工作。
<br/>
<br/>        虽然坐在办公桌前，看着笔记本电脑屏幕上word文档里那些密密麻麻的字，让她感到头晕脑胀得厉害，但是她仍旧不允许自己有一丝马虎，出一丁点的错误。
<br/>
<br/>        落以柔睁大了眼睛，强迫自己全神贯注地看着电脑屏幕，十指快速地敲击着电脑键盘。
<br/>
<br/>        资料全部整理完后，落以柔将它们全部打印了出来，并装订好了，才松口气。
<br/>
<br/>        希望大哥不要生气就好！
<br/>
<br/>        然而，就在她将资料放到傅夜沉的办公桌上后转过身去时，眼前忽然一阵天旋地转，头疼得厉害，直到发黑眼晕后，她整个人开始摇摇晃晃。
<br/>
<br/>        落以柔有些体力不支，视野从模糊再到一片漆黑，她直接晕倒在地。
<br/>
<br/>        不知过了多久，电梯门开了。
<br/>
<br/>        傅夜沉和两个西装革履的中年男子从电梯里走了出来。
<br/>
<br/>        “这里是我的办公室，两位有请。”傅夜沉礼貌性地迎了迎手。
<br/>
<br/>        当他带着身后的两位商业大佬走进自己办公室时，看到自己的办公桌边的地上躺着一个女人，连忙疾步走了过去。
<br/>
<br/>        小落？
<br/>
<br/>        傅夜沉急忙蹲下身去，将落以柔拉了起来，让她的上半身靠在了自己怀中。
<br/>
<br/>        他发觉她的手腕很烫后，又下意识地摸了摸她的额头。
<br/>
<br/>        简直烫得吓人！
<br/>
<br/>        一旁的中年男子见状，好心催促道：“傅总，你赶紧送这位小姐去医院吧！”
<br/>
<br/>        “是啊！别耽误了，救人要紧！”另一个中年男子也跟着附和了一声。
