　　“什么叫没有我的世界？”叶少阳惊问。
<br/>　　“你死了，魂飞魄散，精魄聚生之后，那就不再是你，因为你是全新的，没了过去的记忆，也没了过去的因果缘分，懂了没有？”
<br/>　　叶少阳似懂非懂地点点头。
<br/>　　“我能找到你的时间轴，所以，我能去到你的未来，将东西给你，这么看起来，也算是改变了你的未来。”
<br/>　　“这么说，未来和过去都可以改变，那如果我回到自己时间轴的起点，把我自己给杀了……会发生什么？”
<br/>　　徐福笑起来，“这是个悖论，我没有尝试过，也不敢尝试。但我可以告诉你，你如果改变过去，那么根据时间规则，会衍生出很多时间轴，至于你的那条有没有被改变，这个一看运气，二看你改变了多少，这种改变能够持续多久，在时间长河中，哪怕崔天子的生死簿，也只能定人的生死。”
<br/>　　他的目光落在叶少阳脸上，郑重而又感慨地说道：“你在做一件事的时候，可能努力了十分，但只出现一个意外，你可能就前功尽弃，如果放大到一年、十年、一生，你会发现，你的一生中充满了意外，就是这些意外推动着你走到最终的命运，如果时间跨度更长，从时间的长河中去整个历史，你会发现，一切都是意外，意外碰上意外，有时候就足以改变历史。
<br/>　　千万不要试图对过去做什么，否则事情的结果一定不是你想的那样，打折扣都算是好的，就怕结果南辕北辙，而到了那一步，你再试图去补救，只会越来越糟，一发不可收拾！”
<br/>　　叶少阳点头，虽然没经历过，但他能理解徐福说的道理。“我知道了。”
<br/>　　“不是要你知道，是要你记住！”徐福的声音严厉起来，“轻易不要改变过去！这是第一条规则。”
<br/>　　叶少阳看他说的郑重，用力点了几下头，“第二呢？”
<br/>　　“不要改变未来，一丝一毫都不行，甚至，不要去窥探未来！”
<br/>　　“哦，我记得你说过，人就是活个未知，我赞同你的观点，未知，才有奋斗的乐趣，否则知道结果，不是失望就是索然无味。”
<br/>　　“这只是其一，你如果窥见未来，结果是你想要的，你会忍不住生出懈怠，如不是你想要的，你会失望，并且试图去改变它，这就回到了我们之前说的，你改变的目的是好的，但结果未必能达到，一个意外，就会把事情推向不可掌控的方向，退一步说，就算你达到目的，但因为这个改变，也会影响到别人的命运，造成难以收拾的混乱。还有一点，生死不可逆。生死簿上判你今天死于车祸，你就算避开，一样会死在别的地方。”
<br/>　　“这个我知道。还有吗？”
<br/>　　“还有最重要一点，决定不能见到你自己！不管是未来的你还是过去的你！”
<br/>　　“会发生什么事？”
<br/>　　“会引起两个时间轴的碰撞，很可能两个世界都灰飞烟灭。”
<br/>　　叶少阳深吸一口气，“两个世界的灰飞烟灭，别人也跟着没有了？”
<br/>　　“别的时间轴，一切照旧，但你们两个各自的时间轴会毁灭。”
<br/>　　“你……怎么知道这么清楚？”
<br/>　　叶少阳犹豫了一下，还是问出了这个问题。
<br/>　　徐福淡淡地看了他一眼，“我知道你想问什么，是的，这种事曾发生过，所以……对你来说，我不是你这条时间轴上的徐福。”
<br/>　　叶少阳暗暗吸气，感到了一丝恐惧。
<br/>　　“这件事，你不需要知道详情，但你切不可重复我所做的一切，这三条规则，你都记住了。”
<br/>　　轻易不要改变过去；绝对不能改变未来、不能穿越未来；不能跟自己碰面。
<br/>　　叶少阳点头苦笑道：“什么都不能干，那我要这山海印干什么？”
<br/>　　“它掌控了时间的终极，游离在三界规则之外，但干系也重大，绝不能轻用，但有些事还是可以做的，例如我上次将你们带到民国去避难，避开追杀，只要不做过分的事，你们在过去所做的事都会淹没在时间长河中，被无数个偶然修正。当然，你们那一次做了很多改变，所以衍生出了很多时间轴，幸好你们都没改变过去的自己，所以这条时间轴没受到大的影响。”
<br/>　　他这番话，倒是让叶少阳想起了一些事情，皱眉道：“你上次不是告诉我，发动山海印，耗费的是上面的符文的灵力吗，说什么只有大帝能灌注灵力，又说什么只能用最后一次，怎么后来又用上了，难道你是诓我的？”
<br/>　　徐福笑道：“我只骗了你一点，山海印需要耗费的灵力，不是大帝灌注的，山海印本是天地造化自生，大帝本人也在规则之内，他怎能管规则之外的事？这山海印，积累千万年的灵力，没用一次，灵力就会少一些，吸收灵力却极为缓慢，需要上前年，才够使用一次，因此，一只山海印的灵力，只够用十次左右……但只要你动脑子，那想用上千万次都可以。”
<br/>　　说完望着叶少阳，意思让他思考。
<br/>　　要在平时，叶少阳最烦的就是别人说话卖关子，感觉浪费时间，但是想到这里的时间是永恒的，也不存在浪费，干脆就开动脑经思考起来。
<br/>　　徐福笑咪咪地在一旁等着。
<br/>　　猛然间，叶少阳想到了什么，倒吸一口冷气：“你……夺了别的时间轴的山海印？”
<br/>　　他想到的是，既然一个人的时间轴有无数个，每一个世界之中，又都有一个山海印，别的东西可能拿不到另外的世界去用，但山海印本来就是突破一切规则的，徐福在用光了这个世界的山海印的灵力之后，完全可以穿越到别的时间轴……例如回到几千年前，在山海印没被那个世界的徐福得到之前，把它夺走，拿来自己用……这样就有无数只山海印可以用了。
<br/>　　徐福笑道：“我就知道你会想到这个注意，不错，这个主意我以前也想过，但却难以实施。”
