夏妍妍瞳仁一缩，这道声音是如此的熟悉，是…夏夕绾！
<br/>
<br/>        夏妍妍迅速抬头看向前方，夏夕绾今天穿了一件枚红色的灯笼袖毛衣，下面黑色紧身小脚裤，整个人纤尘清丽的从枢密研究院的大门走了进来。
<br/>
<br/>        枚红色对于女孩而言很难驾驭，但是驾驭住了就像夏夕绾这样在水灵灵的清纯里透出几许动人的明艳，让人移不开目光。
<br/>
<br/>        夏夕绾今天的穿着很适合刚入职场的小白。
<br/>
<br/>        夏夕绾来了。
<br/>
<br/>        她如约而至。
<br/>
<br/>        “绾绾，你怎么来了，你怎么进来的，我可听说枢密研究院并没有录取你。”夏妍妍柔声道。
<br/>
<br/>        夏夕绾跨了一个小包，她勾起红唇，“夏妍妍，你听谁说的我没有被录取？”
<br/>
<br/>        夏妍妍看了赵立英一眼。
<br/>
<br/>        赵立英迅速上前，鄙夷厌恶的看着夏夕绾，“夏夕绾，我们枢密研究院怎么可能录取你这样一个才高中毕业的人呢，你快点离开这里，如果你再不离开的话我就叫保安将你赶出去了！”
<br/>
<br/>        整个网络和海城名媛圈都在压赌注，纵然枢密是圣地，但是这里的动静早就吸引了很多目光，大家都偷偷的拿出手机来拍了。
<br/>
<br/>        夏妍妍眸里闪过了一丝恶毒和幸灾乐祸的光芒，夏夕绾，是你主动送上门来的！
<br/>
<br/>        夏妍妍走过去拉住了夏夕绾的小手，“绾绾，我知道你不喜欢我，对我有误解，但是你不该为了争一口气而私自闯入枢密，这样会打扰到很多人，现在我带你出去好不好？”
<br/>
<br/>        赵立英觉得他的妍妍女神真是太善良了，人间小天使，他当即打抱不平道，“妍妍，这个夏夕绾一直在针对你，还将你家给闹得家破人亡了，你不要这么心善，不要再对夏夕绾好了！”
<br/>
<br/>        夏妍妍柔美的看着赵立英，委屈和可怜，欲说还休，“赵组长，请你不要这样说绾绾。”
<br/>
<br/>        夏夕绾好笑的看着这两个人，“夏妍妍，你魅力真大啊，刚进枢密又多了一个爱慕者了？”
<br/>
<br/>        “绾绾，你…”
<br/>
<br/>        夏妍妍想说话，但是这时枢密研究院的主任周萍走了过来，周萍已经三十五岁了，常年一身黑，为人刻板又严厉，就像是学校里的教导主任，谁都怕她。
<br/>
<br/>        周萍走过来，冷声训斥道，“你们三个人在这里闹什么呢，赵立英，你可是组长，这就是你的带头作用？”
<br/>
<br/>        赵立英迅速道，“周主任，我正要跟你反应这件事，这个夏夕绾贸然闯入了我们枢密研究院，严重打扰了我们的工作，我正想将她给轰出去！”
<br/>
<br/>        夏妍妍觉得这一次夏夕绾真是死定了，谁落在周萍的手里都不会有好下场的！
<br/>
<br/>        刚才已经很多人拍了视频，如果夏夕绾再在周萍手里落下难堪的话，那舆论就精彩了。
<br/>
<br/>        这一次夏夕绾必然会输得很惨！
