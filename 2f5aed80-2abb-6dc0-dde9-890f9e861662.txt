　　眼看燕狂生就要再次对叶尘出手，就在这个时候，
<br/>　　唰！
<br/>　　一道人影忽然拦在了他的面前，赫然正是龙辰。
<br/>　　“燕师弟，你不是叶兄弟的对手，还是就此罢手吧！若你真的想要那位置，我把我的位置让给你，如何？”
<br/>　　燕狂生听到这话，顿时微微一愣，旋即又望了一眼远处，叶尘头顶上空，那道充满毁灭气息的刀芒，最终重重叹了一口气，
<br/>　　“罢了！我认输就是！”
<br/>　　说完这话，燕狂生身体一晃，直接飞身到那峭壁中央，在那十处高台中，随意挑选了一个位置，安坐在其中。
<br/>　　“呵呵，叶兄弟，你还真是让我刮目相看呐！”
<br/>　　三人回到高台上，龙辰冲着叶尘笑呵呵的开口道，眼眸深处，却闪烁着奇异的目光。
<br/>　　叶尘淡淡一笑，知道龙辰对自己的底细，已经有所怀疑，也没有多说什么。
<br/>　　又过了一阵，远处再次传来一道巨响，显然已经又来了新的闯关者。
<br/>　　不一会，那道人影来到众人面前，赫然正是之前，在第四重星域，和叶尘交过手的云破天！
<br/>　　云破天看到龙辰和洛冰玄，并没有感到意外，甚至见到排名在他之下的燕狂生，也觉得在情理之中，毕竟他之前被叶尘打成重伤，因此耽搁了不少时间，被燕狂生超越，自然也在情理之中。
<br/>　　然而，等他看到叶尘，整个人顿时就不好了，
<br/>　　“他居然比我先到了！这怎么可能？”
<br/>　　虽然之前在第四重星域，云破天曾被叶尘击败，可他并不认为自己就比叶尘要差。
<br/>　　在他看来，叶尘不过就是运气好，修为恢复的比他快，利用境界上的压制，所以才能击败他。
<br/>　　可当时，明明是他先离开了第四重星域，叶尘还带着一个身受重伤的洛冰玄，竟然还可以先他一步赶到第七重星域！
<br/>　　这顿时让云破天，感到了极大的威胁！
<br/>　　如果说燕狂生可以超过他，他还勉强可以接受，叶尘不过就是一个名不见经传的无名小子，只有区区元婴境的修为，居然也压他一头，凭什么！
<br/>　　莫非是洛冰玄出手帮了他？
<br/>　　对！一定是这样！否则以他的实力，怎么可能会这么快到达第七重星域？
<br/>　　云破天越想越不忿，最终身体一晃，便来到叶尘所在的高台之前，
<br/>　　“小子！这个位置，你还没有资格享用！滚到下面去吧！”
<br/>　　口气和之前的燕狂生，如出一辙。
<br/>　　在云破天看来，之前在第四重星域，他之所以会败给叶尘，那是因为自己的修为恢复的不够，在境界上被叶尘给压制了，可是现在到了第七重星域，大家的修为，都是元婴境，他又有什么好惧的？
<br/>　　叶尘望着眼前趾高气扬的云破天，顿时笑了，不过却是冷笑。
<br/>　　“你想坐这个位子，那就要看看你没有这个本事了！”
<br/>　　之前对于燕狂生，叶尘多少还是留了几分力的，但对于眼前这个云破天，叶尘可不打算丝毫留手，既然他自己撞到枪口上，直接将杀掉或者赶出迷罗星海最好！
<br/>　　就在叶尘准备起身动手的时候，一旁的洛冰玄却抢先一步站了出来，
<br/>　　“云破天！之前在第四星域，你先是骗我和你一起猎杀冰甲猛犸象，然后却过河拆桥，偷袭于我，这笔账，我们是不是也该好好算算了？”
<br/>　　声音落下的同时，洛冰玄已经直接悍然出手了！
<br/>　　轰！
<br/>　　洛冰玄双指作剑，直接一指刺出，一道冰蓝色的璀璨剑芒，瞬间便朝云破天爆射而去！
<br/>　　云破天显然也没有想到，洛冰玄居然说出手就出手，顿时瞳孔微微一缩，当即身形迅速暴退，足足退到数千米之外，方才避开洛冰玄的那道凌厉剑气。
<br/>　　然而，洛冰玄得势不饶人，傲人的娇躯，在空中一阵闪烁之后，瞬间便追了上去，
<br/>　　轰轰轰！
<br/>　　一连几道剑气攻出，云破天顿时有些手忙脚乱。
<br/>　　他的实力，原本就不及洛冰玄，否则也不会屈居第三的位置，如今又被洛冰玄抢了先手，自然更加不敌，转眼之间，便已经落了下风。
<br/>　　不过这云破天，到底也是不世出的天才人物，战斗经验也极为丰富，虽然被洛冰玄抢占了先机，但是一番谨防死守之下，倒也渐渐站稳了脚跟，勉强应对了下来。
<br/>　　洛冰玄想要彻底击败云破天，显然也不是一件容易的事情。
<br/>　　“奇怪！真是奇怪！”
<br/>　　远处观战的龙辰，忽然摇了摇头，一阵叹息道。
<br/>　　叶尘大为好奇，忍不住问道：
<br/>　　“什么奇怪？”
<br/>　　龙辰望着远处战斗中的两人，不由得再次摇了摇头，缓缓道：
<br/>　　“冰玄仙子修炼的乃是绝情之道，剑法一向以绝情、冷酷而著称！”
<br/>　　“说实话，即便是我，都不愿意轻易和她交手，因为她的剑意之中，不掺杂一丝一毫的感情，几乎可以说是六亲不认！”
<br/>　　“可是现在，冰玄仙子的剑法，似乎不及之前那般凌厉了，多了一分牵挂和迟疑……”
<br/>　　“若非如此，那云破天此时，恐怕早就已经败了！”
<br/>　　龙辰说完这话，不由得转过头，深深的望了叶尘一眼，似笑非笑，道：
<br/>　　“冰玄仙子有如此变化，该不会和叶兄弟有关吧？”
<br/>　　叶尘顿时一阵愕然，旋即脸色大变，口中不由得喃喃道：
<br/>　　“绝情之道……绝情之道……莫非修炼此道，不能动情？难道前世……”
<br/>　　记得前世，洛冰玄后来修为大退，被绝情居逐出了师门，后来彻底消失无踪，难道就是因为动了情不成？
<br/>　　只是不知道，她对谁动了情？
<br/>　　就在叶尘思绪纷飞之时，另外一边，两人的战斗，已经到了白热化的地步，虽然洛冰玄依旧可以占据上风，可是想要击败云破天，却有着不小的难度。
<br/>　　而云破天的心态，渐渐稳定下来之后，顿时忍不住哈哈大笑起来，
<br/>　　“洛冰玄！原来你也不过如此嘛！看来之前，我还真是高估你了！哈哈哈！”
<br/>　　听到云破天肆无忌惮的嘲讽，洛冰玄顿时秀眉紧蹙，脸色变得有些难看起来。
