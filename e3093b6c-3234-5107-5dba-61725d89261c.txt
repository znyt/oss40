　　时间回到数分钟前。
<br/>　　极寒裂缝内，深处。
<br/>　　一道身影，以极其惊人的速度，不断跌落。
<br/>　　忽然，身影之上，一道火光猛然爆发，一双火焰翅膀，凭空而现。
<br/>　　人影，自是萧逸。
<br/>　　“噗。”萧逸猛地吐出一口腥血，脸色苍白至极。
<br/>　　刚才寒冰殿主与冰承天的全力一掌，直直轰于他背上。
<br/>　　这两人的实力，比不夜宫主和雪刀门主强得多。
<br/>　　全力一掌下，其威力，强得难以想象。
<br/>　　这让得萧逸的跌落速度，自然也极其惊人。
<br/>　　萧逸也无法确定自己跌落了多少米。
<br/>　　只能勉强预估，绝对在万米以上。
<br/>　　这道裂缝，果真如传言那般，乃是一噬人深渊，深不见底。
<br/>　　哪怕萧逸已经跌落了如此深度，下方，仍旧是黑漆漆一片，看不到尽头。
<br/>　　“噗。”萧逸再度吐出一口腥血。
<br/>　　他毕竟只有地元九重修为，毫无防御地直接受两位极境强者的攻击，自是吃不消。
<br/>　　若非有寒冰铠甲护身，怕是现在早已重伤昏迷，直直落入这裂缝深渊的尽头。
<br/>　　以前，寒冰铠甲并未展现出太惊人的防御力。
<br/>　　毕竟以前的敌人，实力并不足以给萧逸带来太大的威胁。
<br/>　　反倒是最近，动辄遇到极境强者。
<br/>　　萧逸才真切感受到这件铠甲的强大防御力。
<br/>　　言归正传。
<br/>　　萧逸紫炎火翼一振，艰难地往上飞着。
<br/>　　周遭，是无穷无尽的凛冽罡风，吹袭而来。
<br/>　　极寒裂缝上头的那片风雪之地，本就是由裂缝中的罡风，吹袭而上所形成。
<br/>　　可想而知，裂缝中的罡风，达到何等骇人程度。
<br/>　　刚才的一直跌落，萧逸明显感觉到，越是跌落，深度越深，吹袭的罡风便越是可怕。
<br/>　　现在这个深度出现的罡风，怕是连不夜宫主那个层次的强者，都无法支撑太久。
<br/>　　萧逸之所以能抵挡，全靠紫炎火翼之威。
<br/>　　紫晶灵炎，焚毁万物之效，极寒罡风也抵挡不住。
<br/>　　当然，其代价是紫炎也在不断被抵消。
<br/>　　加之体内‘冰山火海’的惊人爆发力，才让得萧逸能挡下这些罡风。
<br/>　　再一次证明，自己之前没有急着突破，是正确之举。
<br/>　　若少了‘冰山火海’的爆发力，怕是无需半会儿，自己就会被这些罡风撕成碎片。
<br/>　　这里的罡风，尚且有如此威力。
<br/>　　那么，更深处呢。
<br/>　　萧逸深深地看了一眼下方，那如黑洞一般的深渊，让人发怵。
<br/>　　这条裂缝，到底有多深？
<br/>　　下方，隐藏着什么？
<br/>　　又凶险到何种地步？
<br/>　　萧逸不敢想象，也没那个时间想象。
<br/>　　虽现今勉强靠紫炎火翼往上飞跃。
<br/>　　但体内元力，同样在惊人地消耗着。
<br/>　　不能在此多留，必须快速脱身。
<br/>　　嘭…嘭…嘭…
<br/>　　不得不说，萧逸对于火焰的控制力，相当惊人。
<br/>　　紫炎火翼，一边抵消无尽罡风的吹袭，一边御风飞行。
<br/>　　速度虽慢，但萧逸有把握，在元力耗尽前，必能回到上方边缘。
<br/>　　但，当萧逸上升了百米高度左右后。
<br/>　　周遭的罡风威力，竟陡然上升。
<br/>　　“嗯？”萧逸脸色一变。
<br/>　　紫炎火翼，竟被周遭罡风吹得摇曳不停，呈溃散之状。
<br/>　　那种冰冷，甚至让得周遭被紫炎照耀下，仍旧呈结冰之象。
<br/>　　“好强的风力，好冷的罡风。”萧逸眉头紧皱。
<br/>　　正当他强行振翅高飞时。
<br/>　　瞳孔，忽然一缩。
<br/>　　不远处，竟出现了一道幽绿光芒。
<br/>　　在这漆黑环境下，那抹幽光，绿油油的，如鬼魅之火，让人不寒而栗。
<br/>　　一股骇人的暴风漩涡，忽然而生。
<br/>　　强大的吸力，直直吸住萧逸。
<br/>　　“不好。”萧逸脸色大变。
<br/>　　“给我破。”萧逸终归是萧逸，心头并未太过惊骇。
<br/>　　反倒反手取出暴雪剑，重重劈出一道寒冰裂天斩。
<br/>　　萧逸打算以强悍的攻击，抵消那道暴风吸力，然后快速逃离。
<br/>　　但其结果，却是寒冰裂天斩发出的惊人剑气，并未能奈何吸力半分。
<br/>　　几乎是一瞬间，萧逸还未反应过来，身上紫炎火翼，已然被寒冷罡风冻结成冰。
<br/>　　身体，不可抵抗地被直直吸向那抹幽光。
<br/>　　萧逸只觉两眼一黑。
<br/>　　待他恢复视力时，已然置身于一洞穴之内。
<br/>　　“嗯？”萧逸一惊。
<br/>　　极寒裂缝，两边本就是悬崖峭壁。
<br/>　　没想到，这悬崖峭壁中，会有一洞穴。
<br/>　　洞**，白茫茫一片，寒霜密布。
<br/>　　外头，凛冽恐怖的罡风，并未能吹入半分。
<br/>　　洞穴，并不太大，也就几十米长宽。
<br/>　　里头，并无他物或者危险，只有遍地白骨，骇人至极。
<br/>　　咔咔，萧逸脚步踩在了一块骨骸之上。
<br/>　　骨头，只发出微微声响，却无半分破碎痕迹。
<br/>　　很显然，这些尸体的主人，在死前，都必是修为惊人之辈。
<br/>　　“嗯？”萧逸眼睛微眯。
<br/>　　不远处，一具白骨上，怀着一锃亮的令牌。
<br/>　　这是雪翼雕一族的令牌。
<br/>　　在来极寒之地前，萧逸做过详细的调查。
<br/>　　数百年前，雪翼雕一族中，曾有一位极境强者，仗着雪翼雕天生操御风雪之力，打算入极寒裂缝探个究竟。
<br/>　　而其结果，却是这位强者，从此再无音讯。
<br/>　　如果没猜错的话，面前这具白骨的主人，便是当年雪翼雕一族的那位极境强者了。
<br/>　　萧逸眉头紧皱，心中愈发惊骇，也愈发谨慎。
<br/>　　雪翼雕一族，御风控雪之能，相当厉害。
<br/>　　一般巅峰战力层次的强者，其飞行速度，便不亚于极境强者。
<br/>　　而极境强者，其飞行速度，更是比一般的极境武者，强了十倍不止。
<br/>　　那等人物，尚且殒命在此。
<br/>　　这极寒裂缝，有着第一险地之名，该凶险到何等地步？
<br/>　　这时，萧逸目光偏移。
<br/>　　洞穴里头，还有好几具尸体，但并未化成白骨。
<br/>　　而是被寒霜冻结成冰。
<br/>　　尸身完好无损，里头已生机断绝。
<br/>　　想来，是死亡时间，并未太长。
<br/>　　“风雪剑主？”萧逸看向其中一具尸体，脸色一变。
<br/>　　不错，这具尸体，正是风雪剑主。
<br/>　　历代剑主，皆有画像放于宗门，萧逸自然一眼认出。
<br/>　　此时，风雪剑主，已经死去。
<br/>　　周遭的几具尸体，呈面容惊恐之状，手臂，直直地指着后方，似是见了什么恐怖之物。
<br/>　　唯有风雪剑主，一张严肃的面容下，是桀骜与冷傲，并无半分色变。
<br/>　　裂天剑宗，历代最强剑主，果真皆是不凡之辈。
<br/>　　“嗯？”萧逸皱了皱眉头，除了风雪剑主外，另外的尸体，动作很是怪异。
<br/>　　萧逸顺着他们手指的方向，微微转头。
<br/>　　一转头之下，顿时毛发悚然，头皮发麻。
<br/>　　是那抹幽绿光芒。
<br/>　　萧逸刚才还以为自己之前是错觉。
<br/>　　现在看来，哪里是错觉。
<br/>　　那抹幽光，是一张人脸，狞笑着。
<br/>　　萧逸反应极快，反手一剑刺出，直取幽光而去。
<br/>　　如无猜错，这洞**的遍地白骨，和这抹幽光，脱不了干系。
<br/>　　只不过，萧逸全力一剑刺出，暴雪剑穿透了幽光。
<br/>　　但那张人脸，却丝毫没有受损。
<br/>　　“桀桀。”幽光冷笑一声，“小子，小小年纪，竟有如此胆识，厉害。”
<br/>　　“你是何人？”萧逸忍着心头惊骇，冷声道。
<br/>　　“哈哈哈哈。”幽光大笑几声，“千万载过去了，我冰尊者，竟是无人认识了吗？”
