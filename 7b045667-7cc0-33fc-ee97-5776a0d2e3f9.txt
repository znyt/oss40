　　第511章
<br/>　　“放心，放心。”程大雷干脆答应：“不是为了帮侯爷你，我今天干嘛来了。咱们什么关系，岂能和外人合伙对付自己人。”
<br/>　　百里胜仔细打量着程大雷，口中道：“啊呀呀，贤弟当真是个仗义人呐。”
<br/>　　“谁说不是呢，可不一直都是这么仗义。”程大雷道。
<br/>　　“贤弟放心，只要你打下凉州城，为兄就敢在圣上面前，为你争一个凉州王的封号。”
<br/>　　“那个……圣上愿意出兵。”程大雷坚定道：“只要圣上的兵到，里应外合，必拿下凉州。”
<br/>　　“这个……呃，哈哈，贤弟你也该知道，现在不是和相府闹翻脸的时候。圣上对你的支持，也只能是道义上的支持。”
<br/>　　“唉……”程大雷叹口气：“替圣上分忧，替侯爷解难，我本责无旁贷，奈何，蛤蟆城现在缺钱少粮，士兵又病弱，打凉州这件事，怕还要从长计议。”
<br/>　　百里胜闭上嘴巴，约莫是摸清了现在的状况。程大雷的脾气是无利不起早，让他出白功，去和凉州作战，几乎是不可能的事情。
<br/>　　这一点，自己离开长安时就该提前想到的。
<br/>　　而此刻，百里胜还有另一样担心。程大雷拒绝和自己合作，是否会和凉州达成联盟。虽然他嘴上说得漂亮，但百里胜也知道，他的话是信不过的。
<br/>　　百里胜心中忽然一惊，不好，自己还是赶紧离开蛤蟆城。莫让程大雷拿了自己的人头做凉州的投名状。
<br/>　　“哈哈，有贤弟这话，我便可放心了。”百里胜道：“我军中还有些事，虽想和贤弟把盏天明，但实在公务缠身，有缘我们日后再会。”
<br/>　　程大雷点点头：“侯爷要走，我也不能强留。不过，凉州可已知侯爷到了，怕他们会在城外守着，不会轻易放侯爷离开。”
<br/>　　“这……”百里胜一惊，当真不是该如何是好，双手抱拳，冲程大雷深施一礼：“贤弟，可一定要帮帮为兄。”
<br/>　　“侯爷放心，依我看，你暂且再此地留上几日，过几日，我一定给侯爷一个结果。”
<br/>　　百里胜苦思无策，发觉自己竟已陷在蛤蟆城中。而程大雷的话，更是令他心惊，给自己一个结果……莫非是要将自己结果了么。
<br/>　　程大雷起身离去，百里胜却实在难以入眠，在房中翻来覆去，不知该如何怎么办。
<br/>　　这一夜平安过去，百里胜是一夜未眠，到第二天上午，徐神机来请百里胜。
<br/>　　“呀，百里侯爷这是怎么了，难道是招待不周，委屈了侯爷么？”徐神机盯着百里胜两个黑眼圈。
<br/>　　百里胜尴尬的摆摆手，道：“不知徐军师大驾光临，有何事吩咐。”
<br/>　　“我们大当家有请，有要事与百里侯爷相商。”
<br/>　　百里胜一头雾水，不知程大雷有什么目的，如果是要杀自己，当似乎不用这么麻烦。
<br/>　　心中七上八下，一路随着徐神机来到城主府，还未来到大厅，就听得里面响起程大雷宏亮的声音。
<br/>　　“侯爷来了，快快有请。”
<br/>　　百里胜走入大厅，待看清里面的情形，整个人忽然愣住。
<br/>　　只见大厅除了程大雷几个蛤蟆城的人，凉州钟伟虎竟然也在大厅中坐着。
<br/>　　钟伟虎也愣住了，刚才他还和程大雷谈得火热，说要如何对付百里胜。但转眼间，百里胜就出现了。
<br/>　　二人目光相对，脸上的表情都僵固住，一时谁也没有说话。
<br/>　　程大雷哈哈笑了一声，道：“二位怕是不认识吧，我与二位引见一下，这位是百里侯爷，这位是凉州城的钟大人。”
<br/>　　“喔喔，原来是侯爷到了，请恕在下无礼。”
<br/>　　“啊，免礼免礼，钟大人的名字我也是如雷贯耳。”
<br/>　　双方虚伪的客套着，虽然将军府和相府在庙堂上闹得不可开交。都想控制凉州这处塞上江南，然而，彼此表面上还没有扯破脸。虽然心里都巴不得弄死对方，但表面上还得笑脸相迎。
<br/>　　场上，最自在的就是程大雷了，他呵呵笑着，口中道：“百里侯爷马上就要统兵凉州，成了这三关统领，日后咱们三家还需和睦相处，一起发财。”
<br/>　　“和睦相处，和睦相处。”
<br/>　　钟伟虎和百里胜应承着，心里委实不太是滋味。
<br/>　　百里胜眼珠一转，听程大雷话中的意思，还是帮着自己居多。他接了三关统领的位置，官职不小，但压力也巨大。目的，就是为了限制相府在凉州的势力扩张。
<br/>　　可是想要领兵入凉州，就需要宋伯康的同意，所以将军府才想和程大雷合作，吞掉宋伯康。
<br/>　　“等侯爷上任时，我们和凉州王定要好好招待，扫平路上的关隘，免得有路上的山贼草莽找侯爷的麻烦。”
<br/>　　百里胜眼睛一亮，双手抱拳道：“有贤弟这话，我便放心了。”
<br/>　　“啧……”
<br/>　　钟伟虎叹了一声，心道程大雷这是要和百里胜联手的意思啊，放百里胜领兵进来，岂非是给自己后花园放了一把火。
<br/>　　“程当家，程当家……”钟伟虎站起来：“能否借一步说话。”
<br/>　　“这……啊，哈哈，那侯爷就稍等片刻，我去去就来。”
<br/>　　程大雷随着钟伟虎来到一处偏僻的地方，看了看左右无人，钟伟虎道：“程当家，你糊涂啊，凉州是咱们的地盘，怎么能放百里胜进来。”
<br/>　　“唉，钟大人你糊涂啊，我这可是完全为了你考虑。”
<br/>　　“为我考虑？”钟伟虎愣住了。
<br/>　　“看来你想不明白，如果我们把住关隘，不许他们的人进来，这岂不是和朝廷扯破了脸皮。难道凉州王是要造反么？”
<br/>　　“呃……这当然不可能。”
<br/>　　“所以说么，不能把事情做绝。”程大雷道：“放他们进来，他们的人总不能不吃不喝，粮草进来的道路，可都是我们把着呢，我们想给他们吃，不想给他们吃，可不就是看我们心情么。”
<br/>　　钟伟虎眼睛一亮，不可思议的看着程大雷：“程当家，高啊！”
<br/>　　程大雷拍拍钟伟虎的肩膀，道：“现在钟大人明白，谁真正对你好了吧。”
<br/>(本章完)
